# <a name="files-in-this-management-pack"></a>Archivos de este módulo de administración
Para supervisar [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)], primero debe descargar el [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] módulo de administración de [System Center Management Pack Catalog](http://www.microsoft.com/download/details.aspx?id=39617) (<http://www.microsoft.com/download/details.aspx?id=39617>). El [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] módulo de administración incluye los siguientes archivos:  
  
|Nombre de archivo|Nombre para mostrar|Descripción|  
|---------------|------------------|-----------------|  
|Microsoft.BizTalkServer2013.Library.mp|Biblioteca de BizTalk Server|Contiene las clases genéricas, las relaciones y otros bloques de creación de módulo de administración que se usan por otros módulos de administración para proporcionar una supervisión para todas las aplicaciones de BizTalk.|  
|Microsoft.BizTalk.Server.2013.Monitoring.mp|Supervisión de BizTalk Server|Contiene los monitores, reglas y vistas que proporcionan una manera extensible para controlar todas las aplicaciones de BizTalk.|  
|Microsoft.BizTalk.Server.2013.Discovery.mp|Detección de servidor BizTalk Server|Contiene las detecciones que se usan para buscar los distintos componentes de aplicaciones de BizTalk.|
