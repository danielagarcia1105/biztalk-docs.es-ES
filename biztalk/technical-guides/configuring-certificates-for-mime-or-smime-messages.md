# <a name="configuring-certificates-for-mime-or-smime-messages"></a>Configurar certificados para MIME o mensajes SMIME
Para ayudar a proteger de transferencia de datos en [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)], debe asociar certificados instalados en los almacenes de certificados con los artefactos de BizTalk correspondientes. Esto se aplica a los mensajes codificados con MIME/SMIME. También se aplica al transporte de AS2, que transporta los mensajes MIME/SMIME.  

 Utilice la tabla siguiente como referencia para los escenarios de uso de certificados y las opciones de configuración disponibles en [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)].
