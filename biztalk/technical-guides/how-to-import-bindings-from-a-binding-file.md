# <a name="how-to-import-bindings-from-a-binding-file"></a>Cómo importar enlaces desde un archivo de enlace
Puede importar los enlaces de una aplicación de BizTalk en otra aplicación de BizTalk existente mediante el uso de un archivo de enlace. También puede importar todos los enlaces para un grupo en un grupo o los enlaces de un ensamblado en una aplicación.  
  
-   Para obtener información acerca de diversas consideraciones a tener en cuenta al importar enlaces, vea [importar enlaces](http://go.microsoft.com/fwlink/?LinkId=155012) (http://go.microsoft.com/fwlink/?LinkId=155012).  
  
-   Para obtener instrucciones sobre cómo importar enlaces a una aplicación de BizTalk, consulte [cómo importar enlaces en una aplicación de BizTalk](http://go.microsoft.com/fwlink/?LinkId=155013) (http://go.microsoft.com/fwlink/?LinkId=155013).
