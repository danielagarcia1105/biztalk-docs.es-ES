# <a name="how-to-schedule-a-backup-biztalk-server-job"></a>Cómo programar un trabajo de copia de seguridad de BizTalk Server
El trabajo de copia de seguridad de BizTalk Server se ejecuta de acuerdo con lo programado por el servicio Agente SQL Server. Si desea crear copias de seguridad con mayor o menor frecuencia, puede cambiar la programación del trabajo de copia de seguridad de BizTalk Server mediante el Administrador corporativo de SQL Server. Para obtener más información sobre cómo programar un trabajo de copia de seguridad de BizTalk Server, vea [cómo programar el trabajo de copia de seguridad de BizTalk Server](http://go.microsoft.com/fwlink/?LinkId=154674) (http://go.microsoft.com/fwlink/?LinkId=154674) en [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] ayuda.
