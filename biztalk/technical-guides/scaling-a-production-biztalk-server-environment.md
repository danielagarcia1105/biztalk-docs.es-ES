# <a name="scaling-a-production-biztalk-server-environment"></a>Ajuste de escala en un entorno de BizTalk Server de producción
Esta sección proporciona información general sobre el entorno de laboratorio que se usó para realizar pruebas de carga de una solución de BizTalk Server, un resumen de los resultados de pruebas de carga y observaciones generales y recomendaciones en función de los resultados en el laboratorio.  
  
 La información proporcionada en estos temas puede y debe usarse como guía para escalar su [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] entorno. Además de esta guía se recomienda realizar periódicamente pruebas de carga durante el ciclo de desarrollo de su [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] aplicación.
