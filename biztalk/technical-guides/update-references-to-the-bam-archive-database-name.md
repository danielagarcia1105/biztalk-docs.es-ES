# <a name="update-references-to-the-bam-archive-database-name"></a>Actualizar referencias al nombre de base de datos de archivo BAM
Si ha realizado una copia de seguridad de las bases de datos de archivo de BAM, si se produce un error de datos o del sistema podrá restaurar dicha copia a otro equipo, y puede cambiar el nombre de la copia de seguridad.  
  
 Para restaurar las bases de datos de archivo de BAM, siga los pasos descritos en [cómo restaurar bases de datos en el trabajo de copia de seguridad de BizTalk Server](../technical-guides/how-to-restore-databases-in-the-backup-biztalk-server-job.md). Además, debe actualizar los paquetes SSIS de BAM con el nuevo nombre del servidor y el nombre de la base de datos.  
  
## <a name="how-to-update-references-to-bam-archive-database"></a>Cómo actualizar referencias a la base de datos de archivo BAM  
 Para obtener instrucciones sobre cómo actualizar referencias a la base de datos de archivo de BAM, consulte [actualizar referencias a la nueva base de datos de archivo de BAM](../technical-guides/how-to-move-the-bam-archive-database1.md#BKMK_UpdateArch).
