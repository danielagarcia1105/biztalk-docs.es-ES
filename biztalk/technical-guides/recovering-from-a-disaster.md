# <a name="recovering-from-a-disaster"></a>Recuperarse de un desastre
En esta sección se detalla los pasos para transferir las operaciones de producción en el sitio de recuperación ante desastres. Para los fines de esta sección, el sistema de producción en línea se conoce como el sistema de "origen". El sitio de recuperación ante desastres se conoce como el sistema de "destino". Esta sección proporciona información general del proceso de recuperación ante desastres.
