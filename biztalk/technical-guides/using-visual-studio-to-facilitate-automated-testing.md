# <a name="using-visual-studio-to-facilitate-automated-testing"></a>Con Visual Studio para facilitar la prueba automatizada
Visual Studio 2010 proporciona funcionalidad de prueba de carga eficaz que puede simular un perfil de carga de hasta cientos de usuarios que acceden al mismo tiempo a una aplicación de servidor. Esta funcionalidad de la prueba de carga proporciona métricas de tiempo real para los indicadores clave de rendimiento seleccionado, así como la capacidad para almacenar estas métricas en una base de datos para su posterior análisis. Esta sección describe el uso de Visual Studio de pruebas de carga para evaluar el rendimiento de una aplicación de BizTalk Server.
