# <a name="managing-certificates"></a>Administración de certificados
Esta sección describe cómo administrar los certificados digitales que se usan con [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)]. Sus temas describen cómo instalar certificados (y qué carpeta debe instalarlos en) y cómo configurar [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] para utilizar el certificado de MIME/SMIME y AS2 y para su uso con los adaptadores de BizTalk. Microsoft [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)] puede hacer uso de certificados digitales de infraestructura de clave pública (PKI) para los fines de cifrado y descifrado, firma y comprobación (sin repudio) de documentos y la resolución de entidades.  
  
 Para una lista de comprobación de pasos para instalar los certificados, consulte [lista de comprobación: instalación y configuración de certificados](~/technical-guides/checklist-installing-and-configuring-certificates.md).
