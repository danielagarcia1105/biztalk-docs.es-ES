# <a name="moving-non-bam-databases"></a>Mover bases de datos que no son de BAM
Puede usar este procedimiento para mover las bases de datos principales de BizTalk Server a otro servidor. También se puede usar este mismo procedimiento básico para mover las bases de datos de BizTalk Server desde un servidor SQL Server local a un servidor SQL remoto o a un clúster de SQL Server. En esta sección se describe cómo mover la [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] bases de datos que no están relacionados con la de BAM.  
  
## <a name="in-this-section"></a>En esta sección  
 Para mover la implementación de BAM no las bases de datos siguen los pasos descritos en el tema [cómo mover las bases de datos de BizTalk Server](http://go.microsoft.com/fwlink/?LinkId=210646) (<http://go.microsoft.com/fwlink/?LinkId=210646>) en [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] ayuda.  
  
 Esta sección también contiene un tema que describe los procedimientos que se deben seguir después de mover determinado [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] bases de datos. Complete los pasos del tema según corresponda para su entorno.
