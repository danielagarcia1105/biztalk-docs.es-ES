# <a name="monitoring-applications-and-host-instances"></a>Supervisión de aplicaciones e instancias de Host
Esta sección describe la supervisión de aplicaciones de BizTalk y las instancias de host mediante Operations Manager 2010. Ejecutar la herramienta de análisis de rendimiento de registros (PAL) también puede proporcionar detalles que pueden ayudar a supervisar las aplicaciones y las instancias de host. Para obtener más información acerca de esta herramienta, consulte [mediante la herramienta de análisis de rendimiento de registros (PAL)](../technical-guides/using-the-performance-analysis-of-logs-pal-tool.md).
