# <a name="msit-real-world-migration-story-from-gentran-51-to-biztalk-2010"></a>MSIT: Caso de migración del mundo Real desde Gentran 5.1 a BizTalk 2010
Artículo técnico de BizTalk  
  
 **MSIT: Caso de migración real desde Gentran 5.1 a BizTalk 2010**  
  
 **Autor:** Amit Kumar Dua, Microsoft &#124; Banupriya Thirumaran, Microsoft  
  
 **Revisores:** Mandi Ohlinger, Microsoft &#124; Nitin Mehrotra, Microsoft  
  
 **Colaboradores:** Nikhil Tayal, Microsoft &#124; Anil Chandra Lingam, Microsoft  
  
 **Fecha de publicación:** octubre de 2014  
  
 **Se aplica a:** BizTalk Server 2010, servidor Gentran  
  
 **Resumen:** la plataforma de integración de TI de Microsoft (MSIT) usa BizTalk Server y Gentran. Hasta hace poco, Microsoft IT tenía una buena presencia de una superficie Gentran. Con BizTalk Server admite las capacidades de computación y AS2/EDI en la nube, hay una oportunidad para reemplazar Gentran con Microsoft BizTalk Server 2010.  Gentran es compatible con Windows Server 2003 y SQL server 2005; que se encuentran en soporte extendido que pronto se está finalizando. No hay ningún plan para Gentran admitir las plataformas más recientes, incluido Windows Server 2008/2012 y SQL Server 2008/2014.  
  
 MSIT ve esto como una oportunidad para:  
  
-   Obtener en las plataformas más recientes de tecnología  
  
-   Quite la restricción de compatibilidad  
  
-   Simplificar la arquitectura global de la plataforma de integración  
  
-   Asegúrese de la plataforma de integración más sólido y rentable  
  
 BizTalk Server 2010 es una plataforma de integración comprobada, cuenta con numerosas capacidades y características y es compatible con las nuevas tecnologías de plataforma.  
  
 Estas notas del producto se describe la estrategia y pasos para migrar desde Gentran 5.1 a BizTalk Server.
