# <a name="how-to-restore-the-master-secret-server"></a>Cómo restaurar el servidor secreto principal
Para obtener información acerca de cómo restaurar el servidor secreto principal, consulte [cómo restaurar el secreto principal](http://go.microsoft.com/fwlink/?LinkId=151394) (http://go.microsoft.com/fwlink/?LinkId=151394).  
  
> [!NOTE]  
>  Para restaurar el secreto principal de SSO, debe haber creado previamente una copia de seguridad del secreto principal de SSO, siga los pasos descritos en [cómo volver la seguridad del secreto principal](http://go.microsoft.com/fwlink/?LinkId=151395) (http://go.microsoft.com/fwlink/?LinkId=151395).
