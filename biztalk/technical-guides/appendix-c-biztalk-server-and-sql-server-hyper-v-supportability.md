# <a name="appendix-c-biztalk-server-and-sql-server-hyper-v-supportability"></a>Apéndice C: compatibilidad de Hyper-V de BizTalk Server y SQL Server
Los escenarios de prueba se describen en [probar el rendimiento de virtualización de BizTalk Server](../technical-guides/testing-biztalk-server-virtualization-performance.md) realizadas con [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)] y [!INCLUDE[btsSQLServer2008](../includes/btssqlserver2008-md.md)]. [!INCLUDE[btsBizTalkServer2006r3](../includes/btsbiztalkserver2006r3-md.md)]se admite cuando está instalado en un sistema operativo compatible que se ejecuta en Microsoft Virtual Server 2005 o en [!INCLUDE[btsWinSvr2k8](../includes/btswinsvr2k8-md.md)] Hyper-V. Para obtener más información, vea el artículo de Microsoft Knowledge Base "Compatibilidad de Microsoft BizTalk Server en una máquina virtual" disponible en [http://go.microsoft.com/fwlink/?LinkId=148941](http://go.microsoft.com/fwlink/?LinkId=148941).  
  
 Admite la directiva para [!INCLUDE[btsSQLServer2008](../includes/btssqlserver2008-md.md)] cuando está instalado en un sistema operativo compatible que se ejecuta en [!INCLUDE[btsWinSvr2k8](../includes/btswinsvr2k8-md.md)] Hyper-V se documenta en el artículo de Microsoft Knowledge Base "Directiva de soporte técnico para productos de Microsoft SQL Server que se ejecutan en un hardware entorno de virtualización"disponible en [http://go.microsoft.com/fwlink/?LinkId=148942](http://go.microsoft.com/fwlink/?LinkId=148942).  
  
> [!IMPORTANT]  
>  Al redactar de esta guía, la agrupación en clústeres de una instancia de SQL Server en un entorno de Hyper-V no es un escenario admitido.
