# <a name="installing-biztalk-server-2010-accelerator-for-swift"></a>Instalación del Acelerador de BizTalk Server 2010 para SWIFT
![Logotipo de](../technical-guides/media/bts-10-installaccelerator-logo.gif "BTS_10_InstallAccelerator_Logo")  
  
 Artículo técnico de BizTalk  
  
 **Instalar BizTalk Server 2010 Accelerator para SWIFT**  
  
 **Escritor:** Maria Quian, Microsoft Corporation  
  
 **Revisores técnicos:** Andres Naranjo, Mandi Ohlinger  
  
 **Fecha de publicación:** junio de 2012  
  
 **Se aplica a:** Acelerador de BizTalk Server 2010 para SWIFT  
  
 **Resumen:** la base para una correcta instalación de Microsoft BizTalk Server 2010 Accelerator para SWIFT (A4SWIFT) es una descripción de las características de A4SWIFT y los componentes de BizTalk que depende cada característica. Cada característica de A4SWIFT depende de otros componentes de BizTalk configuradas correctamente. Otras consideraciones que afectan a la creación de un entorno de A4SWIFT incluyen una infraestructura de red y un entorno de BizTalk Server ya configurado. Este documento se hacen referencia y usar la documentación existente para la creación de entornos de BizTalk Server. A4SWIFT hereda todos los requisitos de hardware y software para BizTalk Server 2010.  
  
 Una guía de instalación está disponible para cada componente relacionado de A4SWIFT, pero no un solo documento compilado que se elaboran los pasos de instalación y configuración. Este documento se les reunir e identificar algunos de los problemas comunes que se encuentran durante la instalación y configuración.
