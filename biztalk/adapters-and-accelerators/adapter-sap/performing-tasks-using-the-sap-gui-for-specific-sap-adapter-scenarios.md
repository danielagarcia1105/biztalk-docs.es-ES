# <a name="sap-gui-tasks-for-the-sap-adapter-scenarios"></a>Tareas de GUI de SAP para los escenarios de adaptador SAP

## <a name="overview"></a>Información general
Para muchos escenarios que implican la [!INCLUDE[adaptersap](../../includes/adaptersap-md.md)] y el sistema SAP, deberá realizar algunas operaciones en el sistema SAP. Por ejemplo, para recibir una llamada RFC en la [!INCLUDE[adaptersap_short](../../includes/adaptersap-short-md.md)], en primer lugar enviar una solicitud de cambio desde el sistema SAP. Esta sección enumeran los pasos generales para realizar estas tareas en el sistema SAP.  
  
> [!IMPORTANT]
>  Para realizar estas tareas, se recomienda ponerse en contacto con el Administrador de SAP, o consulte las instrucciones de SAP.
