# <a name="module-2-adding-a-new-schemas-project"></a>Módulo 2: Agregar un nuevo proyecto de esquemas
Al instalar Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)], [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] actualiza Microsoft [!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)] para incluir un proyecto de BizTalk básico. Esta plantilla de proyecto proporciona acceso a herramientas como el Diseñador de esquemas, el asignador de BizTalk, el Diseñador de orquestaciones, el Diseñador de canalizaciones y el Explorador de BizTalk y otras herramientas necesarias para crear una solución de BizTalk.
