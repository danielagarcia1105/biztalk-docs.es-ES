# <a name="module-7-testing-a-valid-flat-file-instance"></a>Módulo 7: Probar una instancia de archivo plano válida
En este módulo, envíe un ejemplo válido MT103 que creó en los laboratorios anteriores de puertos de recepción de archivo sin formato en el archivo. Esta tarea comprueba las canalizaciones de recepción que creó en laboratorios anteriores. Microsoft [!INCLUDE[A4SWIFT_CurrentVersion_FirstRef](../../includes/a4swift-currentversion-firstref-md.md)] escribe la salida en formato XML en la carpeta de salida en el puerto de envío que ha seleccionado en la lección anterior, [lección 2: agregar un puerto de envío XML](../../adapters-and-accelerators/accelerator-swift/lesson-2-adding-an-xml-send-port.md).  
  
 Iniciar el archivo copiando un archivo con formato sin formato de SWIFT a la carpeta de entrada del adaptador de recepción. Esta acción da como resultado [!INCLUDE[btaA4SWIFT2.3abbrevnonumber](../../includes/btaa4swift2-3abbrevnonumber-md.md)] enrutamiento de un archivo XML de SWIFT válido a la carpeta de salida.  
  
 En esta tarea, también enviar un mensaje MT103 que no es válido. Utilice el evento para resolver el error.
