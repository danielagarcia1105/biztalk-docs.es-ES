# <a name="getting-started-with-biztalk-accelerator-for-swift"></a>Introducción al Acelerador de BizTalk para SWIFT
Microsoft [!INCLUDE[A4SWIFT_CurrentVersion_FirstRef](../../includes/a4swift-currentversion-firstref-md.md)] proporciona un conjunto de esquemas, las reglas y las herramientas asociadas que le ayudarán a la integración de mensajería de SWIFT con las aplicaciones internas. A4SWIFT requiere MicrosoftBizTalk servidor y usa [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] componentes y características para implementar los esquemas, análisis, validación, las reglas de procesamiento y orquestación. A4SWIFT también implementa la desagrupación de archivos que contienen varios mensajes financieros (FIN), reparación de mensajes y nuevo envío y conciliación de respuestas de FIN.  
  
 Puede obtener experiencia práctica con A4SWIFT, vaya directamente al Tutorial de extremo a otro de A4SWIFT. [Módulo 1: Creación de una solución de SWIFT](../../adapters-and-accelerators/accelerator-swift/module-1-creating-a-swift-solution.md) le ayuda a generar una solución de integration (EAI) de aplicaciones empresariales que automatiza un proceso empresarial interno. [Módulo 2: Agregar un nuevo proyecto de esquemas](../../adapters-and-accelerators/accelerator-swift/module-2-adding-a-new-schemas-project.md) compila el proyecto EAI ayudándole a desarrollar una solución automatizada para procesar mensajes de FIN de SWIFT seleccionados.
