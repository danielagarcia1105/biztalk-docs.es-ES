# <a name="module-4-adding-an-xml-receive-location-and-flat-file-send-port"></a>Módulo 4: Agregar un documento XML de recepción ubicación y el puerto de envío de archivos sin formato
En este módulo, agregue un puerto de recepción y ubicación, y un puerto de envío. Usar el puerto de recepción y la ubicación para recibir mensajes con formato XML MT103 desde una carpeta de archivo de entrada. Usar el puerto de envío para enviar los mensajes de archivo sin formato MT103 validados en una carpeta de archivo de salida.
