# <a name="biztalk-accelerator-for-hl7-components"></a>Acelerador de BizTalk para HL7 componentes
Acelerador de Microsoft BizTalk para HL7 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) entre los componentes incluyen desensambladores de HL7 y XML, ensambladores de HL7 y XML, recibir y enviar una orquestación de lotes, canalizaciones y puertos. Estos componentes caben en el Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] sistema para proporcionar una solución perfecta de atención médica.
