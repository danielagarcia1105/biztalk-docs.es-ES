# <a name="how-biztalk-accelerator-for-hl7-works"></a>Cómo funciona el Acelerador de BizTalk para HL7
Acelerador de Microsoft BizTalk para HL7 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) proporciona una solución sanitaria mediante la creación de Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]. Para ello, proporcione soporte para los mensajes sanitarios y la adición de componentes específicos de atención de salud que evalúe el mensaje de procesamiento a las aplicaciones de atención médica.  
  
 Para obtener más información, consulte [lo que el Acelerador de BizTalk para HL7 agrega a BizTalk Server](../../adapters-and-accelerators/accelerator-hl7/what-biztalk-accelerator-for-hl7-adds-to-biztalk-server.md).
