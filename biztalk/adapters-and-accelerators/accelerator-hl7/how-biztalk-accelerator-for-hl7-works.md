# <a name="how-biztalk-accelerator-for-hl7-works"></a>Cómo funciona el Acelerador de BizTalk para HL7
[!INCLUDE[btsCoName](../../includes/btsconame-md.md)]Acelerador de BizTalk para HL7 ([!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)]) proporciona una solución de atención sanitaria a partir de [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]. Para ello, proporcionar soporte para los mensajes de atención sanitaria y agregar componentes específicos de la cobertura de servicios médicos que personalizan el mensaje de procesamiento a las aplicaciones de atención médica.  
  
 Para obtener más información, consulte [lo que el Acelerador de BizTalk para HL7 agrega a BizTalk Server](../../adapters-and-accelerators/accelerator-hl7/what-biztalk-accelerator-for-hl7-adds-to-biztalk-server.md).
