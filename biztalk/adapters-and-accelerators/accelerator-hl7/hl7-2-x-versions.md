# <a name="hl7-2x-versions"></a>Versiones de HL7 2.X
En la tabla siguiente se enumera las versiones 2.X del nivel de mantenimiento siete (HL7) estándar, las fechas a partir de la organización de HL7 la aprobación y enumera qué versiones del Acelerador de Microsoft BizTalk para HL7 ([!INCLUDE[HL7_CurrentVersion_FirstRef](../../includes/hl7-currentversion-firstref-md.md)]) admite.  


| Versión de HL7 | Fecha de aprobación | Compatible con [!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)] |
|-------------|---------------|------------------------------------------------------------------------------------------------------|
|     2.1     |  31 de diciembre de 1992  |                                                 Sí                                                  |
|     2.2     |  De 1994 01 de Dic  |                                                 Sí                                                  |
|     2.3     |  07 de abril de 1997  |                                                 Sí                                                  |
|    2.3.1    |  17 de mayo de 1999  |                                                 Sí                                                  |
|     2.4     |  01 de Nov de 2000  |                                                 Sí                                                  |
|     2.5     | 26 de junio de 2003 |                                                 Sí                                                  |

> [!NOTE]
>  Dado que las versiones, 2.1, 2.2 y 2.3 están actualizados, se recomienda que use la versión 2.3.1 o posterior del estándar 2.X.  

 Al instalar [!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)], el Asistente para instalación crea el  *\<unidad\>*: \ Archivos de programa\\Microsoft BizTalk \<versión\> Acelerador de la estructura de carpetas HL7\Templates\Schemas que contiene plantillas para admitir los eventos de HL7. Debajo de esta carpeta, un V2. Carpeta X incluye una subcarpeta para cada uno de los HL7 versiones 2.X: 2.1, 2.2, 2.3, 2.3.1, 2.4 y 2.5.
