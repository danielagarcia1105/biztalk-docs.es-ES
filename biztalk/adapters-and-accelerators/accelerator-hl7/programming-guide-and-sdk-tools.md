# <a name="programming-guide-and-sdk-tools"></a>Guía de programación y herramientas de SDK
Este Kit de desarrollo de Software (SDK) proporciona información para desarrolladores que crean aplicaciones con cualquiera de las herramientas de Microsoft [!INCLUDE[btaBTAHL7NoNumber](../../includes/btabtahl7nonumber-md.md)], la programación de aplicaciones pública interfaces (API) o las utilidades y ejemplos proporcionan en el BTAHL7 SDK DE.  
  
## <a name="tips-for-using-help-in-a-developer-environment"></a>Sugerencias para usar la Ayuda en un entorno de desarrollo  
 Ayuda de BTAHL7 contiene características que puede usar para mostrar la documentación para desarrolladores en su idioma preferido y de vínculo entre BTAHL7 y la Ayuda de Visual Studio.  
  
### <a name="using-language-filtering"></a>Usar el filtrado de lenguaje  
 El BTAHL7 [!INCLUDE[btsDotNet](../../includes/btsdotnet-md.md)] referencia de la clase proporciona las firmas y ejemplos de código en varios lenguajes de programación. Las etiquetas siguientes indican la información específica del idioma:  
  
 **[** [!INCLUDE[btsCSharp](../../includes/btscsharp-md.md)] **]**  
  
 **[** [!INCLUDE[btsVBNoVersion](../../includes/btsvbnoversion-md.md)] **]**  
  
 **[C++]**  
  
 **[** [!INCLUDE[btsJScript](../../includes/btsjscript-md.md)] **]**  
  
 Puede personalizar la vista del contenido en las páginas de referencia para mostrar sólo la información en el lenguaje de programación preferido. Para seleccionar un idioma personalizado, use el **filtro de lenguaje** situado en la esquina superior izquierda de esta página. Una vez habilitada, filtrado de lenguaje continúa hasta que se cambie. Puede ver que está habilitado el filtrado de idioma en una página, porque el nombre del lenguaje sigue el título en la barra azul en la parte superior de la página.  
  
### <a name="linking-between-btahl7-help-and-microsoft-visual-studio-help"></a>La vinculación entre BTAHL7 ayuda y ayuda de Microsoft Visual Studio  
 Cuando un miembro se hereda de la [!INCLUDE[btsDotNetFramework](../../includes/btsdotnetframework-md.md)] biblioteca de clases Base, dos vínculos se proporcionan, tal como se muestra en el ejemplo siguiente:  
  
|||  
|-|-|  
|![](../../adapters-and-accelerators/accelerator-hl7/media/protmethod.gif "protmethod") Finalize (se hereda de **System.Object**)|Para obtener más información sobre el espacio de nombres del sistema, consulte [sistema Namespace clase](https://msdn.microsoft.com/library/system(v=vs.110).aspx).|  
  
 Si está viendo BTAHL7 en [!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)] o en el [!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)] documentación, el vínculo en la columna izquierda irá a la página de miembro exacto. Si está viendo BTAHL7 ayuda fuera de [!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)], use el vínculo en la columna derecha para ir a la página de espacio de nombres del sistema en MSDN Library. Tenga en cuenta que el vínculo no irá a la página de miembro concreto.
