# <a name="step-8-restart-biztalk-server"></a>Paso 8: Reinicie el servidor BizTalk Server
En este paso, reinicie el [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] en orden para que los cambios realizados en este tutorial surta efecto.  
  
### <a name="to-restart-biztalk-server"></a>Para reiniciar el servidor BizTalk Server  
  
1.  Iniciar **administración de BizTalk Server**.  
  
2.  En el [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] administración de la consola, expanda **administración de BizTalk Server**, expanda **grupo de BizTalk**y, a continuación, expanda **configuración de plataforma**. Haga clic en **Hosts**y, a continuación, haga clic en **BizTalkServerApplication**.  
  
3.  Haga clic en el servidor y, a continuación, haga clic en **reiniciar**.
