# <a name="step-8-configure-party-information"></a>Paso 8: Configurar la información de entidad
En este paso, configurará la información de entidad para cada uno de los receptores del mensaje (ADT, RX y su) y asignar puertos de envío para habilitar las transformaciones de encabezado de mensaje para el mensaje saliente mediante [!INCLUDE[btaBTAHL71.3abbrevnonumber](../../includes/btabtahl71-3abbrevnonumber-md.md)] Explorador de configuración.  
  
 Este paso consta de los pasos siguientes, que se deben realizar en el orden siguiente:  
  
1.  [Paso 8: configurar la información de entidad para el sistema ADT](../../adapters-and-accelerators/accelerator-hl7/step-8a-configure-party-information-for-the-adt-system.md)  
  
2.  [Paso 8: configurar la información de entidad para el sistema RX](../../adapters-and-accelerators/accelerator-hl7/step-8b-configure-party-information-for-the-rx-system.md)  
  
3.  [Paso 8C: configurar la información de entidad para el sistema de alta](../../adapters-and-accelerators/accelerator-hl7/step-8c-configure-party-information-for-the-hi-system.md)
