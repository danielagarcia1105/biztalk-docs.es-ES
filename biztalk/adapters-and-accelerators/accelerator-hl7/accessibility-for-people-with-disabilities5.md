# <a name="accessibility-for-people-with-disabilities"></a>Accesibilidad para personas con discapacidades
Microsoft se ha comprometido a hacer que sus productos y servicios sean más fáciles de utilizar para todos los usuarios. Para obtener información general sobre las características de accesibilidad y servicios de Microsoft, vea el sitio Web de accesibilidad de Microsoft en [http://go.microsoft.com/fwlink/p/?LinkId=56272](http://go.microsoft.com/fwlink/p/?LinkId=56272).
