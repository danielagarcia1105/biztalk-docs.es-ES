# <a name="hl7-2x-common-schema-files"></a>Archivos de esquema comunes de HL7 2.X
Cada esquema de mensaje HL7 requiere archivos de esquema comunes que se proporcionan definiciones de esquema para tipos de datos, segmentos y tablas. Estos archivos de esquema comunes se definen para cada versión de HL7 2.X. El proyecto implementado en Microsoft [!INCLUDE[btsVStudioNoVersion](../../includes/btsvstudionoversion-md.md)] que contiene el esquema del mensaje debe hacer referencia a una instancia implementada de proyecto comunes que contiene los tres esquemas comunes.
