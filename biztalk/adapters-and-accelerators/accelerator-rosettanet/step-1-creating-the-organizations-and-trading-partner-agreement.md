# <a name="step-1-creating-the-organizations-and-trading-partner-agreement"></a>Paso 1: Creación de las organizaciones y acuerdo de socio comercial
En este paso, agregará la información de la organización mediante la [!INCLUDE[btaBTARNNoVersion](../../includes/btabtarnnoversion-md.md)] consola de administración. Después de agregar la información de contacto para ambas organizaciones, el último paso es configurar un acuerdo de socio comercial para el proceso de interfaz de socio (PIP) 3A2. Si completó el Tutorial de doble acción, ya se ha realizado los pasos necesarios y puede ir a [paso 2: crear Fabrikam LOBWebApplication](../../adapters-and-accelerators/accelerator-rosettanet/step-2-creating-the-fabrikam-lobwebapplication.md). Para crear las organizaciones y el acuerdo de socio comercial, vea lo siguiente:  
  
-   [Paso 1: Crear la organización principal de Fabrikam](../../adapters-and-accelerators/accelerator-rosettanet/step-1-creating-the-fabrikam-home-organization.md)  
  
-   [Paso 2: Crear la organización del asociado de Contoso](../../adapters-and-accelerators/accelerator-rosettanet/step-2-creating-the-contoso-partner-organization.md)  
  
-   [Paso 5: Crear el acuerdo de socio comercial Fabrikam 3A2](../../adapters-and-accelerators/accelerator-rosettanet/step-5-creating-the-fabrikam-3a2-trading-partner-agreement.md)  
  
> [!IMPORTANT]
>  Estos pasos se encuentran en el Tutorial de doble acción. Después de completar cada paso, volver a este tema para seguir el Tutorial de procesos privado.
