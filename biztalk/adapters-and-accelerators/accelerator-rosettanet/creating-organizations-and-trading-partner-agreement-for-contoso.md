# <a name="creating-organizations-and-trading-partner-agreement-for-contoso"></a>Creación de organizaciones y comerciales de acuerdo con el socio para Contoso
En esta sección, agregará la información de la organización es necesario con el Acelerador de BizTalk para RosettaNet Management Console. Después de agregar la información de contacto para ambas organizaciones, el último paso es configurar un acuerdo de socio comercial para el proceso de interfaz de socio (PIP) 3A2. Si se ha completado los pasos en el Tutorial de doble acción, ya ha realizado los pasos necesarios. Puede mover a [crear los esquemas de LOB de Contoso y asignaciones](../../adapters-and-accelerators/accelerator-rosettanet/creating-the-contoso-lob-schemas-and-maps.md). Para crear las organizaciones y el acuerdo de socio comercial, vea lo siguiente:  
  
-   [Paso 1: Crear la organización principal de Contoso](../../adapters-and-accelerators/accelerator-rosettanet/step-1-creating-the-contoso-home-organization.md)  
  
-   [Paso 2: Crear la organización del socio comercial Fabrikam](../../adapters-and-accelerators/accelerator-rosettanet/step-2-creating-the-fabrikam-partner-organization.md)  
  
-   [Paso 5: Crear el acuerdo de socio comercial de 3A2 de Contoso](../../adapters-and-accelerators/accelerator-rosettanet/step-5-creating-the-contoso-3a2-trading-partner-agreement.md)  
  
> [!IMPORTANT]
>  Estos pasos se encuentran en el Tutorial de doble acción. Después de completar cada paso, volver a este tema para seguir el Tutorial de procesos privado.
