# <a name="creating-and-configuring-biztalk-ports-for-contoso"></a>Crear y configurar puertos de BizTalk para Contoso
En esta sección, integrar la solución actual en Microsoft® BizTalk Server. Antes de hacer esto, instale al ensamblado en la caché de ensamblados Global (GAC) y, a continuación, configurarlo en la base de datos de configuración. A continuación, crear y configurar puertos de envío mediante un adaptador SQL y un adaptador HTTP.
