# <a name="accessibility-for-people-with-disabilities"></a>Accesibilidad para personas con discapacidades
Microsoft se ha comprometido a hacer que sus productos y servicios sean más fáciles de utilizar para todos los usuarios. Para obtener información general sobre las características de accesibilidad y servicios de Microsoft, vea el sitio Web de accesibilidad de Microsoft en [http://go.microsoft.com/fwlink/?LinkId=56272](http://go.microsoft.com/fwlink/?LinkId=56272).  
  
 Microsoft [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] ayuda pueden hacer más accesible para personas con discapacidades. Por ejemplo, es posible personalizar el color y el tamaño de las fuentes de los cuadros de diálogo y de la configuración del explorador. El siguiente procedimiento explica cómo encontrar esta información de accesibilidad en Microsoft Document Explorer, la interfaz de usuario en el que se muestra la Ayuda.  
  
### <a name="to-find-information-about-accessibility-options-for-microsoft-biztalk-accelerator-for-rosettanet-help"></a>Para obtener información sobre las opciones de accesibilidad para el Acelerador de Microsoft BizTalk para RosettaNet ayuda  
  
1.  En el **ayuda** menú, haga clic en **ayuda sobre la Ayuda**. El **ayuda sobre la Ayuda** para Microsoft Document Explorer se muestra la página.  
  
2.  En el **ayuda sobre la Ayuda** página, haga clic en **personalizar Microsoft Document Explorer**.  
  
3.  En el **personalizar Microsoft Document Explorer** página, haga clic en **accesibilidad y Microsoft Document Explorer**.
