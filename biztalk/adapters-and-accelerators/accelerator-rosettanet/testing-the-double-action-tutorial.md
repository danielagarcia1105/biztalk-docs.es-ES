# <a name="testing-the-double-action-tutorial"></a>Prueba del tutorial de doble acción
En esta sección usará las soluciones de Contoso y Fabrikam que creó en las secciones anteriores para probar cuatro procesos de interfaz de socio (PIP) diferentes: 0C2, 0C4, 3A2 y 3A4. Use LOBWebApplication en el equipo de Fabrikam para realizar la solicitud. El sistema enviará la solicitud al equipo de Contoso mediante un canal de comunicación mejorada. La orquestación de doble acción generará una respuesta adecuada según el PIP que use.
