# <a name="tracking-sample"></a>Ejemplo de seguimiento
El ejemplo de seguimiento de la interfaz de usuario muestra cómo crear un [!INCLUDE[btsDotNet](../../includes/btsdotnet-md.md)]-según la aplicación Web para mostrar el estado actual de mensajes y los procesos en [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]. La aplicación presenta al usuario un formulario que contenga varios criterios de búsqueda. Cuando el usuario realiza una búsqueda, el ejemplo de seguimiento genera una lista detallada que muestra los mensajes y los procesos de [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] que ajusten a dichos criterios.  
  
 Este ejemplo está disponible como una descarga desde el [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] sitio Web en [http://go.microsoft.com/fwlink/?LinkId=37467](http://go.microsoft.com/fwlink/?LinkId=37467).
