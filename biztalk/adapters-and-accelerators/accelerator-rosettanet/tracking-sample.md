# <a name="tracking-sample"></a>Ejemplo de seguimiento
El ejemplo de seguimiento de la interfaz de usuario muestra cómo crear un [!INCLUDE[btsDotNet](../../includes/btsdotnet-md.md)]-aplicación Web para mostrar el estado actual de mensajes y los procesos de Microsoft basada en [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]. La aplicación presenta al usuario con un formulario que contiene varios criterios de búsqueda. Cuando el usuario realiza una búsqueda, el ejemplo de seguimiento genera una lista detallada que muestra los mensajes y procesa en [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] que se ajusten a los criterios.  
  
 Este ejemplo está disponible como descarga de Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] sitio Web en [ http://go.microsoft.com/fwlink/?LinkId=37467 ](http://go.microsoft.com/fwlink/?LinkId=37467).
