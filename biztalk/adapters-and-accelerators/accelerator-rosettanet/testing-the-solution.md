# <a name="testing-the-solution"></a>Probar la solución
En esta sección, probará la solución completa. Utilice la herramienta de LOBWebApplication creada en la solución de Fabrikam para enviar solicitudes PIP de 3A2 a la aplicación de LOB de Contoso. La orquestación de Contoso privada que ha creado, a continuación, envía una solicitud de precio y disponibilidad de según 3A2 de Contoso al sistema ERP mediante el adaptador SQL para el servidor BizTalk Server. Cuando se recibe la respuesta del sistema ERP, la orquestación llama el motor de reglas de negocios para exigir la emergencia debe directiva empresarial que ha creado.
