# <a name="the-supply-chain-solution"></a>La solución de cadena de suministro
Tecnología puede ayudar a resolver los desafíos empresariales relacionados con las comunicaciones de datos y procesos empresariales. Información de los sistemas de TI pueden ayudar a las empresas de alta tecnología transformar las comunicaciones con sus socios de procesos desconectados, incompatibles a integrada, las estandarizado.  
  
## <a name="supply-chain-integration"></a>Integración de la cadena de suministro  
 Incluso si los procesos de los asociados independientes en una cadena de suministro no son compatibles, puede integrarlos. Se puede asociar junto con la tecnología que transforma un sistema que se caracteriza por una variedad de conexiones personalizadas para uno basado en una única conexión estándar. Para obtener un ejemplo de esta solución, consulte la siguiente figura y compararlo con el sistema que se muestra en la figura [el desafío de proporcionar la cadena](../../adapters-and-accelerators/accelerator-rosettanet/the-supply-chain-challenge.md).  
  
 ![&#60;Ningún cambio&#62;](../../adapters-and-accelerators/accelerator-rosettanet/media/rn3-integrated-supply-chain-scenario.gif "RN3_Integrated_Supply_Chain_Scenario")  
  
 En este sistema, la interfaz estándar entre socios comerciales es una conexión de RosettaNet Implementation Framework (RNIF). El sistema es una implementación compatible con RosettaNet que reemplaza todas las interacciones manuales y personalizadas con conexiones RNIF automatizadas. Esto no impide que las empresas de personalizar sus sistemas, crear sus propios acuerdos con sus socios o mediante otros dispositivos de comunicación cuando quiera o deba. Este sistema ofrece a los asociados de la flexibilidad para personalizar el sistema para sus propios fines. Sin embargo, esto significa que todos los asociados usan procesos coherentes en sus comunicaciones de datos rutinarias.  
  
 Todas las compañías en esta cadena de suministro compartan una conexión RNIF. Para una empresa grande, puede configurar un sistema de empresa totalmente automatizada con servidor MicrosoftBizTalk y [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]. También puede habilitar un socio comercial usar cualquier implementación compatible con RosettaNet aún están instalados, o seguir usando un sistema EDI. Una empresa pequeña, por otro lado, no dispongan de los recursos de un completo [!INCLUDE[btaBTARN3.3abbrevnonumber](../../includes/btabtarn3-3abbrevnonumber-md.md)] sistema. Para ellos, puede configurar un servicio Web que les permite conectarse al sistema de la cadena de suministro. En este caso, el requisito de tecnología única es un explorador Web. En cada uno de estos casos, todas las compañías en el sistema de aprovechar las ventajas de un enfoque unificado a la conexión con sus socios.  
  
## <a name="the-benefits-of-standardization"></a>Las ventajas de estandarización  
 Las empresas pueden intercambiar datos por medios electrónicos sin participar en un sistema integrado. Sin embargo, si los datos mantenidos por separado los socios comerciales se están en diferentes formatos y sus sistemas de protocolos de transporte, aunque diferentes datos, el intercambio de datos puede requerir un esfuerzo significativo para diseñar y personalizar. Puede mejorar considerablemente la eficacia de las transacciones de negocio si cada socio conectado estandariza el formato y el protocolo utilizado con el intercambio de datos.  
  
 La organización RosettaNet creada en estándares que definen cómo los asociados intercambian mensajes a través de Internet (RNIF) y cómo los asociados de estructurar el contenido de los mensajes, procesos de interfaz de socio (PIP). Certificación de RosettaNet listos garantiza la interoperabilidad. Si una solución de integración de RosettaNet certificada, puede ampliarlo a varios socios. Puede controlar las soluciones existentes, tanto si son tecnologías de integración, por ejemplo, EDI y sistemas de back-end, como Oracle y SAP.  
  
 Estándares de RosettaNet centran en las necesidades específicas de la industria de fabricación de alta tecnología. Cuando todos los asociados en una cadena de suministro basar sus esquemas y el procesamiento en los mismos estándares de mensajes, reducen el esfuerzo, costo y riesgo asociado a su intercambio de datos general y obtienen un sistema que satisfaga sus necesidades específicas.  
  
## <a name="visibility-and-cost"></a>Visibilidad y el costo  
 Con un sistema integrado, puede conectarse a cada socio comercial, proporcionar los mecanismos que se enlazan sus procesos juntos y resolver sus diferencias. Un sistema integrado automatiza el intercambio de datos según los estándares del sector, para que pueden ejecutar los procesos empresariales con una mínima intervención humana. Esto proporciona compatibilidad para las comunicaciones y le permite aprovechar las ventajas de sus inversiones en tecnología.  
  
 Con un sistema integrado, tendrá una mejor visibilidad en las posiciones del inventario a lo largo de la cadena de suministro. Puede medir la demanda más rápida y precisa y responder ante tendencias. Puede determinar dónde está un pedido en la cadena de suministro y evaluar con precisión el estado del sistema.  
  
 Con una mayor visibilidad de la cadena de suministro de extremo a otro, puede reducir los costos al reducir el inventario, aumentar vueltas de inventario y reducir los costos de flete. Con un sistema automatizado, puede aumentar la productividad y reducir los costos al reducir las correcciones de errores y de entrada manual de datos. Debido a una mayor dependencia de los procesos automatizados, puede reducir otros costos de comunicaciones.
