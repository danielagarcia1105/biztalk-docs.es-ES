# <a name="how-biztalk-accelerator-for-rosettanet-works"></a>Cómo funciona el Acelerador de BizTalk para RosettaNet
Microsoft® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] proporciona una solución de fabricación de alta tecnología mediante la creación de Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)]. Para ello, que proporciona compatibilidad con RosettaNet Implementation Framework (RNIF) y procesos de interfaz de socio (PIP) y agregar los componentes que personalización el mensaje de procesamiento a las aplicaciones de fabricación de alta tecnología.
