# <a name="generating-and-enabling-certificates"></a>Generar y habilitar certificados
Este tutorial utiliza certificados para firmar y cifrar mensajes enviados por las organizaciones de Contoso y Fabrikam. Si ya ha completado el Tutorial de doble acción, puede omitir este paso y pase a [crear la solución de Contoso](../../adapters-and-accelerators/accelerator-rosettanet/creating-the-contoso-solution.md). Para generar y usar certificados para este tutorial, siga los procedimientos descritos en los temas siguientes:  
  
-   [Paso 1: Crear una entidad de certificación](../../adapters-and-accelerators/accelerator-rosettanet/step-1-creating-a-certification-authority.md)  
  
-   [Paso 2: Crear Public y Private certificados](../../adapters-and-accelerators/accelerator-rosettanet/step-2-creating-public-and-private-certificates.md)  
  
-   [Paso 3: Importar Public y Private certificados](../../adapters-and-accelerators/accelerator-rosettanet/step-3-importing-public-and-private-certificates.md)  
  
-   [Paso 4: Habilitar SSL en IIS](../../adapters-and-accelerators/accelerator-rosettanet/step-4-enabling-secure-sockets-layer-in-iis.md)  
  
> [!IMPORTANT]
>  Estos pasos se encuentran en el Tutorial de doble acción. Después de completar cada paso, volver a este tema para seguir el Tutorial de procesos privado.
