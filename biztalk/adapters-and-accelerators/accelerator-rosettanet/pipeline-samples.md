# <a name="pipeline-samples"></a>Ejemplos de canalizaciones
The Microsoft® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)] Kit de desarrollo de Software (SDK) incluye un ejemplo de canalización de envío y un ejemplo de canalización de recepción. Esta sección proporciona información acerca de la funcionalidad que muestra cada ejemplo, instrucciones para compilar y ejecutar los ejemplos y los resultados que puede esperar.  
  
 Para crear una canalización personalizada, agregue referencias al proyecto de canalización para los Rnpip, RNIFv11 y RNIFv201 ensamblados de esquema que se encuentran en Microsoft. Espacio de nombres Solutions.BTARN.Schemas. Para obtener más información acerca de cómo crear canalizaciones personalizadas, vea "canalizaciones ([!INCLUDE[btsBizTalkServerNoVersion](../../includes/btsbiztalkservernoversion-md.md)] carpeta de ejemplos)" en la Ayuda de BizTalk Server.
