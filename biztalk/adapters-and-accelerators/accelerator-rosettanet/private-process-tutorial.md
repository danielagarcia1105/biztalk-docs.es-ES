# <a name="private-process-tutorial"></a>Tutorial de procesos privados
Este tutorial contiene una solución completa de extremo a otro mediante Microsoft® [!INCLUDE[BTARN_CurrentVersion_FirstRef](../../includes/btarn-currentversion-firstref-md.md)]. El tutorial detalla los pasos que se deben seguir para implementar una solución compatible con RosettaNet mediante la creación de un escenario comercial entre dos empresas ficticias: Contoso, la organización del proveedor y Fabrikam, la organización del comprador.  
  
 El escenario que implementa este tutorial usa el precio de 3A2 y el proceso de interfaz de socio (PIP) de disponibilidad. Antes de una compra, el comprador, Fabrikam, envía una 3A2 precio y una solicitud de disponibilidad al proveedor, Contoso. Este PIP permite a Fabrikam obtener información sobre un determinado producto que pueden, a continuación, se procesan a través de las reglas de negocios para tomar una decisión sobre si se debe comprar el producto. La siguiente ilustración muestra el patrón de comunicación entre el iniciador y Respondedor organizaciones durante un intercambio PIP 3A2.  
  
 ![&#60;Ningún cambio&#62;](../../adapters-and-accelerators/accelerator-rosettanet/media/rn3-intro-eetut-3a2flow.gif "RN3_Intro_EETut_3A2Flow")  
  
 El objetivo de este tutorial se encuentra en la solución de Contoso. Describe distintos aspectos de la creación de una solución basada en RosettaNet, incluida la integración de ERP, la aplicación de directivas empresariales, personalización de procesos privado y promover una comunicación segura.
