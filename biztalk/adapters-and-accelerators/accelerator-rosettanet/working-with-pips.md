# <a name="working-with-pips"></a>Trabajar con PIP
Los temas de esta sección describen cómo trabajar con procesos de interfaz de socio (PIP). PIP define procesos empresariales entre socios comerciales, proporciona definiciones de procesos empresariales comunes para todos los intercambios de mensajes de RosettaNet. PIP especifica el contenido del servicio de mensajes.  
  
 Para obtener más información sobre PIP, consulte [PIP implementación](../../adapters-and-accelerators/accelerator-rosettanet/pip-implementation.md).
