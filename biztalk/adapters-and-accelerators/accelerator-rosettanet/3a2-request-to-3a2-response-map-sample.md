# <a name="3a2-request-to-3a2-response-map-sample"></a>Solicitud de 3A2 al ejemplo de asignación de respuesta de 3A2
El ejemplo _3A2RequestMessageTo3A2ResponseMessage.btm muestra cómo se puede asignar un mensaje de solicitud de 3A2 a un mensaje de respuesta 3A2.  
  
 De forma predeterminada, el [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]® programa de instalación de BizTalk Server instala el ejemplo en \< *unidad*\>: \Program archivos\\ [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] Microsoft BizTalk \<versión \> Accelerator for RosettaNet\SDK\PIPAutomation\DoubleAction.  
  
## <a name="sample-contents"></a>Contenido del ejemplo  
 Este ejemplo muestra cómo asignar los campos de un mensaje de solicitud de 3A2 a un mensaje de respuesta 3A2. Puede utilizar este ejemplo con el ejemplo de orquestación de PIPAutomation de doble acción. El ejemplo de orquestación de PIPAutomation de doble acción, muestra cómo implementar una orquestación para generar automáticamente las respuestas de doble acción 0c2, 0c4, 3A2 y 3A4 PIP.
