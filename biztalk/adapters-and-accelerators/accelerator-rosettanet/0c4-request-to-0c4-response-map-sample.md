# <a name="0c4-request-to-0c4-response-map-sample"></a>Solicitud de 0c4 al ejemplo de asignación de respuesta en 4 0c
El ejemplo de _0C4RequestMessageTo0C4ResponseMessage.btm muestra cómo puede asignar un mensaje de solicitud 4 0c a un mensaje de respuesta en 4 0c.  
  
 De forma predeterminada, el programa de instalación de Microsoft® BizTalk Server instala en el ejemplo de \< *unidad*\>: \Program archivos\\Microsoft BizTalk \<versión\> Acelerador para RosettaNet\SDK\PIPAutomation\DoubleAction.  
  
## <a name="sample-contents"></a>Contenido del ejemplo  
 Este ejemplo muestra cómo asignar los campos de un mensaje de solicitud 4 0c a un mensaje de respuesta en 4 0c. Puede usar este ejemplo con el ejemplo de orquestación PIPAutomation de acción doble. El ejemplo de orquestación PIPAutomation de acción doble muestra cómo implementar una orquestación para generar automáticamente las respuestas de doble acción 0c2, 0c4, 3A2 y 3A4 PIP.
