# <a name="0c4-request-to-0c4-response-map-sample"></a>0c4 la solicitud de ejemplo de asignación de respuesta en 4 0c
El ejemplo _0C4RequestMessageTo0C4ResponseMessage.btm muestra cómo se puede asignar un mensaje de solicitud 4 0c a un mensaje de respuesta en 4 0c.  
  
 De forma predeterminada, el [!INCLUDE[btsCoName](../../includes/btsconame-md.md)]® [!INCLUDE[btsBizTalkServer2006r3](../../includes/btsbiztalkserver2006r3-md.md)] programa de instalación instala el ejemplo en \< *unidad*>: \Program archivos\\ [!INCLUDE[btsCoName](../../includes/btsconame-md.md)] BizTalk \<versión > Accelerator for RosettaNet\SDK\PIPAutomation\DoubleAction.  
  
## <a name="sample-contents"></a>Contenido del ejemplo  
 Este ejemplo muestra cómo asignar los campos de un mensaje de solicitud 4 0c a un mensaje de respuesta en 4 0c. Puede utilizar este ejemplo con el ejemplo de orquestación de PIPAutomation de doble acción. El ejemplo de orquestación de PIPAutomation de doble acción, muestra cómo implementar una orquestación para generar automáticamente las respuestas de doble acción 0c2, 0c4, 3A2 y 3A4 PIP.
