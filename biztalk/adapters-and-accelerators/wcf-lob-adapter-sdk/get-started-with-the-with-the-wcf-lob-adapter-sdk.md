# <a name="get-started-with-the-with-the-wcf-lob-adapter-sdk"></a>Introducción a la con el SDK de adaptador LOB de WCF
Esta sección contiene información relevante para los usuarios que están familiarizados con el [!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]. Contiene temas sobre la instalación, requiere habilidades y conocimientos, tareas comunes de programadores, recursos de la Comunidad disponibles y tutoriales.  

## <a name="prerequisites"></a>Requisitos previos

El [!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)] está destinada a desarrolladores que necesitan integrar sistemas de línea de negocio dinámicos y complejos como parte de un esfuerzo de integración de aplicaciones de empresa. Puede descargar la versión más reciente de la [!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)] en [ http://go.microsoft.com/fwlink/?LinkID=189401 ](http://go.microsoft.com/fwlink/?LinkID=189401).  
  
 Para crear un adaptador mediante el [!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)], le sugerimos que se tienen experiencia con las siguientes tecnologías:  
  
- [!INCLUDE[btsVStudioNet](../../includes/btsvstudionet-md.md)] y el desarrollo de soluciones .NET  
  
- Programación con .NET Framework  
  
- [!INCLUDE[firstref_btsWinCommFoundation](../../includes/firstref-btswincommfoundation-md.md)]  
  
- API de programación XML de .NET  
  
- Lenguaje de descripción de servicios Web (WSDL) y el esquema XML  
  
  También debe estar familiarizado con la operación del sistema de destino, así como la tecnología que se utiliza para exponer la API del sistema de destino.
