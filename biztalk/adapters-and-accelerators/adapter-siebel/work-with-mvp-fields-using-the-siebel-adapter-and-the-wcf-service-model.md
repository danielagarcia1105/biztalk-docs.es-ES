# <a name="run-operations-on-business-components-with-mvg-fields-with-the-siebel-adapter-using-the-wcf-service-model"></a>Ejecute las operaciones en los componentes empresariales con campos de MVG con el adaptador de Siebel mediante el modelo de servicio de WCF
La [!INCLUDE[adaptersiebel](../../includes/adaptersiebel-md.md)] expone las operaciones para asociar, desasociar y consultar los registros secundarios para los componentes empresariales que tienen campos de grupo de varios valores (MVG). Un ejemplo que muestra cómo trabajar con campos MVG en el modelo de servicio WCF se incluye en los ejemplos que se proporcionan con el [!INCLUDE[adaptersiebel_short](../../includes/adaptersiebel-short-md.md)]. Para obtener más información, consulte [ejemplos para el adaptador de Siebel](../../adapters-and-accelerators/adapter-siebel/samples-for-the-siebel-adapter.md).
