# <a name="key-features-in-biztalk-adapter-for-oracle-e-business-suite"></a>Características clave en el adaptador de BizTalk para Oracle E-Business Suite
Esta sección enumeran las características clave en [!INCLUDE[adapteroracleebusinesslong](../../includes/adapteroracleebusinesslong-md.md)].  
  
## <a name="technology-related-features"></a>Características relacionadas con la tecnología  
  
|                                                                           Característica                                                                            |                                                                                                                                                                                                                                                                                                                                                                       Comentario                                                                                                                                                                                                                                                                                                                                                                       |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|                                                        Uso de Windows Communication Foundation (WCF)                                                         |                                                                                                    El [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] se basa en la parte superior de la [!INCLUDE[afproductnamelong](../../includes/afproductnamelong-md.md)] ([!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)]). A su vez, el [!INCLUDE[afproductnameshort](../../includes/afproductnameshort-md.md)] se basa en WCF. Los adaptadores se exponen como canales WCF para los clientes del adaptador. Esto permite la conectividad, el intercambio de metadatos y el intercambio de datos empresariales con sistemas externos.                                                                                                    |
|                                                 Compatibilidad con el modelo de canal WCF y los modelos de servicio WCF                                                 |                                                                                                                                                En WCF *canal* modelo, los clientes del adaptador pueden consumir el [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] directamente enviando y recibiendo mensajes XML.<br /><br /> En WCF *servicio* modelo, los clientes del adaptador pueden generar una clase de proxy de .NET desde el lenguaje de descripción de servicios de Web (WSDL) obtenidos mediante [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)].                                                                                                                                                |
|                                                                        Uso de ODP.NET                                                                        |                                                                                                                                                                                                                                                                               El [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] usa el proveedor de datos de Oracle para .NET (ODP.NET) para interactuar con Oracle E-Business Suite.                                                                                                                                                                                                                                                                               |
|                                                      Dos formas de conectarse a Oracle E-Business Suite                                                       | Los clientes del adaptador pueden conectarse a Oracle E-Business Suite mediante el nombre de servicio de red en el archivo tnsnames.ora o especificar directamente los parámetros de conexión y, lo que elimina la necesidad de usar un nombre de servicio de red o el archivo tnsnames.ora. No requiere el archivo tnsnames.ora para conectarse a Oracle E-Business Suite le evita la incomodidad de actualización manual de los parámetros de conexión (nombre de servicio de red) en el archivo tnsnames.ora en todos los equipos cliente al agregar o actualizar los servidores de Oracle en su entorno. Para obtener más información, consulte [configurar conexión a Oracle E-Business Suite](../../adapters-and-accelerators/adapter-oracle-ebs/configure-the-connection-uri-for-oracle-e-business-suite.md). |
|                                                              Compatibilidad con la autenticación de Windows                                                              |                                                       Los clientes del adaptador pueden usar la autenticación de Windows para conectarse a Oracle E-Business Suite. Autenticación de Windows le permite determinar la identidad del usuario según las credenciales de inicio de sesión de Windows y, por tanto, le ayuda a aprovechar la seguridad del entorno de Windows integrada. Para obtener más información sobre la autenticación de Windows en el [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)], consulte [conectarse a Oracle E-Business Suite Using Windows Authentication](../../adapters-and-accelerators/adapter-oracle-ebs/connect-to-oracle-e-business-suite-using-windows-authentication.md).                                                        |
| Compatibilidad para usar el [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] con Microsoft Office SharePoint Server (MOSS) |                                                                                                                                                                                                                                              Puede usar los adaptadores para presentar los datos de Oracle E-Business Suite en un portal MOSS. Para obtener más información, consulte [con el adaptador de Oracle E-Business con Microsoft Office SharePoint Server](https://msdn.microsoft.com/library/dd788485.aspx).                                                                                                                                                                                                                                               |
  
## <a name="metadata-related-features"></a>Características relacionadas con metadatos  
  
|Característica|Comentario|  
|-------------|-------------|  
|Recuperación por lotes de metadatos|Los clientes del adaptador pueden examinar y buscar los metadatos para todos los artefactos expuestas por Oracle E-Business Suite y la base de datos subyacente. Los artefactos se muestran en función de las credenciales del usuario conectado y son:<br /><br /> -Se agrupan por cada aplicación para los artefactos de Oracle E-Business Suite.<br />-Se agrupan por cada artefacto en Oracle E-Business Suite y la base de datos subyacente.<br />-Se agrupan por cada esquema para los artefactos de base de datos subyacente.|

## <a name="operations-related-features"></a>Características relacionadas con las operaciones  
  
|                                      Característica                                      |                                                                                                                                                                                                                                                                                Comentario                                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|          Compatibilidad con operaciones en tablas y vistas de interfaz           |                                                                                                                                                                          Los clientes del adaptador pueden realizar básicas Insert, Update, Delete y las operaciones Select en las tablas de interfaz de Oracle E-Business Suite. Sólo la operación de selección puede realizarse en las vistas de interfaz de Oracle E-Business Suite.                                                                                                                                                                           |
|                         Soporte técnico para la ejecución de las API de PL/SQL                         |                                                                                                                                                                                                               Los clientes del adaptador pueden ejecutar las API de PL/SQL de Oracle E-Business Suite. API de PL/SQL contienen procedimientos almacenados y funciones dentro de un paquete.                                                                                                                                                                                                                |
|                     Soporte técnico para la ejecución de programas simultáneos                     |                                                                                                                                                                                                                                              Los clientes del adaptador pueden ejecutar programas simultáneos en Oracle E-Business Suite.                                                                                                                                                                                                                                               |
|                        Soporte técnico para ejecutar conjuntos de solicitudes                         |                                                                                          Los clientes del adaptador pueden ejecutar conjuntos de solicitudes en Oracle E-Business Suite. Un conjunto de solicitud es un conjunto de programas simultáneos y puede usarse para ejecutar varios programas simultáneos con parámetros predefinidos. Conjuntos de solicitudes se ejecutan en fases. Para obtener más información acerca de los conjuntos de solicitud, consulte [ http://go.microsoft.com/fwlink/?LinkId=129539 ](http://go.microsoft.com/fwlink/?LinkId=129539).                                                                                          |
|           Compatibilidad con la especificación de valores en línea en la operación de inserción            | Puede usar el **InlineValue** atributo en la operación de inserción para insertar valores calculados en las tablas de interfaz y las tablas de base de datos de Oracle. Esto es un atributo opcional y está disponible para todos los registros de datos simple en una operación de inserción. Si especifica un valor para este atributo, invalida el valor especificado de un registro. Para obtener más información sobre el atributo InlineValue, consulte [operaciones en tablas y vistas de interfaz](../../adapters-and-accelerators/adapter-oracle-ebs/operations-on-interface-tables-and-interface-views.md). |
|                      Soporte técnico para las llamadas entrantes mediante sondeo                      |                                                                                                                   El [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] admite la recepción de "sondeo mensajes basados en" cambio de datos en la que el adaptador ejecuta una instrucción SQL especificada, procedimiento almacenado, función o procedimiento dentro de un paquete, recupera los datos y proporciona el resultado al cliente.                                                                                                                   |
|                   Soporte técnico para invocar funciones y procedimientos                   |                                                                                                                                      Los clientes del adaptador pueden invocar funciones y procedimientos de la base de datos de Oracle subyacente. La [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] expone funciones y procedimientos como las operaciones que se pueden realizar en la base de datos de Oracle subyacente.                                                                                                                                       |
|                         Compatibilidad para operaciones compuestas                          |                                         El [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] permite a los clientes del adaptador realizar operaciones compuestas en Oracle E-Business Suite. Una operación compuesta puede incluir cualquier número de las siguientes operaciones y en cualquier orden:<br /><br /> -Las operaciones en tablas, vistas de interfaz, tablas y vistas.<br />-Almacena los procedimientos, funciones y procedimientos de los paquetes que se exponen como operaciones en el adaptador.                                          |
|         Soporte técnico para la ejecución de instrucciones SQL arbitrarias y los bloques de PL/SQL          |                              El [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] permite a los clientes del adaptador ejecutar instrucciones SQL arbitrarias o bloques de PL/SQL con las operaciones ExecuteNonQuery, ExecuteReader y ExecuteScalar. Para obtener más información acerca de estas operaciones, consulte [compatibilidad con ExecuteNonQuery, ExecuteReader y ExecuteScalar operaciones](../../adapters-and-accelerators/adapter-oracle-ebs/support-for-executenonquery-executereader-and-executescalar-operations.md).                               |
| Compatibilidad con tipos definidos por el usuario (UDT), los parámetros booleanos y tipos de tabla de PL/SQL |                                                                                                     Los clientes del adaptador pueden admitir operaciones en los artefactos que contienen UDT y operaciones en los procedimientos almacenados y funciones que contienen los parámetros booleanos y el tipo de tabla de PL/SQL. Para obtener información sobre la compatibilidad con UDT, vea [compatibilidad con tipos de Oracle User-Defined](../../adapters-and-accelerators/adapter-oracle-ebs/support-for-oracle-user-defined-types2.md).                                                                                                      |
|                     Compatibilidad con entrada y salida de cursores REF cursor                      |                                                                                                                                                                                                                                       Los clientes del adaptador pueden operar con CURSORES REF de entrada y salida sobre los procedimientos y funciones.                                                                                                                                                                                                                                       |
|                       Compatibilidad con tipos de datos de registro de PL/SQL                        |                                                                                                                                                                                                                                  Los clientes del adaptador pueden operar con el registro y los tipos de registro anidados en procedimientos almacenados y funciones.                                                                                                                                                                                                                                   |
|           Compatibilidad con las funciones sobrecargadas y los procedimientos de la API de PL/SQL           |                                                                                   Los clientes del adaptador pueden invocar las funciones sobrecargadas y los procedimientos de las API de PL/SQL. Sin embargo, el [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] no admite el uso de procedimientos sobrecargados con cursores REF cursor de fuertemente tipadas y débilmente tipada. Internamente, el adaptador trata los CURSORES REF fuertemente tipadas y débilmente tipada como simplemente cursores REF cursor.                                                                                    |
|                     Compatibilidad con notificaciones de cambio de base de datos                     |      Los clientes del adaptador pueden recibir notificaciones de cambio de base de datos de base de datos de Oracle en función de una instrucción SELECT de desencadenamiento. La notificación se envía por base de datos de Oracle a los clientes del adaptador como y cuando el conjunto de resultados para que los cambios de la instrucción SELECT. Para obtener más información acerca de las notificaciones de cambio de base de datos, vea [consideración para recibir notificaciones base de datos cambie con el adaptador de base de datos de ORacle](../../adapters-and-accelerators/adapter-oracle-database/before-you-receive-database-change-notifications-using-the-oracle-db-adapter.md).      |
|                               Compatibilidad con sinónimos                                |                                                                     Los clientes del adaptador pueden realizar operaciones en sinónimos creados para tablas, vistas, procedimientos almacenados, funciones y paquetes. Para obtener más información acerca de sinónimos, y cómo puede usar el [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] para realizar operaciones en sinónimos, vea [operaciones en sinónimos](../../adapters-and-accelerators/adapter-oracle-ebs/operations-on-synonyms2.md).                                                                      |

| Compatibilidad con varios idioma (MLS) | Los clientes del adaptador pueden usar el [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] para realizar operaciones en Oracle E-Business Suite y la base de datos subyacente tiene varios paquetes de idioma instalados. Esto implica que los clientes del adaptador pueden realizar operaciones en los datos en varios idiomas. Por ejemplo, puede ver datos localizados de (no en inglés) en una vista de interfaz, o insertar datos localizados en una tabla. El [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] es compatible con tantos idiomas como las que admite la instalación de Oracle E-Business Suite subyacente.<br /><br /> El [!INCLUDE[adapteroraclebusinessshort](../../includes/adapteroraclebusinessshort-md.md)] expone el **MlsSettings** enlaza la propiedad para configurar la característica de compatibilidad con varios idiomas. Para obtener más información acerca de la propiedad de enlace, consulte [Obtenga información sobre el adaptador de BizTalk para propiedades de enlace de Oracle E-Business Suite](../../adapters-and-accelerators/adapter-oracle-ebs/read-about-the-biztalk-adapter-for-oracle-e-business-suite-binding-properties.md). |
