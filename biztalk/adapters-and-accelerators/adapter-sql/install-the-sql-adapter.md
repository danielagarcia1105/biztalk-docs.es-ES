# <a name="install-the-sql-adapter"></a>Instalar al adaptador de SQL
## <a name="install-options"></a>Opciones de instalación
Puede instalar el [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)] de dos maneras:

* Opción 1: Con la [!INCLUDE[adapterpacknoversion](../../includes/adapterpacknoversion-md.md)] instalación: vea [instalar el módulo de adaptador de BizTalk 2016](../../adapters-and-accelerators/install-the-biztalk-adapter-pack-2016.md), o [instalar BizTalk adaptador Pack 2013 R2 y 2013](../../adapters-and-accelerators/install-biztalk-adapter-pack-2013-r2-and-2013.md). También puede instalar a los demás adaptadores de WCF con esta opción.
  
* Opción 2: por sí mismo, si solo desea instalar el [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)] como un adaptador independiente. Para esta opción, utilice a las guías de instalación de esta sección.
  
> [!IMPORTANT]
>  El [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)] incluido con el [!INCLUDE[adapterpacknoversion](../../includes/adapterpacknoversion-md.md)]y el independiente [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)] no pueden coexistir en el mismo equipo.  Si desea que sólo el [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)], a continuación, instale el adaptador independiente. Si cree que puede que desee la [!INCLUDE[adaptersqlshort](../../includes/adaptersqlshort-md.md)], y cualquiera de los demás adaptadores basados en WCF, a continuación, utilice el [!INCLUDE[adapterpacknoversion](../../includes/adapterpacknoversion-md.md)] el programa de instalación.
