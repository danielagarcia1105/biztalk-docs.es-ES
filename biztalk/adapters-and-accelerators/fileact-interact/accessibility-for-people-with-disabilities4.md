# <a name="accessibility-for-people-with-disabilities"></a>Accesibilidad para personas con discapacidades
Microsoft se compromete a hacer que sus productos sean más fáciles de usar. Para obtener información acerca de las opciones de accesibilidad, en la Ayuda en línea de Windows 2000 Server, en la introducción a Windows 2000 del libro, vea "Accesibilidad para personas con discapacidades". En la Ayuda en pantalla de Windows 2000 Professional, vea "Accesibilidad para necesidades especiales."
