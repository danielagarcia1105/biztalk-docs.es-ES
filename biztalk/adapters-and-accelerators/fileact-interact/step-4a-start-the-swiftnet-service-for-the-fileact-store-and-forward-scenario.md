# <a name="step-4a-start-the-swiftnet-service-for-the-fileact-store-and-forward-scenario"></a>El paso 4A: iniciar el servicio SWIFTNet para el escenario de reenvío y almacenamiento de FileAct
Antes de comenzar este paso, debe completar [paso 3D: agregar un puerto de envío de FILEACT para el escenario de al día y el almacenamiento de FileAct](../../adapters-and-accelerators/fileact-interact/step-3d-add-a-fileact-send-port-for-the-fileact-store-and-forward-scenario.md).  
  
### <a name="to-start-the-swiftnet-service"></a>Para iniciar el servicio SWIFTNet  
  
-   Abrir un tipo de símbolo del sistema, de comando remoto de API **SWIFTNET iniciar**, y, a continuación, presione ENTRAR.
