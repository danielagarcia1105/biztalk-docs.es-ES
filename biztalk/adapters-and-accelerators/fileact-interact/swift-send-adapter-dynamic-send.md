# <a name="swift-send-adapter-dynamic-send"></a>Envío dinámico del adaptador de envío SWIFT
El modo de envío dinámico de adaptador de envío extrae los mensajes de la cola. El URI de envío dinámico es:  
  
```  
SWIFT://<queue name>/<Force Acquire>/<Session Mode>/<Order By>/<Recovery mode>  
```  
  
 Puede utilizar una orquestación o una canalización para construir el URI.
