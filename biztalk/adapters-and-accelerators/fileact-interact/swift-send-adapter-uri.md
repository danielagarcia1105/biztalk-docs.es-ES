# <a name="swift-send-adapter-uri"></a>URI del adaptador de envío SWIFT
El URI para el adaptador de envío es el nombre de construido Respondedor según (SWIFT: / /\<nombre de servicio de respuesta\>). Nombre de servicio de respuesta es un nombre distintivo definido mediante SWIFT. Indica a los que debe enviarse el mensaje. SWIFT usa este valor para enrutar el mensaje al destinatario.
