# <a name="step-4a-start-the-swiftnet-service-for-the-fileact-real-time-scenario"></a>El paso 4A: iniciar el servicio SWIFTNet para el escenario en tiempo real de FileAct
Antes de comenzar este paso, debe completar [paso 3E: agregar un puerto de envío de archivo para capturar Sw:ExchangeFileResponse mensaje para el escenario de en tiempo real de FileAct](../../adapters-and-accelerators/fileact-interact/step-3e-add-file-send-port-to-get-sw-exchangefileresponse-message-for-fileact.md).  
  
### <a name="to-start-the-swiftnet-service"></a>Para iniciar el servicio SWIFTNet  
  
-   Abrir un tipo de símbolo del sistema, de comando remoto de API **SWIFTNET iniciar**, y, a continuación, presione ENTRAR.
