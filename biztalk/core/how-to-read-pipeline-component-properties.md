# <a name="how-to-read-pipeline-component-properties"></a>Cómo leer las propiedades de componente de canalización
La ventana Propiedades contiene dos secciones para componentes: propiedades generales y propiedades de componentes. Las propiedades generales son comunes a todos los componentes, aunque sus valores varíen entre ellos.  
  
### <a name="to-read-the-general-properties-for-a-pipeline-component"></a>Para leer las propiedades generales de un componente de canalización  
  
1.  Arrastre el componente de canalización hasta una fase de la canalización o seleccione un componente si ya existe en la canalización.  
  
2.  En la ventana Propiedades, en la **General** sección, realice lo siguiente.  
  
    |Use|Para|  
    |--------------|----------------|  
    |**Nombre**|Indica el nombre del componente.|  
    |**Ensamblado**|Indica el ensamblado y la información .NET asociada para el componente.|  
    |**Description**|Indica el nombre descriptivo del componente de canalización.|  
    |**Administrado**|Indica si el componente de canalización está administrado o no. **Sí** si el componente es un componente administrado. NET; **n** si el componente de canalización es un componente COM.|  
    |**Ruta de acceso**|Indica la ruta de acceso completa al componente .NET.|  
    |**Tipo**|Indica el tipo de componente, el ensamblado y la información .NET asociada para el componente.|
