# <a name="single-sign-on-event-10780"></a>De sesión único: Evento 10780
## <a name="details"></a>Detalles  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  Nombre del producto   |                 Inicio de sesión único (SSO) empresarial                  |
| Versión del producto | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    Identificador del evento     |                           10780                            |
|  Origen del evento   |                           ENTSSO                           |
|    Componente    |                            N/D                             |
|  Nombre simbólico  |                  ENTSSO_E_INVALID_ACCOUNT                  |
|  Texto del mensaje   |      Nombre de cuenta no válido o inexistente.      |
