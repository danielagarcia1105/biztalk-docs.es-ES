# <a name="troubleshooting-biztalk-server-permissions"></a>Solucionar problemas de permisos de BizTalk Server
Los problemas de permisos en BizTalk Server suelen pertenecer a una de estas categorías:  
  
- Permisos de SQL Server  
  
- Permisos de Active Directory en instalaciones multiservidor de BizTalk Server  
  
- Permisos de servicios Web  
  
- Permisos IIS  
  
  Este tema proporciona directrices generales para evitar los problemas de permisos, así como pasos para resolver los problemas en ciertas situaciones.
