# <a name="how-to-reconnect-to-a-group"></a>Cómo volver a conectarse a un grupo
Un grupo que había conectado anteriormente al servidor BizTalk Server puede aparecer, en ocasiones, como no disponible en la consola de administración de BizTalk Server. Por ejemplo, un grupo puede dejar de estar disponible para la consola de administración cuando se reinicia el equipo donde reside el grupo. Puede volver a conectarse a un grupo de BizTalk Server no está disponible mediante el uso de **actualizar** si el grupo no está disponible.  
  
## <a name="prerequisites"></a>Requisitos previos  
 Para llevar a cabo este procedimiento, debe iniciar sesión como miembro del grupo de operadores de BizTalk Server o de administradores de BizTalk Server.  
  
### <a name="to-reconnect-to-a-group"></a>Para volver a conectarse a un grupo  
  
1.  Haga clic en **iniciar**, haga clic en **todos los programas**, haga clic en [!INCLUDE[btsBizTalkServerStartMenuItemui](../includes/btsbiztalkserverstartmenuitemui-md.md)]y, a continuación, haga clic en **administración de BizTalk Server**.  
  
2.  En el árbol de consola, haga doble clic en [!INCLUDE[btsBizTalkServerAdminConsoleui](../includes/btsbiztalkserveradminconsoleui-md.md)].  
  
3.  Haga clic en el **(no disponible)** de grupo y, a continuación, haga clic en **actualizar**.
