# <a name="single-sign-on-event-10840"></a>De sesión único: Evento 10840
## <a name="details"></a>Detalles  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  Nombre del producto   |                 Inicio de sesión único (SSO) empresarial                  |
| Versión del producto | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    Identificador del evento     |                           10840                            |
|  Origen del evento   |                           ENTSSO                           |
|    Componente    |                            N/D                             |
|  Nombre simbólico  |               ENTSSO_E_MAX_PASSWORD_REMINDER               |
|  Texto del mensaje   |  La longitud máxima del aviso de contraseña es de 512 caracteres.   |
  
## <a name="explanation"></a>Explicación  
 La longitud máxima del aviso de contraseña es de 512 caracteres.  
  
## <a name="user-action"></a>Acción del usuario  
 Use un aviso que respete estas pautas.
