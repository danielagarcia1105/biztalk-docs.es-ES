# <a name="managing-servers"></a>Administrar servidores
El nodo Servidores de la consola de administración de BizTalk Server muestra una lista de todos los servidores que pertenecen al grupo de BizTalk Server. Siga los procedimientos que se describen en esta sección para agregar o quitar servidores del grupo, o para mover servidores de un grupo a otro.

-   [Cómo quitar un servidor de un grupo](../core/how-to-remove-a-server-from-a-group.md)
