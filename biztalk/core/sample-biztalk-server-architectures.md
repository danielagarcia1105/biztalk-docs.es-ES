# <a name="sample-biztalk-server-architectures"></a>Arquitecturas de ejemplo de BizTalk Server
Para proporcionar la mayor seguridad posible al entorno de Microsoft® BizTalk® Server, se recomienda utilizar una arquitectura en la que los servidores de BizTalk se distribuyan en función de la importancia de los datos y servicios que contienen, y de las consecuencias de un eventual ataque al servidor.  
  
 Esta sección contiene información para ayudarle a crear una implementación segura de BizTalk Server que le permita afrontar los retos y cubrir las necesidades concretas de la organización y el entorno.
