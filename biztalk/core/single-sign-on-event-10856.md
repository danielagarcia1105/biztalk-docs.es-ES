# <a name="single-sign-on-event-10856"></a>De sesión único: Evento 10856
## <a name="details"></a>Detalles  
  
|                 |                                                                |
|-----------------|----------------------------------------------------------------|
|  Nombre del producto   |                   Inicio de sesión único (SSO) empresarial                    |
| Versión del producto |   [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]   |
|    Identificador del evento     |                             10856                              |
|  Origen del evento   |                             ENTSSO                             |
|    Componente    |                              N/D                               |
|  Nombre simbólico  |               ENTSSO_E_MAPPING_CREATE_RESTRICTED               |
|  Texto del mensaje   | Usuarios de aplicación no puede crear asignaciones para esta aplicación. |
  
## <a name="explanation"></a>Explicación  
 Esta aplicación se configuró de manera que no se permita crear asignaciones a los miembros del grupo Usuarios de aplicación.  
  
## <a name="user-action"></a>Acción del usuario  
 Para crear estas asignaciones, debe solicitárselo a una persona con permisos suficientes.
