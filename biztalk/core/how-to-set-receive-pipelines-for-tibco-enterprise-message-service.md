# <a name="how-to-set-receive-pipelines-for-tibco-enterprise-message-service"></a>Cómo configurar canalizaciones de recepción para TIBCO Enterprise Message Service
El adaptador de Microsoft BizTalk para TIBCO Enterprise Message Service requiere que configure el controlador de recepción y la canalización de recepción.  
  
### <a name="to-set-the-pipeline"></a>Procedimiento para establecer la canalización  
  
1.  En el **controlador de recepción** lista, seleccione **BizTalkServerApplication**.  
  
2.  En el **canalización de recepción** , seleccione la canalización que desee.  
  
3.  Haga clic en **Aceptar**.
