# <a name="sql-adapter"></a>Adaptador SQL
[!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] ya no admite SQL Adapter. Se recomienda usar [!INCLUDE[adaptersql](../includes/adaptersql-md.md)] para conseguir la misma funcionalidad.  
  
 Sin embargo, [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] admite SQL Adapter con los componentes enviados junto con [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)], como la canalización de recepción SQL de mensajes de control EDI o la canalización de la recepción de mensajes de control de reenvío AS2 .
