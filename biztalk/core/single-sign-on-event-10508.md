# <a name="single-sign-on-event-10508"></a>De sesión único: Evento 10508
## <a name="details"></a>Detalles  

|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  Nombre del producto   |                 Inicio de sesión único (SSO) empresarial                  |
| Versión del producto | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    Identificador del evento     |                           10508                            |
|  Origen del evento   |                           ENTSSO                           |
|    Componente    |                            N\D                             |
|  Nombre simbólico  |                 SSO_INFO_SERVICE_REMOVE_OK                 |
|  Texto del mensaje   |                Se ha quitado el servicio SSO.                |

## <a name="explanation"></a>Explicación  
 Este evento de información indica que se quitó el servicio SSO.
