# <a name="using-variables-in-orchestrations"></a>Usar variables en orquestaciones
El Diseñador de orquestaciones proporciona funciones para administrar los datos de una orquestación. Puede definir los tipos de datos en la ventana Vista orquestación, además de crear mensajes, variables y parámetros en la ventana Vista orquestación, y utilizar el Editor de expresiones de BizTalk para agregar lógica con el fin de manipular y probar los valores.
