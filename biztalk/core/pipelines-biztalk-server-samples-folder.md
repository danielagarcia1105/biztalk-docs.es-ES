# <a name="pipelines-biztalk-server-samples-folder"></a>Canalizaciones (carpeta de ejemplos de BizTalk Server)
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] incluye varios ejemplos de canalizaciones en su kit de desarrollo de software (SDK). Esta sección proporciona información detallada acerca de la funcionalidad que muestra cada ejemplo de canalización, instrucciones para la generación y ejecución del ejemplo y los resultados que se pueden esperar.  
  
## <a name="in-this-section"></a>En esta sección  
  
-   [Agregador (ejemplo de BizTalk Server)](../core/aggregator-biztalk-server-sample.md). Muestra la funcionalidad de agregación de mensaje con orquestaciones y canalizaciones.  
  
-   [Controlador de propiedad XPath arbitrario (ejemplo de BizTalk Server)](../core/arbitrary-xpath-property-handler-biztalk-server-sample.md). Muestra cómo escribir un componente de canalización personalizado para promover propiedades específicas de un documento XML enviado a BizTalk Server. La promoción de la propiedad se consigue por medio de una expresión XPath.  
  
-   [Las canalizaciones-AssemblerDisassembler (carpeta de ejemplos de BizTalk Server)](../core/pipelines-assemblerdisassembler-biztalk-server-samples-folder.md)  
  
    -   [FlatFileReceive (ejemplo de BizTalk Server)](../core/flatfilereceive-biztalk-server-sample.md). Muestra cómo procesar mensajes de archivo sin formato delimitado en mensajes XML equivalentes mediante el uso de canalizaciones de BizTalk Server.  
  
    -   [EnvelopeProcessing (ejemplo de BizTalk Server)](../core/envelopeprocessing-biztalk-server-sample.md). Muestra cómo procesar mensajes de archivo sin formato delimitado en mensajes XML con sobres mediante el uso de canalizaciones de BizTalk Server.  
  
    -   [FlatFileSend (ejemplo de BizTalk Server)](../core/flatfilesend-biztalk-server-sample.md). Muestra cómo procesar mensajes XML en mensajes de archivo sin formato posicional equivalentes mediante el uso de canalizaciones de BizTalk Server.  
  
-   [Compone el procesador de mensajes (ejemplo de BizTalk Server)](../core/composed-message-processor-biztalk-server-sample.md). Muestra cómo procesar elementos de línea individuales de mensajes agregados mediante el procesamiento de mensajes compuestos.  
  
-   [CustomComponent (ejemplo de BizTalk Server)](../core/customcomponent-biztalk-server-sample.md). Muestra cómo crear y utilizar un componente de canalización personalizado que modifique un mensaje transmitido.  
  
-   [Resolución de entidades personalizadas (ejemplo de BizTalk Server)](../core/custom-party-resolution-biztalk-server-sample.md). Muestra cómo escribir un componente de canalización personalizado para resolver una entidad personalizada.  
  
-   [MIME (ejemplo de BizTalk Server)](../core/mime-biztalk-server-sample.md). Muestra cómo se pueden codificar mensajes por medio de MIME con canalizaciones de BizTalk.  
  
-   [Componente de resolución de esquema (ejemplo de BizTalk Server)](../core/schema-resolver-component-biztalk-server-sample.md). Muestra cómo extender la funcionalidad del componente de desensamblador de archivos sin formato de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)].  
  
-   [Transformación de XSLT (ejemplo de BizTalk Server) del componente](../core/xslt-transform-component-biztalk-server-sample.md). Muestra cómo escribir un componente de canalización personalizado para transformar un mensaje XML por medio de XSLT.
