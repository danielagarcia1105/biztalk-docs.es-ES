# <a name="system-reference"></a>Referencia del sistema
El **System** espacio de nombres de referencia contiene clases fundamentales y clases base que definen normalmente utilizan tipos de datos de valor y de referencia, eventos y controladores de eventos, interfaces, atributos y excepciones de procesamiento. Esta referencia predeterminada es necesaria para que todos los proyectos de BizTalk funcionen adecuadamente. No elimine la **System** referencia.  
  
 Para obtener más información sobre la **System** hacen referencia, vea "Sistema Namespace" en la colección combinada de Studio Visual en [http://go.microsoft.com/fwlink/?LinkId=193568](http://go.microsoft.com/fwlink/?LinkId=193568).
