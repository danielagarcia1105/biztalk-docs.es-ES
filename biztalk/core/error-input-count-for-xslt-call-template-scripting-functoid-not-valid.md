# <a name="error---input-count-for-xslt-call-template-scripting-functoid-not-valid"></a>Error: el número de entrada para XSLT llamada plantilla Functoid de script no es válido
**Código de error**  
  
 btm1076  
  
 **Explicación**  
  
 El número de parámetros de entrada a la correspondiente **secuencias de comandos** functoid, configurado para usar una plantilla de llamada XSLT de en línea no coincide con el número de parámetros que ésta esperaba.  
  
 **Acción del usuario**  
  
 Asegúrese de que el número de parámetros de entrada a este **secuencias de comandos** functoid coincide con el número de parámetros esperados por la plantilla de llamada XSLT configurado para este alineada **secuencias de comandos** functoid.
