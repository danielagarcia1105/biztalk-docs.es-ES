# <a name="error---generic-native-serialization"></a>Error: serialización nativa genérica
**Código de error**  
  
 btm1049  
  
 **Explicación**  
  
 El mensaje de instancia de salida XML generado por la transformación que especificó la asignación no se pudo convertir al formato nativo especificado por el esquema de destino debido a un error no especificado.  
  
 **Acción del usuario**  
  
 Abra el esquema de destino en el Editor de BizTalk y utilice la **Validar esquema**, **Validar instancia**, y **generar instancia** comandos, que aparecen cuando hace clic en un esquema en el Explorador de soluciones, para aislar el problema.
