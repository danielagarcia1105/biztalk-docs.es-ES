# <a name="error---duplicate-distinguished-field-promotion"></a>Error: promoción de campo distintivo duplicado
**Código de error**  
  
 BEC2015  
  
 **Explicación**  
  
 El nodo indicado en este esquema se está promocionando más de una vez como campo distintivo.  
  
 **Acción del usuario**  
  
 Use el **campos distintivos** pestaña de la **promocionar propiedades** cuadro de diálogo, tiene acceso a través de la **promocionar propiedades** propiedad de la **esquema** nodo, para eliminar todo excepto una de las promociones de campo distintivo que están asociadas con el mismo nodo en este esquema.
