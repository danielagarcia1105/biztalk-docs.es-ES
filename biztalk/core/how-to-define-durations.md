# <a name="how-to-define-durations"></a>Definición de duraciones
La duración especifica la cantidad de tiempo, definida mediante un hito inicial y otro final, de un elemento de actividad dado. Por ejemplo, el número de días de validez de un pedido.  
  
 Use el Asistente para vistas de BAM para establecer o modificar duraciones. Antes de establecer duraciones, use el Asistente para actividad de BAM para crear actividades y elementos empresariales. Para obtener más información sobre cómo utilizar el asistente, consulte [definir actividades económicas y vistas en Excel](../core/defining-business-activities-and-views-in-excel.md).  
  
### <a name="to-define-durations"></a>Procedimiento para definir duraciones  
  
1.  En el Asistente para la vista de BAM, haga clic en **siguiente** hasta que aparezca la segunda **Editar vista de BAM: elementos de vista** página y, a continuación, haga clic en **nueva duración**.  
  
    > [!NOTE]
    >  Debe tener al menos dos hitos o grupos para poder crear una duración.  
  
2.  Escriba un nombre para **nombre de duración**.  
  
3.  En la lista desplegable, seleccione **hito económico de inicio**, **hito económico de fin**, **resolución temporal** y, a continuación, haga clic en **Aceptar**.
