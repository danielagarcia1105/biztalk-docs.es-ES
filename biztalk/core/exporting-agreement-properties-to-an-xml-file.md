### <a name="to-export-agreement-properties-to-an-xml-file"></a><span data-ttu-id="85404-106">Procedimiento para exportar las propiedades de acuerdo a un archivo XML</span><span class="sxs-lookup"><span data-stu-id="85404-106">To export agreement properties to an XML file</span></span>  
  
1.  <span data-ttu-id="85404-107">En la consola de administración de BizTalk Server, haga clic en el **partes** nodo bajo el **administración de BizTalk Server** y **grupo de BizTalk** nodos.</span><span class="sxs-lookup"><span data-stu-id="85404-107">In the BizTalk Server Administration Console, click the **Parties** node under the **BizTalk Server Administration** and **BizTalk Group** nodes.</span></span> <span data-ttu-id="85404-108">En el **entidades y perfiles empresariales** página, haga clic en una entidad y, a continuación, en el perfil de negocio que tenga el acuerdo que desea exportar la configuración de.</span><span class="sxs-lookup"><span data-stu-id="85404-108">In the **Parties and Business Profiles** page, click a party and then the business profile that has the agreement you want to export the settings from.</span></span>  
  
2.  <span data-ttu-id="85404-109">En el **entidades y perfiles empresariales** página, en la **contratos** sección, haga clic en el acuerdo que desea exportar las propiedades y haga clic en **propiedades**.</span><span class="sxs-lookup"><span data-stu-id="85404-109">On the **Parties and Business Profiles** page, in the **Agreements** section, right-click the agreement you want to export properties from and click **Properties**.</span></span>

4.  <span data-ttu-id="85404-111">En el **Guardar como** cuadro de diálogo, vaya a la ubicación donde desea exportar el archivo XML para proporcionar un nombre para el archivo XML y, a continuación, haga clic en **guardar**.</span><span class="sxs-lookup"><span data-stu-id="85404-111">In the **Save As** dialog box, browse to the location where you want to export the XML file to, provide a name for the XML file, and then click **Save**.</span></span>  
  
5.  <span data-ttu-id="85404-112">En el **crear plantilla** cuadro, haga clic en **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="85404-112">In the **Create Template** box, click **OK**.</span></span>
