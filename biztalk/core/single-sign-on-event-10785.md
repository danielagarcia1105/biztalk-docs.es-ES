# <a name="single-sign-on-event-10785"></a>De sesión único: Evento 10785
## <a name="details"></a>Detalles  
  
|                 |                                                                |
|-----------------|----------------------------------------------------------------|
|  Nombre del producto   |                   Inicio de sesión único (SSO) empresarial                    |
| Versión del producto |   [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]   |
|    Identificador del evento     |                             10785                              |
|  Origen del evento   |                             ENTSSO                             |
|    Componente    |                              N/D                               |
|  Nombre simbólico  |                       ENTSSO_E_DB_ACCESS                       |
|  Texto del mensaje   | Error al intentar obtener acceso a la base de datos de SSO. |
  
## <a name="explanation"></a>Explicación  
 Es posible que la base de datos de ENTSSO esté desconectada.  
  
## <a name="user-action"></a>Acción del usuario  
 Solicite al administrador del sistema que compruebe el registro de eventos en el servidor de ENTSSO.
