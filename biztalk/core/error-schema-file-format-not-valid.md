# <a name="error---schema-file-format-not-valid"></a>Error: formato de archivo de esquema no válido
**Explicación**  
  
 El archivo de esquema no contiene una sintaxis del lenguaje de definición de esquemas XML (XSD) válida.  
  
 **Acción del usuario**  
  
 Compruebe el origen del archivo de esquema para determinar la causa del problema.
