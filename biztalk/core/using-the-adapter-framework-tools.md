# <a name="using-the-adapter-framework-tools"></a>Uso de las herramientas de marco de trabajo de adaptadores
El marco de trabajo de adaptadores de BizTalk utiliza las siguientes herramientas y vistas para administrar y desarrollar adaptadores personalizados.  
  
- **Consola de administración de BizTalk Server.** Se trata de una aplicación independiente disponible desde el **iniciar** menú. Se usa para administrar prácticamente cualquier aspecto sobre su [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] las instalaciones. Permite instalar nuevos adaptadores, administrar sus controladores de recepción y envío asociados, así como administrar puertos, ensamblados y orquestaciones.  
  
- **Metadatos de Asistente para agregar adaptador.** Este asistente está disponible en un proyecto de BizTalk en Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]. Está disponible con el botón secundario en la ventana Explorador de soluciones en el nombre del proyecto [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)]y, a continuación, haga clic en **agregar elementos generados**. El Asistente para agregar metadatos de adaptador está disponible para los programadores que deseen realizar comunicaciones con adaptadores de aplicación. Use este asistente para insertar esquemas correspondientes a estos adaptadores de aplicación en el proyecto.  
  
- **Páginas de propiedades del adaptador.** Están disponibles a través de la [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] consola de administración. Use estas páginas para configurar controladores de recepción, ubicaciones de recepción, controladores de envío y puertos de envío para el adaptador.
