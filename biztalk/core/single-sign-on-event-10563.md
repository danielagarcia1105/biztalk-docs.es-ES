# <a name="single-sign-on-event-10563"></a>Inicio de sesión único: Evento 10563
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10563|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|SSO_WARN_PERFMON_FAILED|  
|Texto del mensaje|No se pudo iniciar la supervisión del rendimiento.%r<br /><br /> Código de error: %1|  
  
## <a name="explanation"></a>Explicación  
 No se pudo iniciar la supervisión del rendimiento. El sistema continuará ejecutándose normalmente pero la supervisión del rendimiento no estará disponible.  
  
## <a name="user-action"></a>Acción del usuario  
 Es posible que se deba desinstalar la utilidad PerfMon e instalarla nuevamente.
