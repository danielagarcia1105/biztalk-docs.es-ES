# <a name="password-sync-adapters"></a>Adaptadores de sincronización de contraseña
A *adaptador de sincronización de contraseña* es un componente que propaga los cambios de contraseña a y desde un sistema ajeno a Windows. Aunque los adaptadores de sincronización de contraseñas son similares a las aplicaciones tradicionales de inicio de sesión único, existen varias diferencias:  
  
-   Se administran mediante una interfaz especializada.  
  
-   Se describen mediante un formato XML especializado en el almacén de configuración.  
  
-   Host Integration Server utiliza una característica especializada para organizar los adaptadores en el almacén de configuración.
