# <a name="single-sign-on-event-10838"></a>De sesión único: Evento 10838
## <a name="details"></a>Detalles  
  
|                 |                                                                                                                     |
|-----------------|---------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                              Inicio de sesión único (SSO) empresarial                                              |
| Versión del producto |                             [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                              |
|    Identificador del evento     |                                                        10838                                                        |
|  Origen del evento   |                                                       ENTSSO                                                        |
|    Componente    |                                                         N/D                                                         |
|  Nombre simbólico  |                                          ENTSSO_E_MAX_SSO_DATABASE_CREATE                                           |
|  Texto del mensaje   | La longitud máxima de nombre de base de datos SQL que SSO puede crear es de 123 caracteres, para permitir el nombre de archivo de registro predeterminado. |
  
## <a name="explanation"></a>Explicación  
 La longitud máxima de nombre de base de datos SQL que SSO puede crear es de 123 caracteres, para permitir el nombre de archivo de registro predeterminado.  
  
## <a name="user-action"></a>Acción del usuario  
 Use un nombre que respete estas pautas.
