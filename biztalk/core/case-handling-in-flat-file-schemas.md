# <a name="case-handling-in-flat-file-schemas"></a>Caso de control en esquemas de archivo sin formato

## <a name="overview"></a>Información general
Puede usar el **caso** propiedad para realizar una conversión de datos de archivo sin formato cuando se traducen desde su formato XML equivalente. Cuando el ensamblador de archivo sin formato traduce un mensaje XML en su formato de archivo sin formato equivalente y el **caso** propiedad está establecida en **mayúsculas** o **minúsculas**, todos los datos regido por el correspondiente esquema se convertirá a mayúsculas o minúsculas, respectivamente, durante la traducción.  
  
 Cuando el **caso** propiedad está establecida en **(predeterminado)**, se realiza ninguna conversión de mayúsculas.
