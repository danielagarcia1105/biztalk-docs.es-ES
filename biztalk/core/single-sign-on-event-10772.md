# <a name="single-sign-on-event-10772"></a>De sesión único: Evento 10772
## <a name="details"></a>Detalles  
  
|                 |                                                                       |
|-----------------|-----------------------------------------------------------------------|
|  Nombre del producto   |                       Inicio de sesión único (SSO) empresarial                       |
| Versión del producto |      [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]       |
|    Identificador del evento     |                                 10772                                 |
|  Origen del evento   |                                ENTSSO                                 |
|    Componente    |                                  N/D                                  |
|  Nombre simbólico  |                        ENTSSO_E_NOT_SSO_ADMIN                         |
|  Texto del mensaje   | El usuario no es miembro de la cuenta de administradores de SSO especificada. |
  
## <a name="explanation"></a>Explicación  
 Para realizar esta función, debe ser un administrador de SSO.  
  
## <a name="user-action"></a>Acción del usuario  
 Solicite ayuda al administrador del sistema para ponerse en contacto con el Administrador de SSO.
