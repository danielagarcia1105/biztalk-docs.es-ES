# <a name="bam-and-tracking-assistance"></a>Asistencia para el seguimiento y BAM
También puede solicitar asistencia a través del portal BAM, haga clic en el **asistencia** botón en la página de estado de actividad.  
  
 La solicitud de asistencia se coloca en el registro de eventos de aplicación. El origen de las solicitudes es el servicio Web de BAM, y el Id. de evento es 1000.  
  
 El cuerpo de mensaje se encuentra en el campo Descripción, y contiene el asunto de la solicitud, el texto de la descripción de problema, el Id. de instancia de servicio de BizTalk, el Id. de mensaje, el servidor de bases de datos de administración de BizTalk y el nombre de la base de datos de administración de BizTalk.
