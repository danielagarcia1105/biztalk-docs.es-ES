# <a name="defining-a-bam-view"></a>Definir una vista de BAM
A veces, no conviene exponer todos los datos económicos propios a todos los usuarios empresariales. Puede crear distintas vistas de BAM para exponer distintos tipos de información.  
  
> [!IMPORTANT]
>  Todos los nombres de Alias, Duraciones, Grupos, Dimensiones y Medidas deben ser únicos, ya que formarán parte de la misma vista.  
  
> [!NOTE]
>  Para definir una vista de BAM, debe haber creado al menos una actividad de BAM. Para obtener más información acerca de cómo crear actividades, consulte [cómo definir una actividad económica](../core/how-to-define-a-business-activity.md).  
  
> [!NOTE]
>  Las vistas de BAM aparecen en el libro de Excel como hojas de cálculo. Puede crear muchas vistas para una actividad de BAM. Use las pestañas de hoja de cálculo de la parte inferior de la ventana de la aplicación Excel para seleccionar una vista de BAM y editarla.
