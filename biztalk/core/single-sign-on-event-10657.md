# <a name="single-sign-on-event-10657"></a>Inicio de sesión único: Evento 10657
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10657|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_PASSWORD_SYNC_ADAPTERS_STARTED|  
|Texto del mensaje|Se inició la sincronización de contraseñas para adaptadores externos.%r|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que se inició la sincronización de contraseñas para adaptadores externos.
