# <a name="single-sign-on-event-10650"></a>Inicio de sesión único: Evento 10650
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10650|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_PASSWORD_SYNC_STARTING|  
|Texto del mensaje|Los servicios de sincronización de contraseñas están disponibles para este equipo.%r|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que se está iniciando la sincronización de contraseñas.

-   No es necesario que el usuario realice ninguna acción.
