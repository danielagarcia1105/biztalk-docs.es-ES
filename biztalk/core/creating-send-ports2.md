# <a name="creating-send-ports"></a>Creación de puertos de envío
Al crear un puerto de envío que usa Inicio de sesión único (SSO) y la aplicación afiliada, no hay que especificar loo parámetros Nombre de usuario y Contraseña. Seleccionar **Sí** para **usar SSO**, y, a continuación, seleccione la aplicación afiliada SSO que creó para representar al sistema TIBCO Rendezvous en la lista.
