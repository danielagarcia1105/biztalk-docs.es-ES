# <a name="checklist-undeploy-a-biztalk-application"></a>Lista de comprobación: Anular la implementación de una aplicación de BizTalk

|Paso|Referencia|  
|----------|---------------|  
|Eliminar la aplicación de BizTalk del grupo de BizTalk. Así se quitará de la consola de administración de BizTalk y de las bases de datos de BizTalk Server.|[Cómo eliminar una aplicación de BizTalk del grupo de BizTalk](../core/how-to-delete-a-biztalk-application-from-the-biztalk-group.md)|  
|Desinstalar la aplicación de los equipos en que se ha instalado.|[Cómo desinstalar una aplicación de BizTalk](../core/how-to-uninstall-a-biztalk-application.md)|  
|En caso de que sea necesario, quitar los ensamblados incluidos en la aplicación de la caché de ensamblados global (GAC).|[Cómo desinstalar un ensamblado de la GAC](http://msdn.microsoft.com/library/464706a8-f902-4d05-a724-19169facd2b4)|  
|Quitar otros archivos y configuraciones del equipo local que no se hayan quitado automáticamente en los pasos anteriores.|[Cómo quitar otros archivos y configuraciones para una aplicación de BizTalk](../core/how-to-remove-other-files-and-settings-for-a-biztalk-application.md)|
