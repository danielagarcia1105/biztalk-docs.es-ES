# <a name="transforms-sendport-node"></a>Transformaciones (nodo puertoEnvío)
El nodo Transformaciones del nodo puertoEnvío de un archivo de enlace contiene la colección de transformaciones salientes de un puerto de envío unidireccional que se exporta con el archivo de enlace.  
  
## <a name="nodes-in-the-transforms-node"></a>Nodos del nodo Transformaciones  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Transformación (nodo puertoEnvío transformaciones)](../core/transform-sendport-transforms-node.md)|Grabar|Transformación (ComplexType)|Especifica un asignador de BizTalk Server, o una transformación, es decir, un elemento que representa la asignación entre un esquema de origen y uno de destino.|No requerido|Valor predeterminado: ninguno|
