# <a name="single-sign-on-event-10656"></a>Inicio de sesión único: Evento 10656
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10656|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_PASSWORD_SYNC_ADAPTERS_STARTING|  
|Texto del mensaje|Se está iniciado la sincronización de contraseñas para adaptadores externos.%r|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que se está iniciando la sincronización de contraseñas para adaptadores externos.  
  
## <a name="user-action"></a>Acción del usuario  
  
-   No es necesario que el usuario realice ninguna acción.
