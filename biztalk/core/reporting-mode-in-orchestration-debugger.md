# <a name="reporting-mode-in-orchestration-debugger"></a>Modo de informe en el Depurador de orquestaciones
El modo de informe utiliza eventos de los que se ha hecho un seguimiento para mostrar lo ocurrido. Utiliza los datos de seguimiento mediante la **eventos de orquestación** marca de la opción. Esta marca se debe establecer antes de la ejecución de la instancia de organización. Para habilitar el seguimiento de eventos para una orquestación concreta, consulte [cómo configurar seguimiento para una orquestación](../core/how-to-configure-tracking-for-an-orchestration.md).  
  
 El **eventos de orquestación** opción realiza un seguimiento de la ejecución de cada forma en la orquestación como se produzca. En el modo de informe, se pueden reproducir los pasos o establecer puntos de interrupción en la clase de orquestación para que, de este modo, sea posible depurar nuevas instancias utilizando el modo interactivo.  
  
 Tras ejecutar el proceso empresarial que constituya el centro de interés, se puede utilizar una de las consultas de la vista Informe de la orquestación que se desea examinar.
