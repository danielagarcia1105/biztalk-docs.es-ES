# <a name="bam-interceptor-error-messages"></a>Mensajes de error de interceptor de BAM
Al usar los interceptores WF o WCF BAM, puede encontrar problemas que van desde un archivo de configuración de interceptor no válido a problemas en tiempo de ejecución al extraer propiedades de contexto o intentar realizar un seguimiento de eventos desde la respuesta de una operación unidireccional de WCF.  
  
 La siguiente tabla contiene una lista de los posibles errores de interceptor de BAM con los que puede encontrarse durante la implementación de su solución con bm.exe o durante el funcionamiento de la aplicación habilitada por el interceptor de BAM. Cada error incluye la causa y una o varias posibles soluciones.  
  
|Id. de registro de eventos|Mensaje de error|Causa|Solución|  
|------------------|-------------------|-----------|----------------|  
|1001|El Interceptor de BAM ha detectado el siguiente error: en el proceso de host '{0}', configuración del interceptor para tecnología '{1}' del manifiesto '{2}' contiene errores de validación de esquema en ({3}, \{4\): {5}.|El archivo de configuración de interceptor no valida con respecto al esquema de configuración de interceptor común o el esquema específico de las distintas tecnologías (Windows Workflow Foundation o Windows Communication Framework).|Busque la ubicación mencionada por el error en el archivo de configuración de interceptor y modifíquela para que coincida con el esquema.|  
|1002|El Interceptor de BAM ha detectado el siguiente error: en el host procesan '{0}', no se puede analizar el parámetro de configuración de aplicación '{1}' con valor '{2}'.|La configuración de la aplicación contiene unos valores que no pueden analizarse en el tipo correcto.|Corrija los valores de la configuración de la aplicación.|
