# <a name="error---native-parsing"></a>Error: análisis nativo
**Código de error**  
  
 btm1041  
  
 **Explicación**  
  
 No se pudo analizar el archivo de mensaje de instancia de entrada nativo especificado para la operación Comprobar asignación, por lo que se produjo el error de análisis indicado.  
  
 **Acción del usuario**  
  
 Según resulte necesario y valiéndose del error de análisis indicado, corrija el esquema de origen asociado a la asignación, el mensaje de instancia de entrada nativo del archivo especificado o ambos.
