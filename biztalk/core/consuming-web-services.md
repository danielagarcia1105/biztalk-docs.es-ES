# <a name="consuming-web-services"></a>Consumir servicios web
Consumir servicios Web le permite agregar al proceso empresarial servicios Web existentes. Es posible agregar varios servicios Web a una orquestación única.  
  
 Se puede consumir (llamar) un servicio Web procedente de una orquestación usando puertos Web. Para consumir un servicio Web desde una orquestación, debe crear un puerto Web y construir mensajes Web.  
  
 Puede usar los encabezados SOAP con el servicio Web consumido, cambiar el URI de un servicio Web consumido y establecer el URI de forma dinámica para un servicio Web consumido.  
  
 Para obtener información acerca de cómo configurar un SOAP el controlador de recepción, vea [cómo configurar un controlador de recepción de SOAP](../core/how-to-configure-a-soap-receive-handler.md). Para obtener información acerca de cómo configurar un SOAP ubicación de recepción, consulte [cómo configurar una ubicación de recepción de SOAP](../core/how-to-configure-a-soap-receive-location.md).
