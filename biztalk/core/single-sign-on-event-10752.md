# <a name="single-sign-on-event-10752"></a>De sesión único: Evento 10752
## <a name="details"></a>Detalles  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  Nombre del producto   |                 Inicio de sesión único (SSO) empresarial                  |
| Versión del producto | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    Identificador del evento     |                           10752                            |
|  Origen del evento   |                           ENTSSO                           |
|    Componente    |                            N/D                             |
|  Nombre simbólico  |                    ENTSSO_E_APP_EXISTS                     |
|  Texto del mensaje   |              La aplicación ya existe.               |
  
## <a name="explanation"></a>Explicación  
 Ya existe una aplicación con el mismo nombre.  
  
## <a name="user-action"></a>Acción del usuario  
 Asigne otro nombre a la aplicación nueva.
