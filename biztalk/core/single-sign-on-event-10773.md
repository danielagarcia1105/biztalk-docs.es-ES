# <a name="single-sign-on-event-10773"></a>De sesión único: Evento 10773
## <a name="details"></a>Detalles  
  
|                 |                                                                             |
|-----------------|-----------------------------------------------------------------------------|
|  Nombre del producto   |                          Inicio de sesión único (SSO) empresarial                          |
| Versión del producto |         [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]          |
|    Identificador del evento     |                                    10773                                    |
|  Origen del evento   |                                   ENTSSO                                    |
|    Componente    |                                     N/D                                     |
|  Nombre simbólico  |                        ENTSSO_E_SSO_MUST_BE_DISABLED                        |
|  Texto del mensaje   | Se debe deshabilitar el inicio de sesión único empresarial antes de realizar esta función. |
  
## <a name="explanation"></a>Explicación  
 Se debe deshabilitar el inicio de sesión único empresarial antes de realizar esta función.  
  
## <a name="user-action"></a>Acción del usuario  
 Para obtener más información, consulte [cómo deshabilitar SSO](../core/how-to-disable-sso.md).
