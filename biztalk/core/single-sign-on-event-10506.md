# <a name="single-sign-on-event-10506"></a>Inicio de sesión único: Evento 10506
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10506|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_SERVICE_INSTALL_OK|  
|Texto del mensaje|Se ha instalado el servicio SSO.|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que se instaló el servicio ENTSSO.
