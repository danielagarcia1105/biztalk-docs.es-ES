# <a name="tibco-rendezvous-adapter"></a>Adaptador TIBCO Rendezvous
El adaptador de Microsoft BizTalk para TIBCO Rendezvous permite usar las funciones empresariales de TIBCO Rendezvous en BizTalk Server. En las secciones siguientes se explica cómo se configura el adaptador para obtener acceso a información específica de TIBCO Rendezvous.  
  
## <a name="get-started"></a>Introducción
[Introducción](../core/getting-started-with-biztalk-adapter-for-tibco-rendezvous.md) describe los diferentes mensajes y conceptos y también proporciona información detallada sobre la instalación, esquemas y limitaciones. Avanzar a través de algunos tutoriales para recibir y enviar datos con este adaptador.

## <a name="architecture"></a>Architecture
[Arquitectura](../core/architecture-of-biztalk-adapter-for-tibco-rendezvous.md) describe la conectividad y cómo se pasan los mensajes.

## <a name="security"></a>Seguridad
[Seguridad](../core/security-in-biztalk-adapter-for-tibco-rendezvous.md) implica el uso de Enterprise Single Sign-on (SSO) en BizTalk Server para proteger las aplicaciones que usan este adaptador.

## <a name="create-applications"></a>Crear aplicaciones
[Crear artefactos de la aplicación](../core/developing-applications1.md) le enseñamos a agregar la recepción y envío de artefactos en administración de BizTalk, proporciona información detallada sobre mensajes y asignación de tipos de datos y mensajes de propiedades de contexto.

## <a name="importing-assemblies"></a>Importación de ensamblados
[Importar puertos y ensamblados](../core/deploying-biztalk-adapter-for-tibco-rendezvous.md) se describe cómo importar el archivo de enlace de la aplicación de administración de BizTalk.

## <a name="handle-exceptions"></a>Controlar excepciones
[Control de excepciones](../core/using-biztalk-server-exception-handling4.md) proporciona instrucciones sobre cómo agregar diferentes formas a la orquestación para controlar las excepciones.

## <a name="troubleshoot"></a>Solucionar problemas
[Solución de problemas de TIBCO Rendezvous](../core/troubleshooting-tibco-rendezvous.md) incluye detalles sobre el uso de seguimiento de eventos para Windows.
