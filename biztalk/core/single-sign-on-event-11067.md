# <a name="single-sign-on-event-11067"></a>Inicio de sesión único: Evento 11067
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|11067|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|SSO_ERROR_PASSWORD_SYNC_MIIS_START_FAILED|  
|Texto del mensaje|No se pudo iniciar la sincronización de contraseñas de Windows (desde MIIS).%r<br /><br /> Código de error: %1|  
  
## <a name="explanation"></a>Explicación  
 Este mensaje de información indica que no se pudo iniciar la sincronización de contraseñas de Windows (desde MIIS).  
  
## <a name="user-action"></a>Acción del usuario  
 No se requiere ninguna acción del usuario.
