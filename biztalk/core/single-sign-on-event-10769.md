# <a name="single-sign-on-event-10769"></a>Inicio de sesión único: Evento 10769
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10769|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_APP_INCOMPLETE_FIELDS|  
|Texto del mensaje|No se puede habilitar la aplicación porque los campos están incompletos para esta aplicación.|  
  
## <a name="explanation"></a>Explicación  
 No se puede habilitar la aplicación porque los campos están incompletos para esta aplicación.  
  
## <a name="user-action"></a>Acción del usuario  
 Una vez especificado el número de campos para una determinada aplicación, debe crearlos y rellenarlos antes de que se pueda habilitar la aplicación.
