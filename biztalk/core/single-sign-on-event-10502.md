# <a name="single-sign-on-event-10502"></a>De sesión único: Evento 10502
## <a name="details"></a>Detalles  

|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  Nombre del producto   |                 Inicio de sesión único (SSO) empresarial                  |
| Versión del producto | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    Identificador del evento     |                           10502                            |
|  Origen del evento   |                           ENTSSO                           |
|    Componente    |                            N\D                             |
|  Nombre simbólico  |                  SSO_INFO_SERVICE_STARTED                  |
|  Texto del mensaje   |                Se inició el servicio SSO.                |

## <a name="explanation"></a>Explicación  
 Este evento de información indica que se inició el servicio ENTSSO.
