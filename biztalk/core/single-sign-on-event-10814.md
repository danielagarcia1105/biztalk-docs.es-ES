# <a name="single-sign-on-event-10814"></a>De sesión único: Evento 10814
## <a name="details"></a>Detalles  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  Nombre del producto   |                 Inicio de sesión único (SSO) empresarial                  |
| Versión del producto | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    Identificador del evento     |                           10814                            |
|  Origen del evento   |                           ENTSSO                           |
|    Componente    |                            N/D                             |
|  Nombre simbólico  |               ENTSSO_E_REENCRYPT_IN_PROGRESS               |
|  Texto del mensaje   |         El recifrado de la base de datos de SSO está en curso.         |
  
## <a name="explanation"></a>Explicación  
 La acción no se puede realizar durante el recifrado de la base de datos de SSO.  
  
## <a name="user-action"></a>Acción del usuario  
 Vuelva a intentar la acción más tarde.
