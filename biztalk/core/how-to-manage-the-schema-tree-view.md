1.  <span data-ttu-id="3b391-106">Mueva el puntero del mouse hasta el borde inferior de la ventana de edición principal de Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)], en la que se presenta la vista XSD y la vista de árbol de esquema en paralelo, hasta que el cursor cambie al icono de ajuste de tamaño vertical de ventana estándar.</span><span class="sxs-lookup"><span data-stu-id="3b391-106">Move the mouse pointer to the bottom edge of the Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] main editing window, which displays the schema tree view side-by-side with the XSD view, until the cursor changes to the standard window vertical resizing icon.</span></span>  
  
2.  <span data-ttu-id="3b391-107">Haga clic y mantenga presionado el botón primario del mouse y arrastre el borde de la ventana hacia arriba o hacia abajo.</span><span class="sxs-lookup"><span data-stu-id="3b391-107">Click and hold the left mouse button and drag the window edge either up or down.</span></span>

2.  <span data-ttu-id="3b391-111">Haga clic y mantenga presionado el botón primario del mouse y arrastre el borde del panel hacia la izquierda (más estrecho) o hacia la derecha (más ancho).</span><span class="sxs-lookup"><span data-stu-id="3b391-111">Click and hold the left mouse button and drag the pane edge either left (narrower) or right (wider).</span></span>
