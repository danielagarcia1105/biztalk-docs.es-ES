# <a name="using-single-sign-on"></a>Mediante el inicio de sesión único
Microsoft BizTalk Adapter para TIBCO Enterprise Message Service admite el inicio de sesión único (SSO). Una aplicación afiliada creada por las herramientas de inicio de sesión único empresarial representa un sistema servidor como TIBCO Enterprise Message Service.
