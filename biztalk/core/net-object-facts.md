# <a name="net-object-facts"></a>Hechos de objetos .NET
En el Compositor de reglas de negocio, se puede especificar un ensamblado .NET como origen de datos. A continuación, puede seleccionar una clase o miembro de clase del ensamblado y arrastrarla hasta una definición de vocabulario o regla.  
  
 El procesamiento de objetos .NET impuestos es el tipo de procesamiento más sencillo. El motor procesa cada objeto como una instancia única. Para obtener más información sobre la**Assert** función, vea [las funciones de Control del motor](../core/engine-control-functions.md).
