# <a name="activity-status"></a>Estado de actividad
El área Estado de actividad de los detalles de resultados le permite ver todos los detalles del elemento de resultados en una sola página.  
  
 El área de estado de actividad contiene dos subáreas; los hitos y datos.  
  
## <a name="milestones"></a>Hitos  
 El área Hitos muestra todos los hitos definidos para la actividad seleccionada. Los datos de hitos muestran la fecha y hora en las que se alcanzó cada hito. Algunos hitos son mutuamente excluyentes; Por ejemplo, pedido aprobado y denegado. En los casos de hitos que se excluyen mutuamente, uno de los hitos mostrará los datos de fecha y hora y el otro estará vacío, lo que indicará que no hay datos para él.  
  
## <a name="data"></a>data  
 El área Datos muestra todos los elementos de datos definidos para la actividad. Los elementos de datos se muestran en orden alfabético.
