# <a name="single-sign-on-event-10787"></a>De sesión único: Evento 10787
## <a name="details"></a>Detalles  
  
|                 |                                                                                                                               |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                   Inicio de sesión único (SSO) empresarial                                                   |
| Versión del producto |                                  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                   |
|    Identificador del evento     |                                                             10787                                                             |
|  Origen del evento   |                                                            ENTSSO                                                             |
|    Componente    |                                                              N/D                                                              |
|  Nombre simbólico  |                                                   ENTSSO_E_FLAGS_NOT_VALID                                                    |
|  Texto del mensaje   | Los marcadores especificados no son válidos o no son compatibles entre sí. Para obtener información detallada, consulte el registro de eventos (en el equipo "%1"). |
  
## <a name="explanation"></a>Explicación  
 Los marcadores especificados no son válidos o no son compatibles entre sí.  
  
## <a name="user-action"></a>Acción del usuario  
 Compruebe el registro de eventos en el equipo especificado para ver los marcadores válidos y no válidos de esta aplicación y los errores relacionados.
