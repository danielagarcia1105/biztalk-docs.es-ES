# <a name="single-sign-on-event-10829"></a>De sesión único: Evento 10829
## <a name="details"></a>Detalles  
  
|                 |                                                                                                                             |
|-----------------|-----------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                  Inicio de sesión único (SSO) empresarial                                                  |
| Versión del producto |                                 [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                  |
|    Identificador del evento     |                                                            10829                                                            |
|  Origen del evento   |                                                           ENTSSO                                                            |
|    Componente    |                                                             N/D                                                             |
|  Nombre simbólico  |                                           ENTSSO_E_PSADMIN_INVALID_ADAPTER_TYPE_2                                           |
|  Texto del mensaje   | Tipo de aplicación no válido para adaptador. Los tipos de aplicación válidos son "Adaptador de sincronización de contraseñas" o "Adaptador de grupo de sincronización de contraseñas". |
  
## <a name="explanation"></a>Explicación  
 Tipo de aplicación no válido para adaptador. Los tipos de aplicación válidos son "Adaptador de sincronización de contraseñas" o "Adaptador de grupo de sincronización de contraseñas".  
  
## <a name="user-action"></a>Acción del usuario  
 Use uno de los tipos de aplicación válidos.
