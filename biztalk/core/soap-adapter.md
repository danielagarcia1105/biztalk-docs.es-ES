# <a name="soap-adapter"></a>Adaptador de SOAP
Los servicios web de Microsoft BizTalk Server utilizan el adaptador de SOAP cuando reciben y envían solicitudes de servicio web. El adaptador de SOAP le permite publicar orquestaciones como servicios web y consumir servicios web externos.
