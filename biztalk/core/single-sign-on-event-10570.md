# <a name="single-sign-on-event-10570"></a>De sesión único: Evento 10570
## <a name="details"></a>Detalles  
  
|                 |                                                                             |
|-----------------|-----------------------------------------------------------------------------|
|  Nombre del producto   |                          Inicio de sesión único (SSO) empresarial                          |
| Versión del producto |         [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]          |
|    Identificador del evento     |                                    10570                                    |
|  Origen del evento   |                                   ENTSSO                                    |
|    Componente    |                                     N/D                                     |
|  Nombre simbólico  |                 SSO_WARN_SSO_DISABLED_FOR_CHANGE_SSO_ADMIN                  |
|  Texto del mensaje   | Antes de actualizar el nombre de la cuenta de administradores de SSO, se debe deshabilitar SSO.%r |
  
## <a name="explanation"></a>Explicación  
 Antes de actualizar el nombre de la cuenta de administradores de SSO, se debe deshabilitar SSO.  
  
## <a name="user-action"></a>Acción del usuario  
 Para obtener información acerca de cómo deshabilitar SSO, vea [cómo deshabilitar SSO](../core/how-to-disable-sso.md).
