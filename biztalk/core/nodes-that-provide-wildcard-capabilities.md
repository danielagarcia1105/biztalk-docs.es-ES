# <a name="nodes-that-provide-wildcard-capabilities"></a>Nodos que proporcionan funciones de carácter comodín
El Editor de BizTalk proporciona una asignación directa a conceptos XSD que proporcionan capacidades de carácter comodín para los elementos y atributos. El **cualquier elemento** nodo se asigna directamente en el lenguaje de definición (XSD) de esquemas XML **cualquier** elemento y el **cualquier atributo** nodo se asigna directamente a la XSD  **anyAttribute** elemento. En el resto de esta sección se proporciona información adicional sobre estos nodos.
