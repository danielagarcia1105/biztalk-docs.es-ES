# <a name="synchronizing-passwords"></a>Sincronización de contraseñas
La sincronización de una contraseña se realiza mediante un adaptador de sincronización de contraseñas. Este adaptador debe poder comunicarse con un sistema remoto específico, ajeno a Windows, y debe poder instruir al sistema para actualizar información de contraseñas.
