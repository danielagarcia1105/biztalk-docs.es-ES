# <a name="single-sign-on-event-10782"></a>De sesión único: Evento 10782
## <a name="details"></a>Detalles  
  
|                 |                                                                        |
|-----------------|------------------------------------------------------------------------|
|  Nombre del producto   |                       Inicio de sesión único (SSO) empresarial                        |
| Versión del producto |       [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]       |
|    Identificador del evento     |                                 10782                                  |
|  Origen del evento   |                                 ENTSSO                                 |
|    Componente    |                                  N/D                                   |
|  Nombre simbólico  |                         ENTSSO_E_INVALID_FLAGS                         |
|  Texto del mensaje   | Los marcadores especificados no son válidos o no son compatibles entre sí. |
  
## <a name="explanation"></a>Explicación  
 Los marcadores especificados no son válidos o no son compatibles entre sí.  
  
## <a name="user-action"></a>Acción del usuario  
 Compruebe el registro de eventos para ver los marcadores válidos y no válidos de esta aplicación y los errores relacionados.
