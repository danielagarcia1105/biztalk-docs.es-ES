# <a name="using-certificates-for-party-resolution"></a>Usar certificados para la resolución de entidades
A *entidad* es una entidad externa a BizTalk Server que interactúa con una orquestación. Cuando BizTalk Server recibe un mensaje, usa el certificado de clave pública para determinar quién envió el mensaje y para resolver el remitente en una entidad conocida en el entorno de BizTalk Server.  
  
 Esta sección proporciona información sobre cómo configurar [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] canalizaciones, ubicaciones de recepción, puertos y el entorno de BizTalk Server para resolución de entidades.  
  
 Para obtener más información acerca de la autenticación de un mensaje, vea [autenticación del remitente de un mensaje](../core/authenticating-the-sender-of-a-message.md).
