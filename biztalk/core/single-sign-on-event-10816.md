# <a name="single-sign-on-event-10816"></a>De sesión único: Evento 10816
## <a name="details"></a>Detalles  
  
|                 |                                                               |
|-----------------|---------------------------------------------------------------|
|  Nombre del producto   |                   Inicio de sesión único (SSO) empresarial                   |
| Versión del producto |  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]   |
|    Identificador del evento     |                             10816                             |
|  Origen del evento   |                            ENTSSO                             |
|    Componente    |                              N/D                              |
|  Nombre simbólico  |          ENTSSO_E_EXTERNAL_UPDATE_FAILED_MORE_RECENT          |
|  Texto del mensaje   | Las credenciales externas de la base de datos de SSO son más recientes. |
  
## <a name="explanation"></a>Explicación  
 Se recibió una contraseña desactualizada proveniente del sistema externo y se ignorará.  
  
## <a name="user-action"></a>Acción del usuario  
 No es necesario que el usuario realice ninguna acción.
