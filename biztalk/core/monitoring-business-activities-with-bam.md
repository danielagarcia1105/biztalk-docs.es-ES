# <a name="monitoring-business-activities-with-bam"></a>Supervisión de actividades económicas con BAM
Los Analistas de negocios usan la Supervisión de la actividad económica (BAM) para supervisar datos sobre procesos empresariales. Estos analistas usan el libro de trabajo de BAM en Microsoft Office Excel para definir los datos que deben recopilarse de los procesos empresariales y la forma en que los usuarios empresariales verán esos datos.  
  
 Los analistas de negocios pueden usar BAM para supervisar procesos que abarcan varias aplicaciones. Por ejemplo, un proceso de pedido puede tener partes que se completan a través de distintas aplicaciones, como BizTalk Server, actividad de base de datos y mensajes enviados y recibidos de un sitio web interno, el sitio web de un proveedor y el sitio web de un transportista.  
  
 Los usuarios empresariales pueden usar BAM para ver datos en tiempo real y datos archivados (históricos) sobre procesos empresariales. Por ejemplo, puede que un usuario empresarial desee saber cuándo llega un pedido de más de 100 $ (datos en tiempo real) o cuántos pedidos de más de 100 $ han llegado este mes (datos archivados). BAM agrupa los datos en tiempo real y los datos archivados para mejorar la velocidad con que presenta los datos.  
  
 Para obtener información acerca de los requisitos de hardware y software para instalar BAM, consulte la Guía de instalación de BizTalk Server, en [de BizTalk Server lo que nuevo, instalación, configuración y actualización](../install-and-config-guides/biztalk-server-what-s-new-installation-configuration-and-upgrade.md).
