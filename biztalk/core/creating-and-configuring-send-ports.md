# <a name="creating-and-configuring-send-ports"></a>Crear y configurar puertos de envío

## <a name="overview"></a>Información general
En esta sección, se proporcionan instrucciones acerca del uso de la consola de administración de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] para crear y configurar puertos de envío para una aplicación de BizTalk. Un puerto de envío es una ubicación a la que se envían mensajes o desde la que se reciben mensajes y que, además, queda unívocamente identificado por su nombre.  
  
> [!NOTE]
>  Puede usar el modelo de objetos de Microsoft Windows Management Instrumentation (WMI) para crear y ejecutar scripts que automatizan las tareas administrativas. Para obtener información acerca del uso de WMI, consulte el **referencia de clase WMI** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)].
