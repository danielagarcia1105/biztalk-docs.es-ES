# <a name="error---schema-file-name-not-valid"></a>Error: nombre de archivo de esquema no válido
**Explicación**  
  
 El nombre de archivo del esquema indicado no es válido.  
  
 **Acción del usuario**  
  
 Compruebe que el archivo de esquema existe y que se puede obtener acceso a él en la ubicación especificada.
