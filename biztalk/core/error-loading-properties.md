## <a name="explanation"></a>Explicación  
 Este error se produce cuando el adaptador carga las propiedades para la ubicación de recepción o el puerto de envío desde la base de datos.  
  
## <a name="user-action"></a>Acción del usuario  
 El mensaje de error debe contener la información necesaria para la solución.
