# <a name="single-sign-on-event-10783"></a>Inicio de sesión único: Evento 10783
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10783|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_NO_SECRET|  
|Texto del mensaje|No se puede realizar la tarea de cifrado o descifrado porque el secreto no está disponible en el servidor secreto principal. Consulte el registro de eventos para ver los errores relacionados.|  
  
## <a name="explanation"></a>Explicación  
 Probablemente hubo un problema durante la configuración.  
  
## <a name="user-action"></a>Acción del usuario  
 Consulte el registro de eventos para obtener más información y para ver los errores relacionados.
