# <a name="jd-edwards-enterpriseone-adapter"></a>Adaptador JD Edwards EnterpriseOne
Microsoft BizTalk Adapter para J.D. Edwards EnterpriseOne le permite usar las funciones de negocio de JD Edwards EnterpriseOne en BizTalk Server. En las secciones siguientes se explica cómo configurar el adaptador para obtener acceso a la información de JD Edwards EnterpriseOne.

[Introducción](../core/getting-started-with-biztalk-adapter-for-jd-edwards-enterpriseone.md) enumera los pasos necesarios para instalar el adaptador y siga los pasos algunos tutoriales.

[Seguridad](../core/security-in-biztalk-adapter-for-jd-edwards-enterpriseone.md) implica el uso de Enterprise Single Sign-on (SSO) en BizTalk Server para proteger las aplicaciones que usan este adaptador.
