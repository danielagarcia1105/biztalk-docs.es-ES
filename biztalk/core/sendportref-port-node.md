# <a name="sendportref-port-node"></a>SendPortRef (nodo Puerto)
El nodo SendPortRef del nodo Puerto de un archivo de enlace contiene información acerca de un puerto de envío al que hace referencia un servicio.  
  
## <a name="nodes-in-the-sendportref-node"></a>Nodos del nodo SendPortRef  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|Nombre|Atributo|xs:string|Especifica el nombre de un puerto de envío al que hace referencia un servicio.|No requerido|Valor predeterminado: vacío|
