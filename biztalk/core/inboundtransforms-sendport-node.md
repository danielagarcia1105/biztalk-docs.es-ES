# <a name="inboundtransforms-sendport-node"></a>InboundTransforms (nodo puertoEnvío)
El nodo InboundTransforms del nodo puertoEnvío de un archivo de enlace contiene la colección de transformaciones de entrada de un puerto de envío bidireccional que se exporta con el archivo de enlace.  
  
## <a name="nodes-in-the-inboundtransforms-node"></a>Nodos del nodo InboundTransforms  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Transformación (nodo InboundTransforms)](../core/transform-inboundtransforms-node.md)|Grabar|Transformación (ComplexType)|Especifica un asignador de BizTalk Server, o una transformación, es decir, un elemento que representa la asignación entre un esquema de origen y uno de destino.|No requerido|Valor predeterminado: ninguno|
