# <a name="engine-performance-and-capacity"></a>Capacidad y rendimiento del motor
El rendimiento y la capacidad del motor son consideraciones importantes al reproducir la solución de BizTalk Server. El rendimiento de motor es la capacidad de procesar documentos a lo largo del tiempo y la capacidad de motor es la capacidad del sistema de BizTalk Server de acumular documentos que se reciben pero que no se han procesado aún.
