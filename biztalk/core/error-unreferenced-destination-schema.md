# <a name="error---unreferenced-destination-schema"></a>Error: esquema de destino sin referencia
**Código de error**  
  
 btm1052  
  
 **Explicación**  
  
 El esquema de destino indicado no se encuentra en el proyecto de Microsoft BizTalk Server. **Acción del usuario**  
  
 Agregue el esquema de destino al proyecto de BizTalk que contenga la asignación mediante el **Agregar elemento existente** comandos que están disponibles en la **archivo** menú y en el menú contextual del proyecto de BizTalk en Explorador de soluciones.
