# <a name="distributionlistcollection-node"></a>DistributionListCollection (nodo)
El nodo DistributionListCollection de un archivo de enlace es el nodo primario de todos los nodos DistributionList que contienen información específica acerca de las listas de distribución que se exportan con el archivo de enlace.  
  
## <a name="nodes-in-the-distributionlistcollection-node"></a>Nodos del nodo DistributionListCollection  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[DistributionList](../core/distributionlist-distributionlistcollection-node.md)|Grabar|DistributionList (ComplexType)|Especifica información acerca de una lista de distribución que se exporta con el archivo de enlace.|No requerido|Valor predeterminado: ninguno|
