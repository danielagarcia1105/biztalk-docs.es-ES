# <a name="single-sign-on-event-11041"></a>De sesión único: Evento 11041
## <a name="details"></a>Detalles  
  
|                 |                                                                                                                                                                                              |
|-----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                                                  Inicio de sesión único (SSO) empresarial                                                                                   |
| Versión del producto |                                                                  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                                  |
|    Identificador del evento     |                                                                                            11041                                                                                             |
|  Origen del evento   |                                                                                            ENTSSO                                                                                            |
|    Componente    |                                                                                             N/D                                                                                              |
|  Nombre simbólico  |                                                                                    SSO_WARN_LOCAL_ACCOUNT                                                                                    |
|  Texto del mensaje   | El servicio SSO está en ejecución en una cuenta local. Esto no se recomienda y limitará el funcionamiento de SSO. Para obtener información detallada, consulte la documentación.%r<br /><br /> Cuenta de servicio SSO: %1 |
  
## <a name="explanation"></a>Explicación  
 El servicio SSO está en ejecución en una cuenta local. Esto no se recomienda y limitará el funcionamiento de SSO.  
  
## <a name="user-action"></a>Acción del usuario  
 Para obtener más información, consulte [recomendaciones de seguridad de inicio de sesión único](../core/sso-security-recommendations.md).
