# <a name="silent-configuration-of-the-mqseries-adapter"></a>Configuración silenciosa del adaptador de MQSeries
Utilice el Asistente para configuración de MQSAgent COM+ en la línea de comandos para llevar a cabo una configuración silenciosa. También puede usar la versión de línea de comandos del asistente para quitar MQSAgent. El asistente lee la información de configuración de un archivo XML.
