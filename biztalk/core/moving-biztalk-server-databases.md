# <a name="moving-biztalk-server-databases"></a>Mover bases de datos de BizTalk Server
A veces, en un centro de datos activos cualquiera, es necesario mover las bases de datos a un hardware diferente. Por ejemplo, es posible que se precise mover una base de datos cuando ésta requiera un hardware con mayor capacidad, o cuando el hardware se actualice o se sustituya con nuevos equipos. Los procedimientos de esta sección describen cómo mover las bases de datos de BizTalk Server que se mueven más habitualmente.
