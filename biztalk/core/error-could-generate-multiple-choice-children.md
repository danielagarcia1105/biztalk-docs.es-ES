# <a name="error---could-generate-multiple-choice-children"></a>Error: se generan varios secundarios de elección
**Código de error**  
  
 btm1027  
  
 **Explicación**  
  
 Vínculos al esquema de destino habilitan, indebidamente varios nodos dentro de un **elección** nodo de grupo que se genere.  
  
 **Acción del usuario**  
  
 Cambie los vínculos en el esquema de destino, por lo tanto que solo un único nodo dentro del **elección** se puede generar el nodo de grupo.
