# <a name="interceptor-operations"></a>Operaciones del interceptor
En esta sección se proporciona información acerca de las operaciones comunes de los archivos de configuración de interceptor. Estas operaciones están disponibles en los interceptores Windows Workflow Foundation (WF) y Windows Communication Framework (WCF) y en cualquier otro interceptor que se crean sobre el marco de trabajo de interceptor común. Los interceptores WF y WCF amplían la funcionalidad común proporcionada por estos operadores mediante la definición de operaciones adicionales y específicas del dominio para recuperar y manipular datos de interceptor.  
  
> [!NOTE]
>  Los nombres de operaciones distinguen mayúsculas de minúsculas.
