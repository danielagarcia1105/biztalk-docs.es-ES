# <a name="functoid-properties"></a>Propiedades generales de functoid

## <a name="overview"></a>Información general
Functoids tienen propiedades que están disponibles para examinar y modificar en el [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] ventana Propiedades. Varias propiedades de functoid únicamente están disponibles para algunos de ellos y están deshabilitadas para otros. Una propiedad de functoid, **parámetros de entrada**, es especialmente importante para casi todos los functoids y probablemente se usará con más frecuencia. Para obtener más información sobre la **parámetros de entrada** propiedad, vea [parámetros de entrada de Functoid](../core/functoid-input-parameters.md).  
  
 Todas las propiedades de functoid se clasifican como **General** propiedades. Para obtener una lista completa de propiedades de functoid, consulte el **propiedades generales de Functoid** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)].
