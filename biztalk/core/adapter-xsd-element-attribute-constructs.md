# <a name="adapter-xsd-element-attribute-constructs"></a>Construcciones de elemento-atributo XSD de adaptador
En la siguiente tabla se enumeran las construcciones de elemento-atributo de XSD compatibles con el marco de trabajo de adaptadores. El eje vertical contiene los tipos de elementos compatibles y el eje horizontal contiene los tipos de atributo válidos para cada elemento.  
  
> [!NOTE]
>  Los elementos y los atributos deben tener nombres únicos. Los nombres duplicados producen un error de compilación.  
  
> [!NOTE]
>  El atributo fixed con el aspecto de enumeración no es compatible.  
  
> [!NOTE]
>  La configuración de XSD debe tener un elemento denominado **uri** de tipo XSD: String.
