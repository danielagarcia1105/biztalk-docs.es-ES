# <a name="how-to-construct-a-web-message-with-no-message-parts"></a>Cómo construir un mensaje web sin partes de mensaje
Un mensaje Web que no contiene partes de mensaje es un método que no tiene parámetros de entrada.  
  
### <a name="to-construct-a-web-message-that-does-not-have-message-parts"></a>Para construir un mensaje Web que no contenga partes de mensaje  
  
1.  Con una orquestación abierta, abra el **cuadro de herramientas** y haga clic en el **orquestaciones de BizTalk** ficha.  
  
2.  Agregar un **construir mensaje** forma.  
  
3.  Seleccione el mensaje para la **construir mensaje** forma.  
  
> [!NOTE]
>  No deben tener la **asignación de mensajes** forma dentro de la **construir mensaje** forma.
