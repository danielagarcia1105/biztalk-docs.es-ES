# <a name="single-sign-on-event-10747"></a>De sesión único: Evento 10747
## <a name="details"></a>Detalles  

|                 |                                                                                                                                        |
|-----------------|----------------------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                       Inicio de sesión único (SSO) empresarial                                                        |
| Versión del producto |                                       [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                       |
|    Identificador del evento     |                                                                 10747                                                                  |
|  Origen del evento   |                                                                  N\D                                                                   |
|    Componente    |                                                                  N\D                                                                   |
|  Nombre simbólico  |                                                     SSO_ERROR_UNKNOWN_NOTIFICATION                                                     |
|  Texto del mensaje   | Se recibió un tipo de notificación desconocido.%r<br /><br /> Id. de seguimiento: %1 %r<br /><br /> Adaptador: %2 %r<br /><br /> Tipo de notificación: %3 |

## <a name="explanation"></a>Explicación  
 Este evento de error indica que el servicio SSO detectó un error interno con un tipo de notificación no válido.  

## <a name="user-action"></a>Acción del usuario  
 Para resolver este error, realice lo siguiente:  

- Compruebe los registros de eventos de aplicación y del sistema para eventos asociados.  

  Para obtener más información, vea los recursos siguientes:
