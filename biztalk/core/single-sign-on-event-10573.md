# <a name="single-sign-on-event-10573"></a>Inicio de sesión único: Evento 10573
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10573|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|SSO_WARN_INVALID_SSO_APP_ADMIN_GROUP|  
|Texto del mensaje|Cuenta de administradores afiliados de SSO no válida para actualización de información global.%r<br /><br /> Administradores afiliados de SSO: %1 %r<br /><br /> Cuentas no válidas: %2 %r<br /><br /> Código de error: %3|  
  
## <a name="explanation"></a>Explicación  
 La cuenta de administradores afiliados de SSO no es válida para actualización de información global.  
  
## <a name="user-action"></a>Acción del usuario  
 Este mensaje de advertencia contiene información relacionada con la cuenta de administradores afiliados de SSO y las cuentas no válidas. Revise esta información, realice las correcciones necesarias e intente actualizar nuevamente.
