# <a name="single-sign-on-event-10784"></a>De sesión único: Evento 10784
## <a name="details"></a>Detalles  
  
|                 |                                                                                          |
|-----------------|------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                Inicio de sesión único (SSO) empresarial                                 |
| Versión del producto |                [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                |
|    Identificador del evento     |                                          10784                                           |
|  Origen del evento   |                                          ENTSSO                                          |
|    Componente    |                                           N/D                                            |
|  Nombre simbólico  |                                ENTSSO_E_FILE_OPEN_FAILED                                 |
|  Texto del mensaje   | No se pudo crear o abrir el archivo especificado. Compruebe el nombre de archivo y los permisos de acceso. |
  
## <a name="explanation"></a>Explicación  
 No se pudo crear o abrir el archivo especificado.  
  
## <a name="user-action"></a>Acción del usuario  
 Compruebe el nombre de archivo y los permisos de acceso.
