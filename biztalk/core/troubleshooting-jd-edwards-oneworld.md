# <a name="troubleshooting-jd-edwards-oneworld"></a>Solución de problemas de JD Edwards OneWorld
En esta sección se describen problemas y mensajes de error habituales que puede encontrar en el Adaptador de BizTalk para JD Edwards OneWorld, y se proporcionan posibles correcciones. Además, se trata el uso de Seguimiento de eventos para Windows.  
  
 Puede usar las siguientes herramientas de depuración y seguimiento para solucionar problemas del adaptador:  
  
-   Depuración nativa de BizTalk Server (por ejemplo, el Tipo de seguimiento en su puerto o el Depurador de orquestaciones).  
  
-   Mensajes de error enviados al registro de eventos.  
  
-   Captura de mensajes del transmisor, receptor y administración a través de BTAJDCEOneWorldTrace.cmd y una herramienta que convierte los archivos .etl, como tracerpt.exe o tracedmp.exe.
