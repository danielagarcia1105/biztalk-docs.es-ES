# <a name="host-initiated-sso"></a>SSO iniciado por host
El inicio de sesión único iniciado por host permite a una solicitud del sistema host obtener acceso a un recurso en un sistema de Windows. El sistema host (por ejemplo una cuenta RACF) existe en un entorno que no esté basado en Windows y en el contexto de un usuario de otros sistemas. El almacén de credenciales de inicio de sesión único asigna cuentas de host a cuentas de Windows y permite este acceso.
