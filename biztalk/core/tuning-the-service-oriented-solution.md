# <a name="tuning-the-service-oriented-solution"></a>Ajustar la solución orientada a servicios
Las orquestaciones recopilan información de rendimiento acerca de muchas partes de la aplicación a través de la API Supervisión de la actividad económica (SAE). Los datos pueden resultar útiles para ajustar la aplicación. Por supuesto, también puede usar los contadores de rendimiento integrados.  
  
 Para obtener más información acerca de BAM, consulte [usando Business Activity Monitoring](../core/using-business-activity-monitoring.md). Para obtener más información acerca de cómo la solución usa BAM, consulte [supervisión de la solución orientada a servicios con BAM](../core/monitoring-the-service-oriented-solution-with-bam.md). Para obtener más información acerca de los contadores de rendimiento de BizTalk, consulte [contadores de rendimiento del motor de orquestaciones](../core/orchestration-engine-performance-counters.md).  
  
 También puede ajustar la solución ajustando los valores del host de BizTalk. Para obtener más información, consulte [cómo modificar propiedades de Host](../core/how-to-modify-host-properties.md).
