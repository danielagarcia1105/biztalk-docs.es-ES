# <a name="how-to-search-for-activities"></a>Búsqueda de actividades
Use la Búsqueda de la actividad económica en el portal web de BAM para buscar los datos de seguimiento (también denominados instancias) que se encuentran almacenados en la base de datos de BAM.  
  
 La búsqueda detecta tanto datos activos de los procesos en curso como datos almacenados.  
  
### <a name="to-search-the-bam-database"></a>Procedimiento para buscar en la base de datos de BAM  
  
1.  En Internet Explorer, en el **dirección** , escriba la dirección URL del Portal Web de BAM.  
  
2.  Desde el **Mis vistas** panel de navegación, haga clic en una vista para expandir la vista para mostrar la lista de tareas.  
  
3.  Haga clic en el **búsqueda de actividad** tarea para mostrar una lista de actividades...  
  
4.  Haga clic en una actividad para cargar la página de búsqueda de actividad en el marco de contenido.  
  
5.  En el **consulta** generador elija los parámetros de consulta para la búsqueda.  
  
6.  En el **selector de columnas** seleccionar las columnas que representan los elementos de datos que se devolverán en la búsqueda.  
  
7.  Haga clic en el **Ejecutar consulta** situado en la parte superior derecha del marco de contenido.  
  
     Los resultados se muestran en la **resultados** área no cliente de la página de contenido de búsqueda de actividad.  
  
> [!NOTE]
>  Puede seleccionar hasta 20 columnas.  
  
> [!NOTE]
>  Esta versión no permite agrupar operadores lógicos.
