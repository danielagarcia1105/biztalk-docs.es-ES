# <a name="single-sign-on-event-10771"></a>De sesión único: Evento 10771
## <a name="details"></a>Detalles  
  
|                 |                                                               |
|-----------------|---------------------------------------------------------------|
|  Nombre del producto   |                   Inicio de sesión único (SSO) empresarial                   |
| Versión del producto |  [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]   |
|    Identificador del evento     |                             10771                             |
|  Origen del evento   |                            ENTSSO                             |
|    Componente    |                              N/D                              |
|  Nombre simbólico  |                      ENTSSO_E_NOT_ADMIN                       |
|  Texto del mensaje   | Esta función sólo puede realizarla un administrador local. |
  
## <a name="explanation"></a>Explicación  
 Ciertas funciones sólo pueden ser realizadas por un administrador en el equipo local.  
  
## <a name="user-action"></a>Acción del usuario  
 Solicite ayuda al administrador del sistema para ponerse en contacto con el administrador local de este equipo.
