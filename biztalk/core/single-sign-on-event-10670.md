# <a name="single-sign-on-event-10670"></a>De sesión único: Evento 10670
## <a name="details"></a>Detalles  

|                 |                                                                                                                                      |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                      Inicio de sesión único (SSO) empresarial                                                       |
| Versión del producto |                                      [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                      |
|    Identificador del evento     |                                                                10670                                                                 |
|  Origen del evento   |                                                                ENTSSO                                                                |
|    Componente    |                                                                 N\D                                                                  |
|  Nombre simbólico  |                                                  SSO_INFO_CHANGED_WINDOWS_PASSWORD                                                   |
|  Texto del mensaje   | La contraseña de Windows se cambió correctamente.%r<br /><br /> Id. de seguimiento: %1 %r<br /><br /> Adaptador: %2 %r<br /><br /> Cuenta de Windows: %3 |

## <a name="explanation"></a>Explicación  
 Este evento de información indica que SSO cambió la contraseña de Windows correctamente.  

## <a name="user-action"></a>Acción del usuario  

- No es necesario que el usuario realice ninguna acción.
