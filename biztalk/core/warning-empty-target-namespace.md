# <a name="warning---empty-target-namespace"></a>Advertencia: vacío destino Namespace
**Código de error**  
  
 BEC1007  
  
 **Explicación**  
  
 El **Target Namespace** propiedad de la **esquema** no se estableció el nodo. Aunque no sea estrictamente necesario, se recomienda definir un espacio de nombres de destino único para cada esquema.  
  
 **Acción del usuario**  
  
 A menos que haya una buena razón no a, seleccione la **esquema** nodo de este esquema y, a continuación, en la Microsoft® [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] ventana Propiedades, proporcionar un valor para el **Target Namespace** propiedad que es única a este esquema.
