# <a name="viewing-historical-and-tracked-data"></a>Visualización de datos históricos y de seguimiento
En esta sección se proporciona información específica de la tarea sobre cómo visualizar los datos históricos y de seguimiento de la página Concentrador de grupo de la consola de administración de BizTalk Server.  
  
-   Muestra los pasos de procesamiento que siguen los mensajes. Para obtener más información acerca de los pasos de procesamiento de mensajes, vea [ver el flujo de mensajes](../core/viewing-message-flow.md).  
  
-   Muestra los datos que actualmente se encuentran en la base de datos de seguimiento de BizTalk, así como los datos restaurados desde archivos de datos. Para más información acerca del archivado y los datos de seguimiento, consulte [cómo seleccionar un Live o un origen de datos archivados](../core/how-to-select-a-live-or-archived-data-source.md).  
  
-   Limita el acceso a los datos a los usuarios que disponen de los permisos adecuados. Para obtener más información acerca del acceso a datos, vea [Control de acceso y seguridad de los datos](../core/access-control-and-data-security.md).  
  
-   Permite efectuar una depuración en tiempo real de las orquestaciones. Para más información en tiempo real de depuración, vea [trabajar con la vista depurador de orquestaciones](../core/working-with-the-orchestration-debugger-view.md).
