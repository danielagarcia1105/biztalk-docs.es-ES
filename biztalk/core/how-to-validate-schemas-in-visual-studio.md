# <a name="how-to-validate-schemas-in-visual-studio"></a>Cómo validar esquemas en Visual Studio
Una vez creado un esquema, puede comprobar el trabajo si valida el propio esquema de forma aislada.  
  
 Este tema proporciona instrucciones paso a paso para esta tarea de validación.  
  
### <a name="to-validate-a-schema-in-isolation"></a>Para validar un esquema de forma aislada  
  
1.  En **el Explorador de soluciones**, haga clic en el esquema que desea validar y, a continuación, haga clic en **Validar esquema**.  
  
2.  En la ventana Resultados, vea los resultados. En esta ventana se muestran mensajes para indicar si la operación ha sido correcta o no.  
  
> [!NOTE]
>  Si la ventana Resultados no se abre ni muestra información acerca de si la validación del esquema se ha realizado correctamente o no, puede abrirla manualmente. Para obtener más información acerca de cómo administrar esta ventana, consulte [administrar otras ventanas de Visual Studio](../core/how-to-manage-other-visual-studio-windows.md).
