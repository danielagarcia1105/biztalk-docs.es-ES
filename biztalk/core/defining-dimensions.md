# <a name="defining-dimensions"></a>Definir dimensiones
Microsoft Excel define las dimensiones como categorías que se emplean para organizar los datos de una tabla en niveles que se usarán para realizar análisis. Por ejemplo, una dimensión de datos de ubicación podría contener niveles como ciudad, estado o provincia y región o país. Al crear vistas de BAM en el Asistente para vistas de BAM, puede agregar uno o más de los siguientes tipos de dimensión:  
  
-   [Dimensión de progreso](../core/progress-dimension.md)  
  
-   [Dimensión de datos](../core/data-dimension.md)  
  
-   [Dimensión de tiempo](../core/time-dimension.md)  
  
-   [Dimensión de rango numérico](../core/numeric-range-dimension.md)  
  
 ![](../core/media/bam-olap-cube.gif "bam_olap_cube")  
Datos de agregación precalculados  
  
 Puede agregar nuevas dimensiones en el Asistente para vistas de BAM. Antes de agregar dimensiones, debe usar el asistente para crear vistas de actividades económicas. Para obtener más información sobre cómo utilizar el asistente, consulte [definir actividades económicas y vistas en Excel](../core/defining-business-activities-and-views-in-excel.md).  
  
### <a name="to-add-new-dimensions"></a>Para agregar nuevas dimensiones  
  
1.  En el Asistente para la vista de BAM, haga clic en **siguiente** hasta que vea el **nueva vista de SAE: agregación dimensiones y medidas** página. Haga clic en **nuevas dimensiones**.  
  
2.  Escriba el nombre de la dimensión.  
  
3.  En la lista desplegable, seleccione un tipo de dimensión.  
  
4.  Según el tipo de dimensión seleccionado, rellene los datos apropiados y, a continuación, haga clic en **Aceptar**.
