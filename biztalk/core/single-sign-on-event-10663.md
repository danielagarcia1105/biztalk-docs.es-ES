# <a name="single-sign-on-event-10663"></a>Inicio de sesión único: Evento 10663
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10663|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_EXTERNAL_PASSWORD_CHANGE_RECEIVED|  
|Texto del mensaje|Se recibió un cambio de contraseña externa desde un adaptador.%r<br /><br /> Id. de seguimiento: %1 %r<br /><br /> Adaptador: %2 %r<br /><br /> Cuenta externa: %3 %r<br /><br /> El usuario cliente: %4 %r<br /><br /> Equipo cliente: %5|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que se recibió un cambio de contraseña externa desde un adaptador.  
  
## <a name="user-action"></a>Acción del usuario  
  
-   No es necesario que el usuario realice ninguna acción.
