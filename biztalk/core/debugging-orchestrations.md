# <a name="debugging-orchestrations"></a>Depurar orquestaciones
Mediante el uso de la página Concentrador de grupo en la consola de administración de BizTalk Server, BizTalk permite ver e informar el progreso de alto nivel o sobre detalles de nivel bajo del proceso empresarial y los componentes que implementa, así como realizar el seguimiento de ellos. Además, también se pueden depurar las orquestaciones mediante la escritura de información en el registro de eventos para realizar el seguimiento de la salida o mediante código personalizado para realizar el seguimiento del proceso empresarial.
