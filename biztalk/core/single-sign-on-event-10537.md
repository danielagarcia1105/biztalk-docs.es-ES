# <a name="single-sign-on-event-10537"></a>De sesión único: Evento 10537
## <a name="details"></a>Detalles  

|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  Nombre del producto   |                 Inicio de sesión único (SSO) empresarial                  |
| Versión del producto | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    Identificador del evento     |                           10537                            |
|  Origen del evento   |                           ENTSSO                           |
|    Componente    |                             CO                             |
|  Nombre simbólico  |                   SSO_WARN_SSO_DISABLED                    |
|  Texto del mensaje   |           Actualmente, el sistema SSO está deshabilitado.            |

## <a name="explanation"></a>Explicación  
 Este evento de advertencia indica que el administrador de SSO deshabilitó el sistema SSO.  

## <a name="user-action"></a>Acción del usuario  
 Para resolver esta advertencia, haga lo siguiente:  

- Póngase en contacto con el administrador de SSO para habilitar el sistema SSO. La habilitación de SSO puede realizarse mediante las herramientas de administración de SSO (GUI o línea de comandos).  

  Para obtener más información, consulte los siguientes recursos en la Ayuda de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]:  

- [Cómo habilitar el inicio de sesión único](../core/how-to-enable-sso.md)  

- [Cómo deshabilitar SSO](../core/how-to-disable-sso.md)
