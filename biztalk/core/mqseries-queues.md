# <a name="mqseries-queues"></a>Colas de MQSeries
Para enviar mensajes a colas de MQSeries, puede especificar una cola de definición remota, una cola de alias, una cola de transmisión o una cola local en el administrador de cola.  
  
 Para recibir mensajes de MQSeries Server, puede especificar una cola de transmisión, una cola de alias o una cola local en el administrador de cola.  
  
 Normalmente, se utiliza una definición de cola remota para una cola de envío y una cola de transmisión para recibir mensajes.  
  
 Para obtener información acerca de las diferentes colas de MQSeries, vea la documentación de IBM WebSphere MQ.
