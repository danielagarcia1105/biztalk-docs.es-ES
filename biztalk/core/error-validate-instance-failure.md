# <a name="error---validate-instance-failure"></a>Error: error de la instancia de validación
**Explicación**  
  
 El **Validar instancia** comando no se pudo validar que el archivo de mensaje de instancia indicado se ajusta al esquema indicado.  
  
 **Acción del usuario**  
  
 Examine los mensajes de error que lo acompañan para determinar la causa del error de validación de instancia.
