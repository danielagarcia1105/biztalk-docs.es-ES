# <a name="peoplesoft-add-adapter-wizard"></a>Asistente para agregar adaptador de PeopleSoft
Use el Asistente para agregar adaptador para obtener acceso a la información de metadatos del sistema PeopleSoft.  
  
|Use|Para|  
|--------------|----------------|  
|Seleccionar lista de adaptadores|Seleccione el adaptador en la lista.|  
|SQL Server|Seleccione un servidor SQL Server en la lista y haga clic en **siguiente**.|  
|Base de datos|Se conecta a la misma base de datos que SQL Server.|  
|Puerto|Seleccione el puerto de envío y haga clic en **siguiente**.|
