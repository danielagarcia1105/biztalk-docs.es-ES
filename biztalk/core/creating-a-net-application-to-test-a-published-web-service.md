# <a name="creating-a-net-application-to-test-a-published-web-service"></a>Crear una aplicación .NET para probar un servicio Web publicado
Para probar un servicio Web publicado, puede crear una aplicación de cliente Web de ASP.NET que consuma el servicio Web publicado. La colección de ayuda de [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] contiene un tutorial valioso para crear una aplicación cliente web de ASP.NET. Se puede utilizar el tutorial para probar el servicio Web publicado.  
  
 El tutorial crea una aplicación de cliente Web de ASP.NET, agrega una referencia Web y proporciona código de ejemplo para mostrar cómo obtener acceso al servicio Web publicado. El tutorial le lleva a través de la ejecución de la aplicación en modo de depuración.  
  
 Para obtener información y procedimientos acerca de cómo crear un proyecto de cliente de servicio Web XML, vea "Tutorial: obtener acceso a un XML Web Service Using Visual Basic o Visual C#" en la [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] ayudar a la colección [http://go.microsoft.com/fwlink/?LinkId=62263](http://go.microsoft.com/fwlink/?LinkId=62263). Puede reemplazar el servicio Web utilizado en este ejemplo, TempConvert1, con un servicio Web publicado de BizTalk.
