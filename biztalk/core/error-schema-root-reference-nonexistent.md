# <a name="error---schema-root-reference-nonexistent"></a>Error: referencia raíz de esquema no existe
**Código de error**  
  
 BEC2006  
  
 **Explicación**  
  
 El **referencia raíz** propiedad de la **esquema** nodo hace referencia a un nodo raíz que no existe. El nodo raíz que se hace referencia puede han eliminado o cambiado el nombre después de haberse establecido como el **referencia raíz** valor de propiedad.  
  
 **Acción del usuario**  
  
 Establecer el **referencia raíz** propiedad de la **esquema** nuevo nodo. La lista desplegable de la propiedad incluye todas las elecciones válidas actuales para los nodos raíz.
