# <a name="warning---body-xpath-not-valid"></a>Advertencia: XPath de cuerpo no válido
**Código de error**  
  
 BEC1009  
  
 **Explicación**  
  
 El **XPath de cuerpo** propiedad del nodo raíz indicado en este esquema de sobre hace referencia a un nodo que no es un **registro** nodo, según sea necesario.  
  
 **Acción del usuario**  
  
 Seleccione el nodo raíz indicado y el valor para el **XPath de cuerpo** propiedad que identifique correctamente el nivel superior **registro** nodo del cuerpo del mensaje asociado.
