# <a name="single-sign-on-event-10667"></a>De sesión único: Evento 10667
## <a name="details"></a>Detalles  

|                 |                                                                                                                                         |
|-----------------|-----------------------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                        Inicio de sesión único (SSO) empresarial                                                        |
| Versión del producto |                                       [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                        |
|    Identificador del evento     |                                                                  10667                                                                  |
|  Origen del evento   |                                                                 ENTSSO                                                                  |
|    Componente    |                                                                   N\D                                                                   |
|  Nombre simbólico  |                                          SSO_INFO_SUPPRESSED_DUPLICATE_WINDOWS_PASSWORD_CHANGE                                          |
|  Texto del mensaje   | Se suprimió el cambio de contraseña de Windows duplicado.%r<br /><br /> Id. de seguimiento: %1 %r<br /><br /> Adaptador: %2 %r<br /><br /> Cuenta de Windows: %3 |

## <a name="explanation"></a>Explicación  
 Este evento de información indica que SSO suprimió un cambio de contraseña de Windows duplicado.  

## <a name="user-action"></a>Acción del usuario  

-   No es necesario que el usuario realice ninguna acción.
