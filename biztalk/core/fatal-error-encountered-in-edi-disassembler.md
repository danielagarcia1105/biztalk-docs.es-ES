## <a name="explanation"></a>Explicación  
 Este evento de error, advertencia o información indica que la canalización de recepción no pudo procesar el intercambio entrante debido al error indicado en el desensamblador EDI.  
  
## <a name="user-action"></a>Acción del usuario  
 Para resolver este error, identifique el origen del error indicado y consulte la ayuda del producto para determinar el modo de solucionarlo.
