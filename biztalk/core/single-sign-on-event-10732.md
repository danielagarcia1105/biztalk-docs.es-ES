# <a name="single-sign-on-event-10732"></a>De sesión único: Evento 10732
## <a name="details"></a>Detalles  

|                 |                                                                                                                                        |
|-----------------|----------------------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                       Inicio de sesión único (SSO) empresarial                                                        |
| Versión del producto |                                       [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                       |
|    Identificador del evento     |                                                                 10732                                                                  |
|  Origen del evento   |                                                                 ENTSSO                                                                 |
|    Componente    |                                                                  N\D                                                                   |
|  Nombre simbólico  |                                                    SSO_INFO_PS_SET_WINDOWS_PASSWORD                                                    |
|  Texto del mensaje   | La contraseña de Windows se actualizó correctamente en la base de datos de SSO.%r<br /><br /> Id. de seguimiento: %1 %r<br /><br /> Cuenta de Windows: %2\\%3 |

## <a name="explanation"></a>Explicación  
 Este evento de información indica que la contraseña de Windows se actualizó correctamente en la base de datos de SSO.  

## <a name="user-action"></a>Acción del usuario  

- No es necesaria ninguna acción del usuario  

  Para obtener más información, vea los recursos siguientes:  

- [Sincronización de contraseñas](../core/password-synchronization2.md)
