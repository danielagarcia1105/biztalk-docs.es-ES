# <a name="sendportcollection-node"></a>SendPortCollection (nodo)
El nodo SendPortCollection de un archivo de enlace es el nodo primario de todos los nodos puertoEnvío que contienen información específica acerca de un puerto de envío que se exporta con el archivo de enlace.  
  
## <a name="nodes-in-the-sendportcollection-node"></a>Nodos del nodo SendPortCollection  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[PuertoEnvío](../core/sendport-sendportcollection-node.md)|Grabar|puertoEnvío (ComplexType)|Especifica información acerca de un puerto de envío que se exporta con el archivo de enlace.|No requerido|Valor predeterminado: ninguno|
