# <a name="before-you-use-the-tpe"></a>Antes de usar el TPE
Antes de empezar a trabajar con el Editor de perfiles de seguimiento (TPE), se recomienda que comprenda BizTalk Server y cómo funciona para facilitar la comunicación entre las distintas plataformas y aplicaciones.  
  
 Como programador de soluciones, también es necesario tener un conocimiento especializado acerca de cómo se implementan los procesos empresariales de la organización mediante el uso de orquestaciones de Microsoft BizTalk Server. Debe comprender el modo de uso del Diseñador de orquestaciones para producir orquestaciones. Para obtener más información acerca de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] orquestaciones, consulte [sobre orquestaciones](../core/about-orchestrations.md).  
  
> [!IMPORTANT]
>  Para usar el TPE, debe usar una instalación del producto BizTalk Server completa o personalizada y compruebe que su elección incluye las opciones "SDK y herramientas de programadores" o "Herramientas de administración". Para obtener más información, consulte [Introducción a la instalación de BizTalk Server 2013 y 2013 R2](http://msdn.microsoft.com/library/8041926c-cfc9-4eaf-9c28-a2c6e8015bc5).  
  
> [!NOTE]
>  TPE no realiza el seguimiento de partes de mensajes .NET. No puede ver el esquema de mensajes para estos tipos de mensajes.
