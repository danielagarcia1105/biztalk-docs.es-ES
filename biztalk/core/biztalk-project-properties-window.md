# <a name="biztalk-project-properties-window"></a>Ventana de propiedades del proyecto de BizTalk
La ventana de propiedades del proyecto para un proyecto BizTalk de [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] contiene las propiedades siguientes:  
  
## <a name="bpel-compliance"></a>Compatibilidad con BPEL  
 Especifica si se debe generar salida compatible con el lenguaje de ejecución de procesos empresariales (BPEL).  
  
## <a name="project-file"></a>Archivo de proyecto  
 Nombre del archivo que contiene la generación, la configuración y otra información sobre el proyecto.  
  
## <a name="project-folder"></a>Carpeta Proyecto  
 Ubicación del archivo del proyecto.
