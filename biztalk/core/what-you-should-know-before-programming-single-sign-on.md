# <a name="what-you-should-know-before-programming-single-sign-on"></a>Lo que debe saber antes de programar el inicio de sesión único
Para utilizar esta documentación de manera efectiva, debe estar familiarizado con lo siguiente:  
  
-   Sistemas operativos Microsoft [!INCLUDE[btsPlatformsComApis](../includes/btsplatformscomapis-md.md)]. Debe estar especialmente familiarizado con las características de seguridad de Windows.  
  
-   Características administrativas del inicio de sesión único (SSO) empresarial, especialmente el modo de llevar a cabo acciones administrativas mediante la interfaz de usuario.  
  
 En función de la API y del desarrollo que esté utilizando, también debe estar familiarizado con las siguientes interfaces:  
  
-   COM  
  
-   .NET Framework y Common Language Runtime  
  
-   Red de Windows y, en concreto, cómo enviar y recibir vales
