# <a name="developing-custom-components"></a>Desarrollar componentes personalizados
Los componentes personalizados se desarrollan y utilizan en BizTalk Server para modificar o extender algunos elementos de infraestructura de BizTalk Server.  Principalmente, esto incluye elementos tales como: enviar o recibir componentes del adaptador que se basan en el marco de trabajo y se usa para controlar los protocolos de transporte específicos; componentes de canalización usados en cualquiera de envío o recepción etapas de canalización; y functoids usados en las asignaciones.  Puede extender [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] con código personalizado de varias maneras. En los temas siguientes se describe cómo hacerlo.
