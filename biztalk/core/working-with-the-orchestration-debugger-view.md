# <a name="working-with-the-orchestration-debugger-view"></a>Trabajar con la vista Depurador de orquestaciones
Para obtener acceso a la vista Depurador de orquestaciones, haga clic con el botón secundario en una celda de una lista Resultados y seleccione Depurador de orquestaciones.  
  
 El Depurador de orquestaciones presenta la actividad de procesamiento técnico de una sola instancia. La vista muestra una imagen representada de la orquestación.  
  
 Desde la vista Depurador de orquestaciones, puede ir a la vista Flujo de mensajes.
