# <a name="single-sign-on-event-10500"></a>Inicio de sesión único: Evento 10500
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10500|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_ERROR_EVENT_LOG_FAILED|  
|Texto del mensaje|El sistema de registro de eventos detectó un error. Es posible que los mensajes del registro de eventos no sean correctos para esta aplicación.|  
  
## <a name="explanation"></a>Explicación  
 Este evento de error indica que el registro de eventos generó un error.  
  
## <a name="user-action"></a>Acción del usuario  
 Para resolver este error, realice lo siguiente:  
  
-   No se requiere ninguna acción.
