# <a name="information-worker-technologies"></a>Tecnologías de los trabajadores de la información
El motor de Microsoft BizTalk Server proporciona compatibilidad para los procesos empresariales automatizados que abarquen varias aplicaciones. Pero una vez que estos procesos se han creado, los trabajadores de información que las utilizan, trabajadores de una empresa, no son programadores: tienen otros requisitos. Por ejemplo, tal vez necesiten supervisar distintos aspectos de la actividad relacionados con los negocios o trabajar con algunos de sus socios comerciales de formas diferentes.  
  
 BizTalk Server proporciona varios componentes que solucionar estos problemas. Estos componentes se agrupan en la Supervisión de la actividad económica, que se describe en esta sección.
