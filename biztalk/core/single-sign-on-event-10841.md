# <a name="single-sign-on-event-10841"></a>De sesión único: Evento 10841
## <a name="details"></a>Detalles  
  
|                 |                                                                                |
|-----------------|--------------------------------------------------------------------------------|
|  Nombre del producto   |                           Inicio de sesión único (SSO) empresarial                            |
| Versión del producto |           [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]           |
|    Identificador del evento     |                                     10841                                      |
|  Origen del evento   |                                     ENTSSO                                     |
|    Componente    |                                      N/D                                       |
|  Nombre simbólico  |                   ENTSSO_E_MAPPINGS_ONLY_FOR_DOMAIN_ACCOUNTS                   |
|  Texto del mensaje   | Las asignaciones se pueden crear solo para cuentas de dominio de Windows (no para cuentas locales). |
  
## <a name="explanation"></a>Explicación  
 Las asignaciones se pueden crear solo para cuentas de dominio de Windows (no para cuentas locales).  
  
## <a name="user-action"></a>Acción del usuario  
 Para obtener más información, consulte [asignaciones de SSO](../core/sso-mappings.md).
