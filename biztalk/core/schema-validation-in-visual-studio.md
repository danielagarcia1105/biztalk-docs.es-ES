# <a name="schema-validation-in-visual-studio"></a>Validación de esquemas en Visual Studio
Después de que haya terminado de crear un esquema o en cualquier otro momento durante su creación, lo puede validar para determinar si tiene alguna incoherencia interna u otros problemas que puedan impedir que se utilice eficazmente para procesar los mensajes de instancia.  
  
 Para validar un esquema, utilice la **Validar esquema** comando en el menú contextual asociado al esquema en el Explorador de soluciones. Los resultados de la validación aparecen en la ventana Resultados de [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)].  
  
 Para obtener instrucciones detalladas paso a paso acerca de cómo validar un esquema, vea [validar esquemas](../core/how-to-validate-schemas-in-visual-studio.md).
