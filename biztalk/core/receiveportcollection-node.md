# <a name="receiveportcollection-node"></a>ReceivePortCollection (nodo)
El nodo ReceivePortCollection de un archivo de enlace es el nodo principal para todos los nodos puertoRecepción que contienen información específica acerca de un puerto de recepción que se exporta con el archivo de enlace.  
  
## <a name="nodes-in-the-receiveportcollection-node"></a>Nodos del nodo ReceivePortCollection  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[PuertoRecepción](../core/receiveport-receiveportcollection-node.md)|Grabar|puertoRecepción (ComplexType)|Especifica información acerca de un puerto de recepción que se exporta con el archivo de enlace.|No requerido|Valor predeterminado: ninguno|
