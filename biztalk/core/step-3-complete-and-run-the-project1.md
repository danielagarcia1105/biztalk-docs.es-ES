# <a name="step-3-complete-and-run-the-project"></a>Paso 3: Completar y ejecutar el proyecto
Tras agregar la clave de ensamblado y enlazar los puertos, ya puede probar la orquestación.  
  
 Para la prueba, puede usar el ejemplo Sample XML BeginDoc si ha utilizado el objeto idéntico. Si no es así, es posible [paso 4: crear un XML de ejemplo de BeginDoc](../core/step-4-create-a-sample-xml-begindoc2.md) tal y como se trata de un desencadenador de inicio de todo el proceso.  
  
### <a name="to-complete-and-run-the-project"></a>Para completar y ejecutar el proyecto  
  
1.  En el Explorador de soluciones, haga clic en el proyecto y, a continuación, seleccione **generar**.  
  
2.  Haga clic en el proyecto y, a continuación, seleccione **implementar**.  
  
3.  En el Explorador de BizTalk, seleccione **enlazar**, **dar de alta** y **iniciar** para probar su orquestación.
