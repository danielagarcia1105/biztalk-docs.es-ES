# <a name="single-sign-on-event-10846"></a>Inicio de sesión único: Evento 10846
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10846|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_HOST_GROUP_ONE_ACCOUNT_ONLY|  
|Texto del mensaje|Las aplicaciones de grupo host no pueden tener varias cuantas especificadas para la cuenta de Windows.|  
  
## <a name="explanation"></a>Explicación  
 Las aplicaciones de grupo host no pueden tener varias cuantas especificadas para la cuenta de Windows.  
  
## <a name="user-action"></a>Acción del usuario  
 Para obtener más información sobre aplicaciones de grupo Host, consulte [aplicaciones afiliadas de SSO](../core/sso-affiliate-applications.md).
