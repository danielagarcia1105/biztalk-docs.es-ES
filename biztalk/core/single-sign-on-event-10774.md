# <a name="single-sign-on-event-10774"></a>De sesión único: Evento 10774
## <a name="details"></a>Detalles  
  
|                 |                                                                                      |
|-----------------|--------------------------------------------------------------------------------------|
|  Nombre del producto   |                              Inicio de sesión único (SSO) empresarial                               |
| Versión del producto |              [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]              |
|    Identificador del evento     |                                        10774                                         |
|  Origen del evento   |                                        ENTSSO                                        |
|    Componente    |                                         N/D                                          |
|  Nombre simbólico  |                          ENTSSO_E_SSO_NOT_IN_NEW_SSO_ADMIN                           |
|  Texto del mensaje   | La cuenta de servicio SSO no es miembro de la cuenta de administradores de SSO especificada. |
  
## <a name="explanation"></a>Explicación  
 Para realizar cambios en la cuenta de administradores de SSO, debe ser miembro de esa cuenta.  
  
## <a name="user-action"></a>Acción del usuario  
 Solicite ayuda al administrador del sistema para ponerse en contacto con un miembro de la cuenta de administradores de SSO.
