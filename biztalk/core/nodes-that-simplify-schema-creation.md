# <a name="nodes-that-simplify-schema-creation"></a>Nodos que simplifican la creación de esquemas
Los nodos que simplifican la creación de esquemas son una categoría miscelánea de dos tipos de nodos que no se pueden clasificar en ninguna de las demás categorías y que facilitan la creación de esquemas. El primer nodo secundario, el **esquema** nodo, corresponde a la **esquema** elemento de lenguaje de definición (XSD) de esquemas XML, que es el elemento que actúa como el elemento más externo en todos los esquemas XSD.  
  
 El otro nodo de simplificación desempeña un rol más importante en este sentido. El **grupo de atributos** nodo corresponde directamente a la **attributeGroup** elemento en XSD y proporciona un mecanismo para definir un grupo de atributos relacionados, que siempre ocurren juntos, en un único equipo ubicación y, a continuación, utilizar dicha definición de grupo en varias ubicaciones en todo el esquema. En el resto de esta sección se proporciona información adicional sobre estos nodos.
