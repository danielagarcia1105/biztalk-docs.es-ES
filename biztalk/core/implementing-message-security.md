# <a name="implementing-message-security"></a>Implementar seguridad en los mensajes
Puede configurar el entorno de Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] para usar certificados para el envío y recepción de mensajes firmados y cifrados. Mediante los certificados para el cifrado y la aplicación de firmas digitales, BizTalk Server puede realizar lo siguiente:  
  
-   Enviar y recibir datos de confianza.  
  
-   Asegurarse de que los datos que procesa son seguros.  
  
-   Asegurarse de que las entidades autorizadas reciben sus mensajes.  
  
-   Asegurarse de que recibe mensajes de las entidades autorizadas.  
  
 En esta sección se proporciona información acerca de cómo configurar canalizaciones, ubicaciones de recepción, puertos y el entorno de BizTalk Server para implementar seguridad en los mensajes.  
  
 Para obtener más información acerca de la seguridad de mensaje, consulte [planear la seguridad de mensaje](../core/planning-message-security.md).
