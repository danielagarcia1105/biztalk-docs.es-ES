# <a name="single-sign-on-event-10839"></a>Inicio de sesión único: Evento 10839
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10839|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_MAX_PASSWORD|  
|Texto del mensaje|La longitud máxima de contraseña es de 256 caracteres.|  
  
## <a name="explanation"></a>Explicación

## <a name="user-action"></a>Acción del usuario  
 Use una contraseña que respete estas pautas.
