# <a name="tutorials-using-the-microsoft-biztalk-adapter-for-tibco-rendezvous"></a>Tutoriales: Usar el adaptador de Microsoft BizTalk para TIBCO Rendezvous
El Adaptador de Microsoft BizTalk para TIBCO Rendezvous proporciona un entorno de desarrollo y en tiempo de ejecución para la automatización y la administración de procesos de línea empresarial. En esta sección se describen dos ejemplos de SDK que presentan un ejercicio completo de configuración e implementación del adaptador para que pueda empezar a trabajar rápidamente.  
  
 Estos tutoriales incluyen información detallada sobre la adición de un adaptador a BizTalk Server, la creación de puertos de envío y recepción adecuados, la creación e implementación de ejemplos y el procesamiento de datos de prueba con los ejemplos.
