# <a name="how-to-use-web-services-in-orchestrations"></a>Cómo usar servicios web en orquestaciones
Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] proporciona compatibilidad integrada para servicios Web. BizTalk Server permite reutilizar y agregar todos los servicios Web existentes en las orquestaciones. Las orquestaciones también se publican o se exponen como servicios Web para diferenciar la lógica de servicios Web de la lógica de procesos empresariales.  
  
## <a name="examples-of-using-web-services"></a>Ejemplos de uso de servicios Web  
 En los siguientes ejemplos se muestran algunos modos en los que se pueden utilizar los servicios Web:  
  
-   Descargue el ejemplo SDK "Publicar y consumir Web Services with SOAP Headers" de [http://go.microsoft.com/fwlink/?LinkId=73703](http://go.microsoft.com/fwlink/?LinkId=73703).  
  
-   Descargue el ejemplo SDK "Consume Web Services with Array Parameters" de [http://go.microsoft.com/fwlink/?LinkId=73703](http://go.microsoft.com/fwlink/?LinkId=73703).  
  
-   Descargue el ejemplo SDK "Consuming Web Services", un ejemplo de escenario exclusivo de mensajería, de [http://go.microsoft.com/fwlink/?LinkId=73703](http://go.microsoft.com/fwlink/?LinkId=73703).
