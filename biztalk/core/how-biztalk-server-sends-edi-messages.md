# <a name="how-biztalk-server-sends-edi-messages"></a>Cómo BizTalk Server envía mensajes EDI
Si [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] envía un mensaje EDI, lleva a cabo la búsqueda de acuerdo y la detección de esquemas, valida el mensaje, envía una confirmación (si fuese necesario) y serializa el lote de EDI. Este procesamiento lo realiza el ensamblador EDI en la canalización de envío EDI.
