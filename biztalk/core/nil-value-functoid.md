# <a name="nil-value-functoid"></a>Valor nulo (functoid)

## <a name="description"></a>Description
El **valor nulo** functoid establece un elemento de destino como **Nil**. Este functoid resulta útil para incluir elementos vacíos en el mensaje de salida. Pueden combinarse con otros functoids para crear un **Nil** valor basándose en condiciones específicas.
