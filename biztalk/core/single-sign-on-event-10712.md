# <a name="single-sign-on-event-10712"></a>De sesión único: Evento 10712
## <a name="details"></a>Detalles  

|                 |                                                                                                                                                                                    |
|-----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                                             Inicio de sesión único (SSO) empresarial                                                                              |
| Versión del producto |                                                             [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                             |
|    Identificador del evento     |                                                                                       10712                                                                                        |
|  Origen del evento   |                                                                                       ENTSSO                                                                                       |
|    Componente    |                                                                                        N\D                                                                                         |
|  Nombre simbólico  |                                                                          SSO_WARN_PS_NOTIFICATION_TOO_OLD                                                                          |
|  Texto del mensaje   | Se descartó una notificación porque era demasiado antigua.%r<br /><br /> Id. de seguimiento: %1 %r<br /><br /> Adaptador: %2 %r<br /><br /> Tipo de notificación: %3 %r<br /><br /> Datos adicionales: %4 |

## <a name="explanation"></a>Explicación  
 Este evento de advertencia indica que se descartó una notificación de sincronización de contraseñas porque era demasiado antigua.  

## <a name="user-action"></a>Acción del usuario  

- No es necesario que el usuario realice ninguna acción.  

  Para obtener más información, vea los recursos siguientes:  

- [Sincronización de contraseñas](../core/password-synchronization2.md)
