# <a name="success---validate-instance-generated-xml-output"></a>Correcto: Validar instancia generó resultado XML
**Explicación**  
  
 La validación de instancia generó el archivo de salida XML indicado. Este archivo contiene el Equivalente XML del mensaje de instancia de entrada con formato distinto de XML.  
  
 **Acción del usuario**  
  
 Según corresponda, inspeccione el archivo de salida XML generado.
