# <a name="error---property-schema-structure-not-valid"></a>Error: estructura de esquema de propiedad no es válido
**Código de error**  
  
 BEC2007  
  
 **Explicación**  
  
 Se ha detectado que este esquema de propiedades tiene una estructura que no es válida. La estructura de esquemas de propiedades se limita a uno o varios **elemento de campo** nodos como elementos secundarios de la **esquema** nodo.  
  
 **Acción del usuario**  
  
 Corrija la estructura de este esquema de propiedad para que la **esquema** nodo solo tiene **elemento de campo** nodos como elementos secundarios.
