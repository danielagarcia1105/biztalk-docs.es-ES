# <a name="single-sign-on-event-10722"></a>De sesión único: Evento 10722
## <a name="details"></a>Detalles  

|                 |                                                                                                 |
|-----------------|-------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                    Inicio de sesión único (SSO) empresarial                                    |
| Versión del producto |                   [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                    |
|    Identificador del evento     |                                              10722                                              |
|  Origen del evento   |                                             ENTSSO                                              |
|    Componente    |                                               N\D                                               |
|  Nombre simbólico  |                                SSO_INFO_REPLAY_FILE_CLOSED_IDLE                                 |
|  Texto del mensaje   | Se cerró el archivo de reproducción porque no está actualmente en uso.%r<br /><br /> Nombre de archivo: %1 |

## <a name="explanation"></a>Explicación
