# <a name="schemas-node"></a>Esquemas (nodo)
Use la **esquemas** nodo para mostrar los esquemas implementados en los esquemas de aplicación y del sistema actuales que están disponibles en BizTalk. Haga clic en el **esquemas** nodo para mostrar los siguientes comandos:  
  
-   **Ocultar esquemas de propiedades**. Cuando está activada, filtra los esquemas utilizados para promocionar valores específicos de un mensaje de instancia en el contexto del mensaje. Al seleccionar este elemento de menú, se sustituye en el **esquemas** menú del nodo por **mostrar esquemas de propiedades**.  
  
-   **Mostrar esquemas de propiedades**. Cuando está activada, filtra los esquemas de propiedad. Al seleccionar este elemento de menú, se sustituye en el **esquemas** menú del nodo por **ocultar esquemas de propiedades**.
