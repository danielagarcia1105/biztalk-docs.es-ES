# <a name="btadchangerequestaction"></a>BTAD_ChangeRequestAction
BTAD_ChangeRequestAction contiene un valor que indica una acción de solicitud de cambio.  
  
## <a name="remarks"></a>Comentarios  
 En la siguiente tabla se describen los posibles valores para BTAD_ChangeRequestAction.  
  
|Value|Description|  
|-----------|-----------------|  
|Crear|Importar sin sobrescritura especificada.|  
|Update|Importar o instalar con sobrescritura especificada.|  
|Eliminar|Desinstalar|
