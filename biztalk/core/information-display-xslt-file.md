# <a name="information---display-xslt-file"></a>Información: Mostrar archivo XSLT
**Código de error**  
  
 btm1055  
  
 **Explicación**  
  
 Se ha escrito en el archivo indicado el XSLT generado para realizar la asignación especificada de los mensajes de instancia que se ajustan al esquema de origen en mensajes de instancia que se ajustan al esquema de destino.  
  
 **Acción del usuario**  
  
 Puede obtener acceso al XSLT resultante mediante el vínculo proporcionado y utilizarlo para realizar varias tareas en otras herramientas, como un depurador XSLT.
