# <a name="creating-send-ports"></a>Creación de puertos de envío
Al crear un puerto de envío que usa el inicio de sesión único (SSO) y la aplicación afiliada, no rellena los parámetros de Nombre de usuario y Contraseña. Seleccionar **Sí** para **usar SSO** y seleccione las aplicaciones afiliadas SSO que creó para representar el sistema TIBCO Enterprise Message Service en la lista.
