# <a name="nesting-orchestrations"></a>Anidar orquestaciones
Puede usar el **orquestación de llamada** forma y el **Iniciar orquestación** forma que se va a invocar una orquestación desde otra. Puede anidar las orquestaciones en niveles de profundidad arbitrarios así: por ejemplo, una orquestación de llamada puede llamar a una orquestación de terceros, que puede llamar a un cuarto y así sucesivamente.
