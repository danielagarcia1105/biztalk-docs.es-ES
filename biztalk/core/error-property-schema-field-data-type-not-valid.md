# <a name="error---property-schema-field-data-type-not-valid"></a>Error: tipo de datos de campo de esquema de propiedad no es válido
**Código de error**  
  
 BEC2010  
  
 **Explicación**  
  
 El **tipo de datos** propiedad de **elemento de campo** nodos en esquemas de propiedad deben establecerse en uno de los tipos de datos indicados. No puede utilizar otros tipos de datos en esquemas de propiedades porque la promoción de propiedades se limita a los datos de este tipo.  
  
 **Acción del usuario**  
  
 Seleccione el functoid **elemento de campo** nodo y, a continuación, en el Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] ventana de propiedades, cambiar la **tipo de datos** propiedad en uno de los tipos de datos indicados.
