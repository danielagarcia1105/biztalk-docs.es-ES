# <a name="single-sign-on-event-10849"></a>De sesión único: Evento 10849
## <a name="details"></a>Detalles  
  
|                 |                                                                                  |
|-----------------|----------------------------------------------------------------------------------|
|  Nombre del producto   |                            Inicio de sesión único (SSO) empresarial                             |
| Versión del producto |            [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]            |
|    Identificador del evento     |                                      10849                                       |
|  Origen del evento   |                                      ENTSSO                                      |
|    Componente    |                                       N/D                                        |
|  Nombre simbólico  |                     ENTSSO_E_DIRECT_SYNC_NOT_ALLOWED_CREATE                      |
|  Texto del mensaje   | No se puede crear una aplicación con el marcador "sincronización directa de contraseñas" especificado. |
  
## <a name="explanation"></a>Explicación  
 No se puede crear una aplicación con el marcador "sincronización directa de contraseñas" especificado.  
  
## <a name="user-action"></a>Acción del usuario  
 Cree la aplicación sin el marcador de sincronización directa de contraseñas. A continuación, agregue y cree los campos, habilite la aplicación y especifique el marcador de sincronización directa de contraseñas.
