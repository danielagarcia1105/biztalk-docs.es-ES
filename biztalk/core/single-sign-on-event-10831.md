# <a name="single-sign-on-event-10831"></a>Inicio de sesión único: Evento 10831
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10831|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_SSO_DATABASE_EXISTS|  
|Texto del mensaje|La base de datos de SSO especificada ya existe.|  
  
## <a name="explanation"></a>Explicación  
 Durante la configuración, se intentó crear una base de datos que ya existe.  
  
## <a name="user-action"></a>Acción del usuario  
 Compruebe el sistema o asigne otro nombre a la base de datos.
