# <a name="configuring-flow-control-shapes"></a>Configurar las formas de Control de flujo
El Diseñador de orquestaciones proporciona una serie de formas que se pueden utilizar para controlar el flujo de la orquestación, ya sea explícitamente mediante la **acciones paralelas** forma o condicionalmente mediante el **decisión**, **Escuchar**, o **bucle** formas. También proporciona la **retraso** forma para que pausar el proceso empresarial y proporciona la **Suspend** forma y el **Terminate** forma suspender y finalizar el orquestación.
