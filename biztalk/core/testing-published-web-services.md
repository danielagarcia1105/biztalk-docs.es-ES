# <a name="testing-published-web-services"></a>Probar de servicios web publicados
A medida que desarrolle un servicio Web, puede ser interesante volver a publicarlo para probarlo. Cada vez que vuelva a publicar el servicio Web, debe restablecer los servicios de Internet Information Services (IIS).  
  
 Para probar el servicio Web publicado puede crear una aplicación de cliente de servicio Web o puede utilizar un explorador como Internet Explorer.
