# <a name="how-to-use-the-schema-collection-property-editor"></a>Cómo usar el Editor de propiedades de la colección de esquema
El Editor de propiedades de colección de esquema se utiliza para seleccionar esquemas para el componente de canalización.  
  
### <a name="to-use-the-schema-collection-property-editor"></a>Para utilizar el Editor de propiedades de colección de esquema  
  
1.  En el Editor de propiedades de colección de esquema, en el **esquemas disponibles** , seleccione los esquemas que deben estar disponibles para el componente de canalización. Utilice la tecla CTRL para seleccionar más de un esquema.  
  
2.  Haga clic en **Agregar**.  
  
     Los esquemas seleccionados aparecerán ahora en el **esquemas agregados** cuadro.  
  
3.  Haga clic en **Aceptar**.
