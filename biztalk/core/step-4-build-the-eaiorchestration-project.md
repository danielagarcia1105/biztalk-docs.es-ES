# <a name="step-4-build-the-eaiorchestration-project"></a>Paso 4: Generar el proyecto EAIOrchestration
![Paso 4 de 4](../adapters-and-accelerators/adapter-oracle-ebs/media/step-4of4.gif "Step_4of4")  
  
 **Tiempo en completarse:** 5 minutos  
  
 **Objetivo:** en este paso, se compila el proyecto EAIOrchestration en un ensamblado.  
  
 **Propósito:** BizTalk Server requiere que todos los artefactos que se compilarán en ensamblados. NET.  
  
## <a name="prerequisites"></a>Requisitos previos  
 Tenga en cuenta los siguientes requisitos antes de iniciar este paso:  
  
-   Antes de comenzar este paso, debe completar los siguientes:  
  
    -   [Paso 1: Agregar el proyecto EAIOrchestration a la solución](../core/step-1-add-eaiorchestration-project-to-the-solution.md)  
  
    -   [Paso 2: Definir el proceso empresarial](../core/step-2-define-the-business-process.md)  
  
    -   [Paso 3: Agregar puertos a la orquestación](../core/step-3-add-ports-to-the-orchestration.md)  
  
## <a name="procedures"></a>Procedimientos  
  
#### <a name="to-build-the-eaiorchestration-project"></a>Para crear el proyecto EAIOrchestration  
  
-   En el Explorador de soluciones, haga clic en **EAIOrchestration**y, a continuación, haga clic en **generar**.  
  
     En la parte inferior de la pantalla debe aparecer:  
  
    ```  
    ========== Build: 1 succeeded or up-to-date, 0 failed, 0 skipped ==========  
    ```  
  
## <a name="what-did-i-just-do"></a>Síntesis  
 En este paso, ha compilado el proyecto EAIOrchestration.
