# <a name="single-sign-on-event-10716"></a>Inicio de sesión único: Evento 10716
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10716|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_NEW_REPLAY_DIR|  
|Texto del mensaje|El nuevo directorio de archivos de reproducción se creó correctamente.%r<br /><br /> El usuario cliente: %1 %r<br /><br /> Directorio de archivos de reproducción: %2|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que la sincronización de contraseñas de SSO creó un nuevo directorio de archivos de reproducción correctamente.  
  
## <a name="user-action"></a>Acción del usuario  
  
-   No es necesario que el usuario realice ninguna acción.
