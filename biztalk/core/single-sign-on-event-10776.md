# <a name="single-sign-on-event-10776"></a>Inicio de sesión único: Evento 10776
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10776|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_INVALID_ACCOUNT_FORMAT|  
|Texto del mensaje|Formato de nombre de cuenta no válido. Las cuentas de dominio deben incluir el nombre de dominio. Las cuentas locales no deben incluir un nombre de dominio o equipo.|  
  
## <a name="explanation"></a>Explicación  
 Formato de nombre de cuenta no válido. Las cuentas de dominio deben incluir el nombre de dominio. Las cuentas locales no deben incluir un nombre de dominio o equipo.  
  
## <a name="user-action"></a>Acción del usuario  
 Especifique otro nombre de cuenta.
