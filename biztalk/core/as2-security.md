# <a name="as2-security"></a>Seguridad de AS2
La característica AS2 nativa de Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] le permite mantener la seguridad de los mensajes AS2 mediante claves de cifrado, firmas digitales y certificados. Para ello, es necesario que [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] y los certificados dispongan de la configuración adecuada.  
  
 Para obtener información acerca de la estructura de un mensaje de AS2 cifrado o firmado, vea [mensajes AS2](../core/as2-messages.md).
