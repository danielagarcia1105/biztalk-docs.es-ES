# <a name="access-control-to-business-information"></a>Control de acceso a información empresarial
Únicamente los administradores de BizTalk Server pueden obtener acceso a los datos de seguimiento, tales como las propiedades de mensaje y los datos de cuerpo de mensaje, en la base de datos de seguimiento de BizTalk. Un Operador de BizTalk Server tiene acceso a los datos de flujo de mensajes de la base de datos de seguimiento de BizTalk.
