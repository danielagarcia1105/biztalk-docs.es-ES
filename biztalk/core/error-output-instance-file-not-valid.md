# <a name="error---output-instance-file-not-valid"></a>Error: archivo de instancia de salida no válido
**Explicación**  
  
 El valor especificado para la **nombre de archivo de instancia de salida** propiedad del archivo de esquema no es un nombre de archivo válido.  
  
 **Acción del usuario**  
  
 Haga clic en el archivo de esquema correspondiente en Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] el Explorador de soluciones y, a continuación, haga clic en **propiedades** para abrir el **páginas de propiedades** cuadro de diálogo para este esquema. En este cuadro de diálogo, en la **General** , haga clic en el botón de puntos suspensivos (**...** ) botón en el campo de valor de la **nombre de archivo de instancia de salida** propiedad y, a continuación, utilice la **Seleccionar archivo de salida** cuadro de diálogo para identificar el archivo que va a contener la instancia de salida Mensaje. También puede escribir el nombre de archivo directamente en el campo de valor de la **nombre de archivo de instancia de salida** propiedad.
