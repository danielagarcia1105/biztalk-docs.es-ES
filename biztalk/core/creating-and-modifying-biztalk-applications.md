# <a name="creating-and-modifying-biztalk-applications"></a>Crear y modificar aplicaciones de BizTalk
En esta sección se describe cómo usar la consola de administración de BizTalk Server o herramienta de línea de comandos BTSTask para crear, configurar y modificar aplicaciones de BizTalk.  
  
 Para obtener información general acerca de cómo crear, administrar, implementar y actualizar aplicaciones de BizTalk, consulte [implementación de aplicación de BizTalk de descripción y administración](../core/understanding-biztalk-application-deployment-and-management.md). Para obtener información acerca de cómo administrar artefactos, incluida la edición de sus propiedades, vea [administrar artefactos](../core/managing-artifacts.md). Para obtener instrucciones sobre cómo implementar aplicaciones de BizTalk, consulte [implementar aplicaciones de BizTalk](../core/deploying-biztalk-applications.md).
