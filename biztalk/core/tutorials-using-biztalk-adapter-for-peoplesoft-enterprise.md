# <a name="tutorials-using-biztalk-adapter-for-peoplesoft-enterprise"></a>Tutoriales: Usar el adaptador de BizTalk para PeopleSoft Enterprise
Microsoft BizTalk Adapter para PeopleSoft Enterprise es un entorno de desarrollo y tiempo de ejecución para la administración y automatización de la administración del proceso de línea de negocio. En esta sección se describen dos ejemplos de SDK que presentan un ejercicio completo de configuración e implementación del adaptador para que pueda empezar a trabajar rápidamente.  
  
 Estos tutoriales incluyen información detallada sobre la adición de un adaptador a BizTalk Server, la creación de puertos de envío y recepción adecuados, la creación e implementación de ejemplos y el procesamiento de datos de prueba con los ejemplos.
