# <a name="how-to-use-an-xml-schema-as-a-data-source"></a>Cómo utilizar un esquema XML como origen de datos
### <a name="to-specify-an-xml-schema-as-a-data-source"></a>Para especificar un esquema XML como origen de datos  
  
1. En el Explorador de hechos, haga clic en el **esquemas XML** ficha.  
  
2. Haga clic en el **esquemas** carpeta y, a continuación, haga clic en **examinar**.  
  
3. En el **archivos de esquema** cuadro de diálogo, seleccione un archivo XSD y, a continuación, haga clic en **abierto**.
