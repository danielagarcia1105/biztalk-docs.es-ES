# <a name="instance-validation"></a><span data-ttu-id="dc26c-102">Validación de instancia</span><span class="sxs-lookup"><span data-stu-id="dc26c-102">Instance Validation</span></span>
<span data-ttu-id="dc26c-103">El Editor de BizTalk, se invoca el **IInstanceValidator.ValidateInstance** método de una extensión cuando se cumplen las condiciones siguientes:</span><span class="sxs-lookup"><span data-stu-id="dc26c-103">BizTalk Editor invokes the **IInstanceValidator.ValidateInstance** method of an extension when the following conditions are met:</span></span>  
  
-   <span data-ttu-id="dc26c-104">La extensión está establecida como estándar mediante el uso de la **estándar** propiedad de la **esquema** nodo.</span><span class="sxs-lookup"><span data-stu-id="dc26c-104">The extension is set as standard by using the **Standard** property of the **Schema** node.</span></span>  
  
-   <span data-ttu-id="dc26c-105">En el **páginas de propiedades** cuadro de diálogo asociado con el esquema, el **genere el tipo de salida de instancia** propiedad está establecida en **nativo.**</span><span class="sxs-lookup"><span data-stu-id="dc26c-105">On the **Property Pages** dialog box associated with the schema, the **Generate Instance Output Type** property is set to **Native.**</span></span>  
  
-   <span data-ttu-id="dc26c-106">En el **páginas de propiedades** cuadro de diálogo asociado con el esquema, el **nombre de archivo de instancia de entrada** propiedad se establece en el nombre de un archivo que contiene el mensaje de instancia que se debe validar.</span><span class="sxs-lookup"><span data-stu-id="dc26c-106">On the **Property Pages** dialog box associated with the schema, the **Input Instance Filename** property is set to the name of a file containing the instance message to be validated.</span></span>  
  
 <span data-ttu-id="dc26c-107">El archivo especificado en el **nombre de archivo de instancia de entrada** propiedad se pasa como un parámetro a la **IInstanceValidator.ValidateInstance** método.</span><span class="sxs-lookup"><span data-stu-id="dc26c-107">The file specified in the **Input Instance Filename** property is passed as a parameter to the **IInstanceValidator.ValidateInstance** method.</span></span>  
  
 <span data-ttu-id="dc26c-108">Si se producen errores, mensajes de error se devuelven como una matriz de **IValidationInfo** objetos y se muestran en la [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] ventana Lista de tareas.</span><span class="sxs-lookup"><span data-stu-id="dc26c-108">If errors occur, error messages are returned as an array of **IValidationInfo** objects, and are displayed in the [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] Task List window.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="dc26c-109">Si el esquema contiene instrucciones pattern y el archivo pasado a la **ValidateInstance** método generado mediante la correspondiente **generar instancia** comando, no puede pasar el mensaje de instancia validación.</span><span class="sxs-lookup"><span data-stu-id="dc26c-109">If the schema contains pattern statements, and the file passed to the **ValidateInstance** method was generated by using the corresponding **Generate Instance** command, the instance message may not pass validation.</span></span>
