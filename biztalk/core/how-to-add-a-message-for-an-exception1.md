# <a name="how-to-add-a-message-for-an-exception"></a>Cómo agregar un mensaje para una excepción
En el tema siguiente se describe cómo agregar un mensaje para una excepción.  
  
### <a name="to-add-a-message-for-an-exception"></a>Para agregar un mensaje para una excepción  
  
1.  En el **Vista orquestación** ventana, haga clic en **mensajes** y seleccione **nuevo mensaje**.  
  
     De este modo, se crea Message_3, que puede asignar específicamente al error.  
  
2.  Haga clic en **Message_3** y seleccione **propiedades**.  
  
3.  En el **propiedades** ventana, establezca el **tipo de mensaje**.  
  
     Seleccione **clases .net**y, a continuación, seleccione **SystemString**.
