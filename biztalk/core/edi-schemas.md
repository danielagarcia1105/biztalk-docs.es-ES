# <a name="edi-schemas"></a>EDI personalizados
EDI y AS2 de Microsoft [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] usan los siguientes tipos de esquemas para procesar los mensajes entrantes y generar mensajes salientes:  
  
-   Esquemas de documentos que definen el cuerpo del mensaje.  
  
-   Esquemas de servicio de encabezado que definen el sobre del mensaje y esquemas de control de confirmación que definen el sobre y el cuerpo de las confirmaciones técnicas y funcionales.  
  
-   Esquemas XML de intercambio (por lotes) que definen el sobre de un lote conservado.  
  
-   Un esquema de propiedad que define las propiedades de contexto EDI. Para obtener más información, consulte [propiedades de contexto EDI](../core/edi-context-properties.md).
