# <a name="error---failed-write-extension-xml-file"></a>Error: no se pudo escribir el archivo XML de extensión
**Código de error**  
  
 btm1058  
  
 **Explicación**  
  
 Se encontró el error indicado al intentar escribir el XML de objeto de extensión generado por esta asignación en el archivo indicado.  
  
 **Acción del usuario**  
  
 En función del error indicado, realice la acción apropiada para corregir la causa del error. Por ejemplo, libere espacio en el disco duro.
