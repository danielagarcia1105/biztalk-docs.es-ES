# <a name="using-the-microsoft-biztalk-loadgen-2007-tool"></a>Uso de la herramienta de Microsoft BizTalk LoadGen 2007
La herramienta Microsoft BizTalk LoadGen 2007 está diseñada para aquellos programadores y profesionales de TI que deseen simular cargas en un servidor BizTalk Server. Gracias a ella, en una implementación de BizTalk es posible simular la carga que se produce en la fuerza y el rendimiento de los instrumentos. Además, los programadores también podrán utilizarla para simular la carga en transportes personalizados.  
  
> [!NOTE]
>  Descargar [LoadGen](https://www.microsoft.com/download/details.aspx?id=14925). 
  
 Esta herramienta solo debe emplearse en un entorno de prueba, no en un entorno de producción. Esta herramienta se suministra "como está", pero no incluye soporte técnico.  
  
> [!NOTE]
>  Puede usar la herramienta de generación de cargas de BizTalk Server con el adaptador de MSMQ, pero debe realizar algunos pasos adicionales. Para obtener instrucciones, consulte [uso de LoadGen 2007 con MSMQ](../core/using-loadgen-2007-with-msmq.md).  
  
> [!NOTE]
>  LoadGen no puede admitirse en los equipos de 64 bits. Puede utilizar LoadGen de forma remota en un equipo de 32 bits para generar cargas en un servidor de 64 bits.
