# <a name="adapter-samples---development"></a>Ejemplos de adaptador: Desarrollo
En esta sección se proporciona información detallada acerca de la funcionalidad que muestra cada ejemplo de adaptador, instrucciones para la generación y ejecución del ejemplo, así como los resultados que se pueden esperar.  
  
## <a name="in-this-section"></a>En esta sección  
  
- [(Ejemplo de BizTalk Server) del adaptador de archivo](../core/file-adapter-biztalk-server-sample.md). Muestra cómo escribir un. NET adaptador basado en archivo en Microsoft Visual C#.  
  
- [Adaptador de HTTP (ejemplo de BizTalk Server)](../core/http-adapter-biztalk-server-sample.md). Muestra cómo se implementan los paradigmas de comunicación de solicitud-respuesta y de petición-respuesta que se usan en [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)].  
  
- [SubmitDirect (ejemplo de BizTalk Server)](../core/submitdirect-biztalk-server-sample.md). Muestra dos maneras de enviar mensajes a BizTalk Server desde aplicaciones basadas en .NET mediante adaptadores.  
  
- [Adaptador transaccional (ejemplo de BizTalk Server)](../core/transactional-adapter-biztalk-server-sample.md). Muestra cómo crear y usar una Coordinador de transacciones distribuidas de Microsoft (MSDTC) transacción explícita contra una base de datos durante el procesamiento de un mensaje de BizTalk Server.
