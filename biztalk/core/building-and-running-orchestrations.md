# <a name="building-and-running-orchestrations"></a>Generar y ejecutar orquestaciones
Después de diseñar una orquestación, puede compilarla y ejecutarla para comprobar su integridad y exactitud.  
  
 Una vez compilada la orquestación sin errores, puede implementarla en un entorno de pruebas antes de realizar cualquier modificación necesaria y, eventualmente, implementarla para ejecutarla en un entorno de producción.
