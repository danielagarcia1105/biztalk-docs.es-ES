# <a name="single-sign-on-applications"></a>Aplicaciones de inicio de sesión único
Desde una perspectiva de programación, puede escribir dos tipos diferentes de las aplicaciones que usan el inicio de sesión único: una aplicación tradicional de inicio de sesión único que utiliza la interfaz de inicio de sesión único para interactuar con las aplicaciones remotas y un adaptador de sincronización de contraseña que usa el Interfaz de contraseña de aplicación auxiliar de sincronización (PS) para sincronizar contraseñas en toda la empresa.
