# <a name="error---test-map-failure"></a>Error: error de asignación de prueba
**Código de error**  
  
 btm1054  
  
 **Explicación**  
  
 Error en la operación Comprobar asignación para la asignación indicada.  
  
 **Acción del usuario**  
  
 Con los mensajes de error que lo acompañan, determine la causa del error de la operación Comprobar asignación, realice los cambios necesarios en la asignación o en sus esquemas de origen, de destino o en ambos.
