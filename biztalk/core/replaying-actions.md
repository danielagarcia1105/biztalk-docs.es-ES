# <a name="replaying-actions"></a>Reproducir acciones
El Depurador de orquestaciones permite reproducir, paso a paso, las acciones de una orquestación. Cuando se abre una orquestación, el cursor se sitúa en la primera acción (por ejemplo, Inicialización). Utilice las flechas Arriba y Abajo para avanzar o retroceder por la orquestación. Siga el progreso a medida que la orquestación representada va pasando de una forma a otra.
