# <a name="configuring-ports-for-an-edi-solution"></a>Configurar puertos para una solución EDI
Para recibir y enviar mensajes EDI así como confirmaciones, cree los siguientes puertos de recepción y de envío:  
  
-   Un puerto de recepción de archivos unidireccional que reciba confirmaciones e intercambios EDI o un puerto de recepción de archivos de solicitud-respuesta bidireccional que reciba los intercambios EDI y envíe confirmaciones.  
  
-   Un puerto de envío de archivos unidireccional que envíe confirmaciones o intercambios EDI.
