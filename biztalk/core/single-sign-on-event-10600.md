# <a name="single-sign-on-event-10600"></a>Inicio de sesión único: Evento 10600
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10600|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|SSO_ERROR_CS_CALLBACK_ACCESS_DENIED|  
|Texto del mensaje|Acceso denegado al servidor del almacén de configuración.%r|  
  
## <a name="explanation"></a>Explicación  
 Se envió un mensaje al servidor del almacén de configuración pero no fue aceptado. Existen diversos motivos para este evento, como protocolo incorrecto o falta de permisos de seguridad suficientes.  
  
## <a name="user-action"></a>Acción del usuario  
 Tenga en cuenta la información de este mensaje y cualquier información relevante en el registro de eventos y póngase en contacto con los servicios de soporte técnico de Microsoft.
