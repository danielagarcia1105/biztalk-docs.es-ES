# <a name="verifying-the-deployment-setup"></a>Comprobar la configuración de implementación
Antes de usar BizTalk Server para importar un archivo de enlace, compruebe lo siguiente:  
  
-   Deben existir las carpetas para las respuestas y ser idénticas en el equipo nuevo, o bien debe editar el archivo.  
  
-   Las contraseñas del sistema TIBCO Enterprise Message Service, si están presentes en la configuración, se guardan como ***** en el archivo de enlace. Para obtener más información, consulte [limitaciones de la implementación](../core/deployment-limitations1.md).
