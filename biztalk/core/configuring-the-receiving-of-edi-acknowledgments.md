# <a name="configuring-the-receiving-of-edi-acknowledgments"></a>Configurar la recepción de confirmaciones EDI
En [!INCLUDE[prague](../includes/prague-md.md)], con la introducción del acuerdo bidireccional, puede configurar la recepción de una confirmación en el mismo acuerdo que configure para enviar una confirmación. Para ello, especifique las propiedades en las diferentes pestañas de los acuerdos unidireccionales. Para obtener información sobre cómo configurar la recepción de una confirmación, vea [configurar el envío y recepción de confirmaciones EDI](../core/configuring-the-sending-and-receiving-of-edi-acknowledgments.md).
