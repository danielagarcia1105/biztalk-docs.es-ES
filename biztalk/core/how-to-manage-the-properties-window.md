# <a name="how-to-manage-the-properties-window"></a>Cómo administrar la ventana Propiedades
El Editor de BizTalk utiliza la ventana Propiedades de Microsoft [!INCLUDE[btsVStudioNoVersion](../includes/btsvstudionoversion-md.md)] para mostrar los valores de las propiedades asociadas a los nodos en la vista de árbol de esquema y proporciona una interfaz para modificar estos valores. En este tema se proporcionan instrucciones paso a paso para administrar ciertas características de la ventana Propiedades.  
  
### <a name="to-turn-property-descriptions-on-and-off"></a>Para activar y desactivar las descripciones de propiedad  
  
-   En la ventana Propiedades, haga clic en cualquier nombre de propiedad y, a continuación, haga clic en **descripciones**.  
  
    > [!NOTE]
    >  Utilice este procedimiento para activar y desactivar la presentación de propiedades.  
  
### <a name="to-arrange-properties-alphabetically-within-their-categories"></a>Para organizar las propiedades por orden alfabético dentro de sus categorías  
  
-   Haga clic en el **por categorías** botón de barra de herramientas encima de la lista de propiedades en la ventana Propiedades.  
  
### <a name="to-arrange-properties-alphabetically-without-regard-to-their-categories"></a>Para organizar las propiedades por orden alfabético sin tener en cuenta sus categorías  
  
-   Haga clic en el **alfabético** botón de barra de herramientas encima de la lista de propiedades en la ventana Propiedades.
