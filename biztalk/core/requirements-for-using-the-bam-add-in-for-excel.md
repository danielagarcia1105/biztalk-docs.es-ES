# <a name="requirements-for-using-the-bam-add-in-for-excel"></a>Requisitos para usar el complemento BAM para Excel
Los analistas de negocios usan el complemento BAM para Excel a fin de definir los eventos y los datos que desean someter a seguimiento, así como las agregaciones y agrupaciones lógicas de las vistas de datos sometidos a seguimiento a las que los usuarios podrán tener acceso.  
  
## <a name="bam-add-in-for-excel-requirements"></a>Requisitos del complemento BAM para Excel  
 Puede ejecutar el complemento BAM para Excel en cualquier equipo; no es necesario que [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] esté instalado. No obstante, para que el complemento BAM para Excel funcione, primero deberá instalar los archivos que se enumeran aquí. Consulte la documentación de Microsoft Excel para obtener información sobre dónde encontrar estos archivos.  
  
-   Excel.exe  
  
-   Vbe7.dll  
  
-   Stdole2.tlb  
  
-   Mso.dll  
  
-   Fm20.dll  
  
-   Scrrun.dll  
  
-   Mscomctl.ocx  
  
-   Msado15.dll
