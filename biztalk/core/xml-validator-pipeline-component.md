# <a name="xml-validator-pipeline-component"></a>Componente de canalización Validador XML
El componente de canalización Validador de XML puede utilizarse tanto en canalizaciones de envío como de recepción en cualquier fase, excepto para desensamblar o ensamblar. El componente Validador de XML valida el mensaje según el esquema o los esquemas especificados y si el mensaje no se ajusta a estos esquemas, el componente genera un error y el motor de mensajería coloca el mensaje en la cola de suspensión.  
  
 Para obtener información acerca de cómo configurar el componente de canalización de validador de XML, vea [cómo configurar el componente de canalización de validador XML](../core/how-to-configure-the-xml-validator-pipeline-component.md).
