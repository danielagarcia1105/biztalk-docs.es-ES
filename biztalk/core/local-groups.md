# <a name="local-groups"></a>Grupos locales
El Administrador de configuración crea las cuentas de grupo y de usuario de Windows necesarias de forma predeterminada si se instala BizTalk Server con todo el software necesario en un solo equipo. BizTalk Server admite cuentas locales de grupo y de usuario sólo en configuraciones de un solo equipo.
