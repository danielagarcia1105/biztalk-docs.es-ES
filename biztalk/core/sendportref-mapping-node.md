# <a name="sendportref-mapping-node"></a>SendPortRef (nodo Mapping)
El nodo SendPortRef del nodo Mapping de un archivo de enlace enumera los nombres de los puertos de envío a los que hace referencia el nodo principal de asignación.  
  
## <a name="nodes-in-the-sendportref-node"></a>Nodos del nodo SendPortRef  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|Nombre|Atributo|xs:string|Especifica el nombre de un puerto de envío al que hace referencia el nodo principal de asignación.|No requerido|Valor predeterminado: vacío|
