# <a name="what-is-a-real-time-aggregation"></a>¿Qué es una agregación en tiempo real?
El analista de negocios puede definir la tabla dinámica como agregación en tiempo real (ATR) o agregación programada. Una ATR ofrece vista actualizada al minuto de los datos; por ejemplo, indica la posición de un pedido de compra específico en el proceso de envío. Puede actualizar la pantalla para actualizar la vista de los datos a lo largo del día.
