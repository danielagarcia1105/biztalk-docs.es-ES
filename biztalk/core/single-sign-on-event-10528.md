# <a name="single-sign-on-event-10528"></a>Inicio de sesión único: Evento 10528
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10528|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_REENCRYPT_OK|  
|Texto del mensaje|El recifrado de la base de datos de SSO se completó correctamente.%r<br /><br /> Número de credenciales que se vuelve a cifrar: %1 %r<br /><br /> MSID actual: %2 %r<br /><br /> MSID anterior: %3|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que el recifrado de la base de datos de SSO se completó correctamente.

-   No es necesario que el usuario realice ninguna acción.
