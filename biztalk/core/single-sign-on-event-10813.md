# <a name="single-sign-on-event-10813"></a>De sesión único: Evento 10813
## <a name="details"></a>Detalles  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  Nombre del producto   |                 Inicio de sesión único (SSO) empresarial                  |
| Versión del producto | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    Identificador del evento     |                           10813                            |
|  Origen del evento   |                           ENTSSO                           |
|    Componente    |                            N/D                             |
|  Nombre simbólico  |         ENTSSO_E_WINDOWS_UPDATE_FAILED_MORE_RECENT         |
|  Texto del mensaje   |  La contraseña de Windows de la base de datos de SSO es más reciente.  |
  
## <a name="explanation"></a>Explicación  
 Se recibió una contraseña desactualizada proveniente del sistema de Windows y se ignorará.  
  
## <a name="user-action"></a>Acción del usuario  
 No es necesario que el usuario realice ninguna acción.
