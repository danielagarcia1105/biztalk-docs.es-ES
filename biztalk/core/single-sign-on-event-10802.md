# <a name="single-sign-on-event-10802"></a>Inicio de sesión único: Evento 10802
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10802|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_WRONG_SESSION|  
|Texto del mensaje|Se recibió una solicitud para la sesión incorrecta. Es posible que se la haya desconectado.|
