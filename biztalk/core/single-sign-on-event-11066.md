# <a name="single-sign-on-event-11066"></a>De sesión único: Evento 11066
## <a name="details"></a>Detalles  
  
|                 |                                                            |
|-----------------|------------------------------------------------------------|
|  Nombre del producto   |                 Inicio de sesión único (SSO) empresarial                  |
| Versión del producto | [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)] |
|    Identificador del evento     |                           11066                            |
|  Origen del evento   |                           ENTSSO                           |
|    Componente    |                            N/D                             |
|  Nombre simbólico  |            SSO_INFO_PASSWORD_SYNC_MIIS_STARTED             |
|  Texto del mensaje   |    Se inició la sincronización de contraseñas de Windows (desde MIIS).%r    |
  
## <a name="explanation"></a>Explicación  
 Este mensaje de información indica que se inició la sincronización de contraseñas de Windows (desde MIIS).  
  
## <a name="user-action"></a>Acción del usuario  
 No se requiere ninguna acción del usuario.
