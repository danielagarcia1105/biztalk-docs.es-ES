# <a name="distributionlistref-port-node"></a>DistributionListRef (nodo Puerto)
El nodo DistributionListRef del nodo Puerto de un archivo de enlace contiene información acerca de una lista de distribución a la que hace referencia un servicio.  
  
> [!NOTE]
>  A las listas de distribución se hace referencia como grupos de puertos de envío en la consola de administración de BizTalk Server.  
  
## <a name="nodes-in-the-distributionlistref-node"></a>Nodos del nodo DistributionListRef  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|Nombre|Atributo|xs:string|Especifica el nombre de una lista de distribución a la que hace referencia un servicio.|No requerido|Valor predeterminado: vacío|
