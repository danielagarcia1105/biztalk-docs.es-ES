# <a name="required-ports-for-biztalk-server"></a>Puertos requeridos para BizTalk Server
Para obtener información completa acerca de cómo proteger la implementación de BizTalk Server, vea [arquitecturas de servidor de BizTalk de ejemplo](../core/sample-biztalk-server-architectures.md).  
  
 Esta sección contiene información sobre los puertos que tiene que abrir en los servidores de seguridad para que los componentes de BizTalk Server puedan comunicarse entre sí.  
  
 Si en su entorno de producción dispone de otros servidores además de los que utiliza BizTalk Server, puede que tenga que abrir más puertos. Para obtener más información sobre los requisitos de puerto para el Windows Server System™, consulte [http://go.microsoft.com/fwlink/?LinkId=25713](http://go.microsoft.com/fwlink/?LinkId=25713).
