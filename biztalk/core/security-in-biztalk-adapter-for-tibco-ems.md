# <a name="security-in-biztalk-adapter-for-tibco-ems"></a>Seguridad del adaptador de BizTalk para TIBCO EMS

## <a name="overview"></a>Información general
Microsoft BizTalk Adapter para TIBCO Enterprise Message Service admite el inicio de sesión único (SSO). Una aplicación afiliada creada por las herramientas de inicio de sesión único empresarial representa un sistema servidor como TIBCO Enterprise Message Service.  

En esta sección se proporcionan directrices sobre el modo de implementar el adaptador de BizTalk para TIBCO Enterprise Message Service en un entorno seguro.  
  
> [!CAUTION]
>  Se recomienda encarecidamente que restringen el uso del adaptador de BizTalk para Enterprise Message Service a usuarios autorizados únicamente, como los archivos de cliente se conectan directamente a las aplicaciones de línea de negocio.
