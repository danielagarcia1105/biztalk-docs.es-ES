# <a name="single-sign-on-event-10659"></a>Inicio de sesión único: Evento 10659
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10659|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_PASSWORD_SYNC_WINDOWS_STARTING|  
|Texto del mensaje|Se está iniciando la sincronización de contraseñas de Windows (desde PCNS).%r|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que se está iniciando la sincronización de contraseñas de Windows.  
  
## <a name="user-action"></a>Acción del usuario  
  
-   No es necesario que el usuario realice ninguna acción.
