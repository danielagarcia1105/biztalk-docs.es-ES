# <a name="single-sign-on-event-10810"></a>Inicio de sesión único: Evento 10810
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10810|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_HISSO_INVALID_CREDENTIALS|  
|Texto del mensaje|Las credenciales especificadas no son válidas o no coinciden con las de la base de datos de SSO.|  
  
## <a name="explanation"></a>Explicación  
 Las credenciales especificadas no son válidas o no coinciden con las de la base de datos de SSO.  
  
## <a name="user-action"></a>Acción del usuario  
 Configure la información en la base de datos de SSO para que coincida con la información del sistema externo.
