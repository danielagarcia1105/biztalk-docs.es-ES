# <a name="trackedpropertynames-schema-node"></a>TrackedPropertyNames (nodo Esquema)
El nodo TrackedPropertyNames de un archivo de enlace contiene elementos que especifican a qué propiedades del esquema primario hay que hacer un seguimiento.  
  
## <a name="nodes-in-the-trackedpropertyname-node"></a>Nodos del nodo TrackedPropertyName  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|string|Elemento|xs:string|Especifica un nombre de propiedad al que se realizará un seguimiento.|No requerido|Valor predeterminado: vacío|
