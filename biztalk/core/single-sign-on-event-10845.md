# <a name="single-sign-on-event-10845"></a>De sesión único: Evento 10845
## <a name="details"></a>Detalles  
  
|                 |                                                                                                                       |
|-----------------|-----------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                               Inicio de sesión único (SSO) empresarial                                               |
| Versión del producto |                              [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                               |
|    Identificador del evento     |                                                         10845                                                         |
|  Origen del evento   |                                                        ENTSSO                                                         |
|    Componente    |                                                          N/D                                                          |
|  Nombre simbólico  |                                                  ENTSSO_E_DB_ACCESS2                                                  |
|  Texto del mensaje   | Error al intentar obtener acceso a la base de datos de SSO. Para obtener información detallada, consulte el registro de eventos (en el equipo "%1"). |
  
## <a name="explanation"></a>Explicación  
 Error al intentar obtener acceso a la base de datos de SSO.  
  
## <a name="user-action"></a>Acción del usuario  
 Consulte el registro de eventos en el equipo especificado para obtener información adicional del servidor SQL Server.
