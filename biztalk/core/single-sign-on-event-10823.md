# <a name="single-sign-on-event-10823"></a>De sesión único: Evento 10823
## <a name="details"></a>Detalles  
  
|                 |                                                                                                              |
|-----------------|--------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                          Inicio de sesión único (SSO) empresarial                                           |
| Versión del producto |                          [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                          |
|    Identificador del evento     |                                                    10823                                                     |
|  Origen del evento   |                                                    ENTSSO                                                    |
|    Componente    |                                                     N/D                                                      |
|  Nombre simbólico  |                                      ENTSSO_E_PSADMIN_INVALID_APP_TYPE                                       |
|  Texto del mensaje   | Tipo de aplicación no válido para la aplicación. Los tipos de aplicación válidos son "Individual", "Grupo" o "Grupo host". |
  
## <a name="explanation"></a>Explicación  
 Tipo de aplicación no válido para la aplicación. Los tipos de aplicación válidos son "Individual", "Grupo" o "Grupo host".  
  
## <a name="user-action"></a>Acción del usuario  
 Use uno de los tipos de aplicación válidos.
