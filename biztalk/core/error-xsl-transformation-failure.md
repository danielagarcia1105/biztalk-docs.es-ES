# <a name="error---xsl-transformation-failure"></a>Error: error de transformación XSL
**Código de error**  
  
 btm1050  
  
 **Explicación**  
  
 El error XSLT indicado se ha producido durante el proceso de transformación.  
  
 **Acción del usuario**  
  
 En función del error XSLT indicado, modifique la asignación y los esquemas de origen y destino para corregir la causa del error.
