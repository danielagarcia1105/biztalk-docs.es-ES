# <a name="how-to-add-web-messages"></a>Cómo agregar mensajes web
Cree variables de mensajes Web en la ventana Vista orquestación. Debe establecer el tipo de mensaje para el mensaje Web como un tipo de mensaje Web.  
  
### <a name="to-add-a-web-message"></a>Para agregar un mensaje Web  
  
1.  Con una orquestación abierta, en el **vista** menú, haga clic en **otras ventanas**y, a continuación, haga clic en **Vista orquestación**.  
  
2.  Haga clic en el **mensaje** nodo y, a continuación, haga clic en **nuevo mensaje**.  
  
3.  Mensaje_*x* aparece bajo la **mensaje** nodo.  
  
4.  En el **propiedades** ventana, desde el **tipo de mensaje** lista desplegable, seleccione un tipo de mensaje Web.  
  
5.  Desplácese por el cuadro desplegable hasta el tipo de mensaje Web que desea utilizar y, a continuación, haga clic en él para seleccionarlo.  
  
     La siguiente ilustración muestra la **Vista orquestación** ventana con el elemento mensaje_1 seleccionado y la ventana Propiedades, seleccione un sitio Web tipo de mensaje para mensaje_1.
