# <a name="how-to-identify-bottlenecks-in-the-biztalkdtadb-database"></a>Cómo identificar cuellos de botella en la base de datos BizTalkDTADb
Para identificar cuellos de botella en la base de datos BizTalkDTADb, realice los siguientes pasos:  
  
1.  Asegúrese de que servicio del Agente SQL está en ejecución.  
  
2.  Asegúrese de que el trabajo de archivo y purga esté en ejecución y que se esté realizando correctamente.  
  
3.  Compruebe que hay suficiente espacio disponible en disco para los archivos DTADb y para el crecimiento de la base de datos.
