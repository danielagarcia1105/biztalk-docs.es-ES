# <a name="messaging-payload-view"></a>Carga de mensajería (vista)
La vista Carga de mensajería muestra el esquema del mensaje XML asociado a la acción seleccionada (por ejemplo, Mensaje enviado o recibido). La vista está disponible desde el menú contextual para algunas de las formas de la vista Programación de orquestación.  
  
## <a name="working-with-the-messaging-payload-view"></a>Trabajar con la vista Carga de mensajería  
 Al hacer clic en el botón Seleccionar origen de eventos y en el elemento de menú Seleccionar esquema de carga de mensajería, se selecciona su programación de orquestación. Una vez seleccionado el esquema, puede arrastrar los elementos de este esquema a las carpetas de elementos de datos de la actividad. De esta manera, se indica que desea extraer en esta acción esos datos de de la expresión XPath específica del mensaje.  
  
 Puede abrir las vistas de carga de mensajería desde la programación de orquestación al hacer clic con el botón secundario en una forma que contenga una carga de mensaje. De esta manera, se abrirá el menú contextual desde el que puede hacer clic en el elemento de menú Carga de mensajería para recuperar el esquema de propiedad de mensajería.  
  
 Para obtener más información acerca de las expresiones de XPath, vea [utilizar XPaths en las asignaciones de mensajes](../core/using-xpaths-in-message-assignments.md).
