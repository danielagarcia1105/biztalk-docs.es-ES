# <a name="what-is-a-scheduled-aggregation"></a>¿Qué es una agregación programada?
Una agregación programada representa una instantánea de la empresa a una hora determinada, por ejemplo un resumen de los envíos de la mañana. Pregunte al administrador de base de datos cuándo se procesan las agregaciones y podrá consultar los datos históricos.
