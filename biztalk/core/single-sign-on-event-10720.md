# <a name="single-sign-on-event-10720"></a>De sesión único: Evento 10720

|                 |                                                                                                                                                                                                                                |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                                                                   Inicio de sesión único (SSO) empresarial                                                                                                    |
| Versión del producto |                                                                                   [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                                                                   |
|    Identificador del evento     |                                                                                                             10720                                                                                                              |
|  Origen del evento   |                                                                                                             ENTSSO                                                                                                             |
|    Componente    |                                                                                                              N\D                                                                                                               |

 Este evento de información indica que el cambio de contraseña externa se almacenó correctamente en el archivo de reproducción.
