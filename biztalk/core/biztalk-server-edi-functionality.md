# <a name="biztalk-server-edi-functionality"></a>Funcionalidad EDI de BizTalk
[!INCLUDE[prague](../includes/prague-md.md)] procesa mensajes EDI usando una combinación de características principales de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] y características de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] específicas de EDI. Esto permite que [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] lleve a cabo el procesamiento que es exclusivo de la mensajería EDI, al mismo tiempo que aprovecha su funcionalidad de mensajería principal.  
  
 En esta sección se describe cómo funciona la mensajería EDI básica y cómo [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] la implementa. También se describe cómo se puede aprovechar una definición de acuerdo de socio comercial en [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] para la mensajería EDI, el procesamiento EDI en el lado de recepción y el procesamiento EDI en el lado de envío.  
  
 Para obtener una descripción de cómo se admite el procesamiento de EDI en [!INCLUDE[prague](../includes/prague-md.md)] y otras versiones de [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]y para obtener una descripción de la compatibilidad con los estándares EDI y compatibilidad con esquema de documento EDI, vea [EDI admite problemas](../core/edi-support-issues.md).
