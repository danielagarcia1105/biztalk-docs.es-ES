# <a name="single-sign-on-event-10588"></a>Inicio de sesión único: Evento 10588
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10588|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|SSO_WARN_SQL_NOT_FOUND|  
|Texto del mensaje|No se encontró el servidor SQL Server especificado. Compruebe si el nombre del servidor SQL Server es correcto y si SQL Server está en ejecución en ese equipo.%r<br /><br /> Nombre de SQL Server: %1|  
  
## <a name="explanation"></a>Explicación  
 No se encontró el servidor SQL Server especificado en el mensaje.  
  
## <a name="user-action"></a>Acción del usuario  
 Compruebe si el nombre del servidor SQL Server es correcto y si SQL Server está en ejecución en ese equipo.
