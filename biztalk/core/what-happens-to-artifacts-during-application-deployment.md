# <a name="what-happens-to-artifacts-during-application-deployment"></a>¿Qué le sucede a los artefactos durante la implementación de aplicaciones?
Los temas de esta sección describen los que le sucede a los artefactos asociados a una aplicación cuando se agregan o se quitan artefactos, se exportan e importan aplicaciones y artefactos, y se instalan y desinstalan aplicaciones. Se tratan asuntos como los siguientes:  
  
-   Cómo se agregan y se quitan los artefactos de las bases de datos de BizTalk Server  
  
-   Cómo se copian y se eliminan los artefactos del sistema de archivos local  
  
-   Cómo se agregan o se quitan los artefactos (ensamblados .NET y componentes COM) del Registro de Windows  
  
-   Cómo se instalan o desinstalan los artefactos (ensamblados de BizTalk y ensamblados .NET) en la caché de ensamblados global  
  
-   Cómo se agregan certificados al almacén de certificados local
