# <a name="lesson-2-define-the-business-process"></a>Lección 2: Definir el proceso empresarial
En esta lección, creará el segundo proyecto de la solución Integration(EAI) de aplicación de empresa. Este proyecto contiene una orquestación.  
  
 En el escenario EAI, la aplicación de almacén envía un mensaje de reposición de inventario para [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] para su procesamiento. [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)]utiliza la orquestación que ha creado para automatizar el proceso empresarial. La orquestación proporciona el flujo de trabajo, las acciones y las expresiones para mover los mensajes por el sistema y procesar su contenido.  
  
 Por último, compile el proyecto antes de iniciar [lección 3: implementar la solución](../core/lesson-3-deploy-the-solution.md).  
  
 Para obtener más información, consulte [crear orquestaciones utilizando Diseñador de orquestaciones](../core/creating-orchestrations-using-orchestration-designer.md).
