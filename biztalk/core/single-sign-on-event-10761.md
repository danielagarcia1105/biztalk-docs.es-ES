# <a name="single-sign-on-event-10761"></a>Inicio de sesión único: Evento 10761
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10761|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_EMPTY_EXISTING_VALUE|  
|Texto del mensaje|El contenedor de propiedades incluye un valor VT_NULL para indicar que debe conservarse el valor de propiedad existente, pero el valor existente está vacío.|

 Código de error interno.
