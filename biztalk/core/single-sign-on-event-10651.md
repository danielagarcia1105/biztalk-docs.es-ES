# <a name="single-sign-on-event-10651"></a>Inicio de sesión único: Evento 10651
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10651|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_PASSWORD_SYNC_STARTED|  
|Texto del mensaje|Los servicios de sincronización de contraseñas se inicializaron correctamente.%r|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que se inicializaron los servicios de sincronización de contraseñas.

-   No es necesario que el usuario realice ninguna acción.
