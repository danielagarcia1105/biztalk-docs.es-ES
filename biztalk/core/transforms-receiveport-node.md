# <a name="transforms-receiveport-node"></a>Transformaciones (nodo puertoRecepción)
El nodo Transformaciones del nodo puertoRecepción de un archivo de enlace contiene la colección de transformaciones de entrada de un puerto de recepción unidireccional que se exporta con el archivo de enlace.  
  
## <a name="nodes-in-the-transforms-node"></a>Nodos del nodo Transformaciones  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Transformación (nodo transformaciones)](../core/transform-transforms-node.md)|Grabar|Transformación (ComplexType)|Especifica un asignador de BizTalk Server, o una transformación, es decir, un elemento que representa la asignación entre un esquema de origen y uno de destino.|No requerido|Valor predeterminado: ninguno|
