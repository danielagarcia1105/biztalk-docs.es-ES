# <a name="troubleshooting-jd-edwards-enterpriseone"></a>Solución de problemas de JD Edwards EnterpriseOne

## <a name="overview"></a>Información general
En esta sección se describen los problemas y mensajes de error comunes que se pueden encontrar en el adaptador de BizTalk para JD Edwards EnterpriseOne y proporcione posibles correcciones. Además, se trata el uso de Seguimiento de eventos para Windows.  
  
 Puede usar las siguientes herramientas de depuración y seguimiento para solucionar problemas del adaptador:  
  
-   Depuración nativa de BizTalk Server (por ejemplo, el Tipo de seguimiento en su puerto o el Depurador de orquestaciones).  
  
-   Mensajes de error enviados al registro de eventos.  
  
-   Captura de mensajes del transmisor, receptor y gestión mediante BTAJDEEnterpriseOneTrace.cmd y una herramienta que convierte archivos .etl, como tracerpt.exe o tracedmp.exe.
