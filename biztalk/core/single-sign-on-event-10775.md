# <a name="single-sign-on-event-10775"></a>De sesión único: Evento 10775
## <a name="details"></a>Detalles  
  
|                 |                                                                                                                          |
|-----------------|--------------------------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                                Inicio de sesión único (SSO) empresarial                                                 |
| Versión del producto |                                [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                                |
|    Identificador del evento     |                                                          10775                                                           |
|  Origen del evento   |                                                          ENTSSO                                                          |
|    Componente    |                                                           N/D                                                            |
|  Nombre simbólico  |                                                 ENTSSO_E_APP_ADMIN_SAME                                                  |
|  Texto del mensaje   | Se especificó que la cuenta de administradores de aplicación de esta aplicación sea la cuenta de administradores afiliados de SSO. |
  
## <a name="explanation"></a>Explicación  
 La aplicación se marcó para usar la cuenta de administradores afiliados de SSO como la cuenta de administradores de aplicación. Como consecuencia, no se puede establecer ni cambiar la cuenta de administradores de aplicación.  
  
## <a name="user-action"></a>Acción del usuario  
 Solicite ayuda al administrador del sistema.
