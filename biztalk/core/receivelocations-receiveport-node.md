# <a name="receivelocations-receiveport-node"></a>ReceiveLocations (nodo puertoRecepción)
El nodo ReceiveLocations del nodo puertoRecepción de un archivo de enlace es el nodo principal de todos los nodos ReceiveLocation que contiene información específica acerca de las ubicaciones de recepción que se exportan con el archivo de enlace.  
  
## <a name="nodes-in-the-receivelocations-node"></a>Nodos del nodo ReceiveLocations  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[ReceiveLocation](../core/receivelocation-receivelocations-node.md)|Grabar|ReceiveLocation (ComplexType)|Especifica información acerca de una ubicación de recepción que se exporta con el archivo de enlace.|No requerido|Valor predeterminado: ninguno|
