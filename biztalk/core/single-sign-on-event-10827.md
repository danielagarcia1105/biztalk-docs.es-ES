# <a name="single-sign-on-event-10827"></a>De sesión único: Evento 10827
## <a name="details"></a>Detalles  
  
|                 |                                                                                                      |
|-----------------|------------------------------------------------------------------------------------------------------|
|  Nombre del producto   |                                      Inicio de sesión único (SSO) empresarial                                       |
| Versión del producto |                      [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]                      |
|    Identificador del evento     |                                                10827                                                 |
|  Origen del evento   |                                                ENTSSO                                                |
|    Componente    |                                                 N/D                                                  |
|  Nombre simbólico  |                             ENTSSO_E_PSADMIN_INVALID_GROUP_ADAPTER_TYPE                              |
|  Texto del mensaje   | Tipo de aplicación no válido para adaptador de grupo. El tipo de aplicación válido es "Adaptador de grupo de sincronización de contraseñas". |
  
## <a name="explanation"></a>Explicación  
 Tipo de aplicación no válido para adaptador de grupo. El tipo de aplicación válido es "Adaptador de grupo de sincronización de contraseñas".  
  
## <a name="user-action"></a>Acción del usuario  
 Use uno de los tipos de aplicación válidos.
