# <a name="using-the-bam-wcf-and-wf-interceptors"></a>Utilizar los interceptores WCF y WF de BAM
Los interceptores de BAM amplían la funcionalidad del interceptor de BAM para [!INCLUDE[btsBizTalkServerNoVersion](../includes/btsbiztalkservernoversion-md.md)] en Windows Workflow Foundation (WF), Windows Communication Framework (WCF) y otros entornos en tiempo de ejecución. Mediante el uso de un interceptor BAM, puede llevar a cabo un seguimiento de sus procesos empresariales sin volver a compilar su solución WF o WCF: la integración se realiza a través de un archivo de configuración mediante XML y una serie de elementos que asignan eventos de aplicación a actividades de BAM y definen los datos, el Id. de correlación, el token de continuación y otros artefactos requeridos y opcionales.
