# <a name="single-sign-on-event-10821"></a>De sesión único: Evento 10821
## <a name="details"></a>Detalles  
  
|                 |                                                                                    |
|-----------------|------------------------------------------------------------------------------------|
|  Nombre del producto   |                             Inicio de sesión único (SSO) empresarial                              |
| Versión del producto |             [!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]             |
|    Identificador del evento     |                                       10821                                        |
|  Origen del evento   |                                       ENTSSO                                       |
|    Componente    |                                        N/D                                         |
|  Nombre simbólico  |                     ENTSSO_E_ADAPTER_ASSIGNED_TO_GROUP_ADAPTER                     |
|  Texto del mensaje   | No se puede eliminar el adaptador porque actualmente está asignado a un adaptador de grupo. |
  
## <a name="explanation"></a>Explicación  
 No se puede eliminar el adaptador porque actualmente está asignado a un adaptador de grupo.  
  
## <a name="user-action"></a>Acción del usuario  
 Quite la asignación del adaptador y, a continuación, elimínelo.
