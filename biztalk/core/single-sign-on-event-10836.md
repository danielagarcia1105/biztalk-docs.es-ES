# <a name="single-sign-on-event-10836"></a>Inicio de sesión único: Evento 10836
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10836|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_FILE_PASSWORD_TOO_SHORT|  
|Texto del mensaje|La contraseña especificada es demasiado corta; debe tener siete caracteres como mínimo.|  
  
## <a name="explanation"></a>Explicación  
 La contraseña del archivo de copia de seguridad debe tener siete caracteres como mínimo.  
  
## <a name="user-action"></a>Acción del usuario  
 Especifique una contraseña que tenga siete caracteres como mínimo.
