# <a name="programming-with-enterprise-single-sign-on"></a>Programar con Enterprise Single Sign-On
El inicio de sesión único (SSO) empresarial es una tecnología implementada en el servicio de inicio de sesión único empresarial (ENTSSO), que permite a los usuarios obtener acceso a múltiples servicios remotos sin tener que recorrer una serie de pantallas de inicio de sesión adicionales.
