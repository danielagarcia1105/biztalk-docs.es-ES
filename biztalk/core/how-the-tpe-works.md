# <a name="how-the-tpe-works"></a>Cómo funciona el TPE
Un analista de negocios puede especificar la vista de los datos que desea ver para, por ejemplo, solucionar los problemas de un pedido de cliente determinado o analizar las tendencias de eventos empresariales específicos y los hitos del proceso de envío. Para extraer los datos que necesite el analista de negocios, el programador usa TPE para llevar a cabo la asignación entre los datos de eventos empresariales específicos y la orquestación real. Una vez que se ha creado, guardado, probado e implementado el perfil de seguimiento, el analista de negocios puede recuperar los datos recopilados para analizar un aspecto concreto de un proceso empresarial.
