# <a name="how-to-invoke-the-orchestration-debugger-and-the-message-flow-views"></a>Invocación del depurador de orquestación y vistas de flujo de mensajes
En la página Concentrador de grupo de la Consola de administración de BizTalk Server, el panel Resultados de la consulta muestra la salida de una consulta. En el menú contextual puede ver la vista Depurador de orquestaciones o Flujo de mensajes.  
  
## <a name="prerequisites"></a>Requisitos previos  
 Para llevar a cabo este procedimiento, debe haber iniciado la sesión como miembro del grupo de administradores de BizTalk Server.  
  
### <a name="to-display-the-orchestration-debugger-or-message-flow-views"></a>Procedimiento para ver las vistas Depurador de orquestaciones o Flujo de mensajes  
  
1.  Para mostrar la vista de flujo de mensaje, haga clic en una celda asociada con una instancia de orquestación en el **resultados de la consulta** lista y seleccione **flujo de mensajes**.  
  
2.  Para mostrar el depurador de orquestaciones, haga clic en una celda asociada con una instancia de orquestación en el **resultados de la consulta** lista y seleccione **depurador de orquestaciones**.
