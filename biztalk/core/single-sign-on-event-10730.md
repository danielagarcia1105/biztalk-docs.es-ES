# <a name="single-sign-on-event-10730"></a>Inicio de sesión único: Evento 10730
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10730|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_PROGRESS_RECORD_READ|  
|Texto del mensaje|El número de registro se leyó correctamente desde el archivo de progreso.%r<br /><br /> Adaptador: %1 %r<br /><br /> Nombre de archivo: %2 %r<br /><br /> Número de registro: %3|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que el número de registro se leyó correctamente desde el archivo de progreso.
