# <a name="configuring-the-wcf-adapter-to-intercept-bam-data"></a>Configurar el adaptador de WCF para interceptar datos de BAM
Esta sección contiene información acerca de los pasos que se deben seguir para configurar el adaptador de WCF de BizTalk para que funcione con el interceptor de WCF de BAM.  
  
 Los pasos básicos son los siguientes:  
  
-   Agregar el comportamiento de BAM al archivo machine.config.  
  
-   Crear un adaptador de WCF de BizTalk.  
  
-   Configurar la intercepción de WCF de BAM.
