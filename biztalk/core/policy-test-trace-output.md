# <a name="policy-test-trace-output"></a>Resultado de seguimiento de pruebas de directivas
Cuando prueba su directiva, el responsable de las pruebas muestra el resultado en la ventana de información de la versión. Esta sección describe las actividades que se incluyen en el seguimiento.  
  
> [!NOTE]
>  Si se impone una clase derivada pero las reglas se escriben directamente con respecto a los miembros de la clase base, se impone una instancia de la clase base y las condiciones se evalúan con respecto a esa instancia.
