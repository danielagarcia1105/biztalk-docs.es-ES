# <a name="btadinstallmode"></a>BTAD_InstallMode
BTAD_InstallMode describe el modo de instalación de la implementación de las aplicaciones de BizTalk.  
  
## <a name="remarks"></a>Comentarios  
 En la siguiente tabla se describen los posibles valores para BTAD_InstallMode.  
  
|Value|Description|  
|-----------|-----------------|  
|Install|Crea o actualiza BizTalkHostInstance (el equipo local)|  
|Importar|Crea o actualiza ConfigurationDb (base de datos de administración de BizTalk para el grupo)|  
|Desinstalar|Elimina de BizTalkHostInstance (equipo local)|
