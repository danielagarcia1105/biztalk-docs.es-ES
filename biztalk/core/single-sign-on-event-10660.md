# <a name="single-sign-on-event-10660"></a>Inicio de sesión único: Evento 10660
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10660|  
|Origen del evento|ENTSSO|  
|Componente|N\D|  
|Nombre simbólico|SSO_INFO_PASSWORD_SYNC_WINDOWS_STARTED|  
|Texto del mensaje|Se inició la sincronización de contraseñas de Windows (desde PCNS).%r|  
  
## <a name="explanation"></a>Explicación  
 Este evento de información indica que se inició la sincronización de contraseñas para adaptadores externos.  
  
## <a name="user-action"></a>Acción del usuario  
  
-   No es necesario que el usuario realice ninguna acción.
