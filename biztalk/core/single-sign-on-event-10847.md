# <a name="single-sign-on-event-10847"></a>Inicio de sesión único: Evento 10847
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10847|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_MAPPINGS_ONLY_FOR_DOMAIN_USER_ACCOUNTS|  
|Texto del mensaje|En el caso de las aplicaciones individuales, las asignaciones sólo pueden crearse para usuarios de dominio de Windows (no para cuentas locales o de grupo).|  
  
## <a name="explanation"></a>Explicación  
 En el caso de las aplicaciones individuales, las asignaciones sólo pueden crearse para usuarios de dominio de Windows (no para cuentas locales o de grupo).  
  
## <a name="user-action"></a>Acción del usuario  
 Para obtener más información sobre las aplicaciones individuales, consulte [aplicaciones afiliadas de SSO](../core/sso-affiliate-applications.md).
