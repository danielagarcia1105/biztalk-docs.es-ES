# <a name="single-sign-on-event-10833"></a>Inicio de sesión único: Evento 10833
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|Inicio de sesión único (SSO) empresarial|  
|Versión del producto|[!INCLUDE[btsSSOVersion](../includes/btsssoversion-md.md)]|  
|Identificador del evento|10833|  
|Origen del evento|ENTSSO|  
|Componente|N/D|  
|Nombre simbólico|ENTSSO_E_SSO_DATABASE_NOT_CONFIGURED|  
|Texto del mensaje|La base de datos de SSO especificada no está configurada.|  
  
## <a name="explanation"></a>Explicación  
 El sistema no puede encontrar la base de datos de SSO especificada.  
  
## <a name="user-action"></a>Acción del usuario  
 Confirme que está especificando los nombres correctos para la base de datos de SSO y el servidor de SQL Server.
