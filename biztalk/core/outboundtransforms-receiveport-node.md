# <a name="outboundtransforms-receiveport-node"></a>OutboundTransforms (nodo puertoRecepción)
El nodo OutboundTransforms del nodo puertoRecepción de un archivo de enlace contiene la colección de transformaciones de salida de un puerto de recepción bidireccional que se exporta con el archivo de enlace.  
  
## <a name="nodes-in-the-outboundtransforms-node"></a>Nodos del nodo OutboundTransforms  
 En la siguiente tabla se enumeran las propiedades que se pueden definir para este nodo de archivo de enlace:  
  
|**Nombre**|**Tipo de nodo**|**Tipo de datos**|**Description**|**Restricciones**|**Comentarios**|  
|--------------|-------------------|-------------------|---------------------|----------------------|------------------|  
|[Transformación (nodo OutboundTransforms)](../core/transform-outboundtransforms-node.md)|Grabar|Transformación (ComplexType)|Especifica un asignador de BizTalk Server, o una transformación, es decir, un elemento que representa la asignación entre un esquema de origen y uno de destino.|No requerido|Valor predeterminado: ninguno|
