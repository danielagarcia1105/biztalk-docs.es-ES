# <a name="error---node-not-valid-for-distinguished-field-promotion"></a>Error: nodo no válido para promoción de campo distintivo
**Código de error**  
  
 BEC2014  
  
 **Explicación**  
  
 No se puede promocionar el nodo indicado como Campo distinguido porque no puede alojar el contenido de texto necesario. Solo **elemento de campo** nodos, **atributo de campo** nodos, o **registro** con un contenido simple o mixto puede promocionarse como campos distintivos.  
  
 **Acción del usuario**  
  
 Use la **campos distintivos** pestaña de la **promocionar propiedades** cuadro de diálogo para quitar la promoción de propiedad no permitida.
