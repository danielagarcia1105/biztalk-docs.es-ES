# <a name="create-and-delete-adapter-handlers"></a>Crear y eliminar controladores de adaptador

## <a name="overview"></a>Información general
En esta sección se describe cómo crear y eliminar controladores de adaptadores en la consola de administración de BizTalk Server. Los procedimientos que se proporcionan en esta sección son genéricos para varios adaptadores nativos. Esta sección no contiene procedimientos específicos de un único adaptador. Para obtener más información, consulte **crear un FTP de recepción controlador mediante WMI** [!INCLUDE[ui-guidance-developers-reference](../includes/ui-guidance-developers-reference.md)].
