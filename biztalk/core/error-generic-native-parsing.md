# <a name="error---generic-native-parsing"></a>Error: análisis nativo genérico
**Código de error**  
  
 btm1042  
  
 **Explicación**  
  
 No se pudo analizar el archivo de mensaje de instancia de entrada nativo especificado para la operación Comprobar asignación y se produjo un error grave de análisis genérico.  
  
 **Acción del usuario**  
  
 Según resulte necesario, corrija el esquema de origen asociado a la asignación o el mensaje de instancia de entrada nativo del archivo especificado, o ambos. Considere trabajar en el Editor de BizTalk para aislar el problema. El **Validar esquema** y **generar instancia** comandos, que aparecen cuando hace clic en un esquema en el Explorador de soluciones, son útiles para encontrar errores de esquema.
