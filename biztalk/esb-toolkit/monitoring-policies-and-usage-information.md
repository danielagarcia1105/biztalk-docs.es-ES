# <a name="monitoring-policies-and-usage-information"></a>Información de uso y las directivas de supervisión
El Administrador de servicios de SOA proporciona un mecanismo que permite crear y editar las directivas de supervisión. Por ejemplo, la figura 1 muestra la pantalla para aplicar una plantilla de directiva y activar la supervisión de una aplicación.  
  
 ![Características de supervisión](../esb-toolkit/media/ch9-monitoringfeatures.jpg "Ch9-MonitoringFeatures")  
  
 **Figura 1**  
  
 **El Administrador de servicios de SOA que muestra las características de supervisión disponibles en la página de área de trabajo**  
  
 El Administrador de servicios de SOA también proporciona información de uso de aplicaciones, servicios y puertos de envío y ubicaciones de recepción individuo. Esta información incluye gráficos que muestran el uso relativo de cada operación y el uso con el tiempo, el uso del servicio y la información en tiempo de respuesta. Figura 2 se muestran algunos ejemplos.  
  
 ![Gráficos de supervisión](../esb-toolkit/media/ch9-monitoringcharts.jpg "Ch9-MonitoringCharts")  
  
 **Figura 2**  
  
 **Una selección de los gráficos que puede generar el Administrador de servicios de SOA para ayudar en la supervisión de rendimiento y el comportamiento de servidor y la aplicación**  
  
 Además, SOA Service Manager incluye características que le permiten ver los detalles de los mensajes individuales, incluido el tamaño, el tiempo de respuesta, nombre de la operación y los mensajes SOAP asociados, tal como se muestra en la figura 3.  
  
 ![Detalles del mensaje de SOAP](../esb-toolkit/media/ch9-soapmessagedetails.jpg "Ch9-SoapMessageDetails")  
  
 **Figura 3**  
  
 **El Administrador de servicios de SOA muestra los detalles de un mensaje individual**
