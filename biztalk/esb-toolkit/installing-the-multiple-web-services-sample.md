# <a name="installing-the-multiple-web-services-sample"></a>Instalar el ejemplo de servicios Web varios
Esta sección describe el proceso de instalación del ejemplo de varios servicios Web. Debe instalar los ejemplos en [instalar y ejecutar el ejemplo de itinerario en rampa](../esb-toolkit/installing-and-running-the-itinerary-on-ramp-sample.md) y [instalar y ejecutar el ejemplo de resolución dinámica](../esb-toolkit/installing-and-running-the-dynamic-resolution-sample.md) antes de instalar y utilizar este ejemplo.  
  
 **Para instalar el ejemplo de varios servicios Web**  
  
1.  En el Explorador de Windows, abra la carpeta \Source\Samples\MultipleWebServices\Install\Scripts.  
  
2.  Haga doble clic en el archivo de script denominado Setup_bin.cmd.
