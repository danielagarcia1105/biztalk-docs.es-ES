# <a name="installing-and-running-the-itinerary-on-ramp-sample"></a>Instalar y ejecutar el ejemplo en rampa itinerario
El ejemplo de itinerario en rampa muestra cómo puede configurar las propiedades de itinerarios de ESB y ejecución unidireccional y los escenarios de mensajería de solicitud-respuesta, como las siguientes:  
  
-   Ejecutar un itinerario que utiliza servicios de mensajería  
  
-   Ejecutar un itinerario que usa orquestaciones  
  
-   Ejecutar una combinación de itinerarios pasos que utilizan servicios de mensajería y orquestaciones  
  
 El ejemplo de itinerario en rampa utiliza una aplicación de cliente de prueba de Windows Forms para ejecutar los métodos del servicio Web de itinerario y mostrar los mensajes devueltos. Puede abrir el proyecto de cliente de prueba para ejecutarlo y examinar el código para ver cómo puede usar el servicio Web de itinerario en su propia arquitectura orientada a servicios (SOA) y aplicaciones de ESB.
