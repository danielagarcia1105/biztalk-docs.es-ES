# <a name="installing-and-running-the-itinerary-on-ramp-sample"></a>Instalar y ejecutar el ejemplo de rampa de itinerario
El ejemplo de rampa de itinerario muestra cómo puede configurar las propiedades itinerarios de ESB y ejecución unidireccional y escenarios de mensajería de solicitud y respuesta, como las siguientes:  

- Ejecutar un itinerario que utiliza servicios de mensajería  

- Ejecutar un itinerario que usa las orquestaciones  

- Ejecutar una combinación de itinerarios pasos que usan servicios de mensajería y orquestaciones  

  El ejemplo de rampa de itinerario usa una aplicación de cliente de prueba de Windows Forms para ejecutar los métodos del servicio Web de itinerario y mostrar los mensajes devueltos. Puede abrir el proyecto de cliente de prueba para ejecutarlo y examinar el código para ver cómo puede usar el servicio Web de itinerario en su propia arquitectura orientada a servicios (SOA) y aplicaciones de ESB.
