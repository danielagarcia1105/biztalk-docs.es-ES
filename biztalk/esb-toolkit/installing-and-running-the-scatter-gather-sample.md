# <a name="installing-and-running-the-scatter-gather-sample"></a>Instalar y ejecutar el ejemplo de dispersión y recopilación
El ejemplo de dispersión y recopilación muestra cómo se puede desarrollar una orquestación como un paso personalizado itinerario para aplicar el modelo de diseño de dispersión y recopilación resolver y enviando solicitudes asincrónicas a varios servicios y agregar sus respuestas asincrónicas en un mensaje de respuesta única para reenviar a otra ubicación de punto de conexión. El ejemplo usa la aplicación de cliente de prueba proporcionado con el ejemplo de itinerario en rampa y artefactos incluidos con el ejemplo de resolución dinámica.
