# <a name="using-the-helper-classes"></a>Uso de las clases auxiliares
[!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]incluye un conjunto de clases auxiliares que utilizan los otros procesos, servicios y componentes en el Kit de herramientas. Dado que estas clases auxiliares también son públicas, se puede usar en su propio código según sea necesario.  
  
 En algunos casos, es necesario utilizarlos para proporcionar los parámetros del tipo adecuado para los métodos de otras clases de ESB. Por ejemplo, los métodos de servicio Web de resolución devuelven una colección genérica de instancias de la **resolución** clase.  
  
 Las secciones siguientes describen la aplicación auxiliar de clases se incluyen en [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]:  
  
-   [La clase MapHelper](../esb-toolkit/the-maphelper-class.md)  
  
-   [La clase de resolución](../esb-toolkit/the-resolver-class.md)  
  
-   [La clase ResolverDictionary](../esb-toolkit/the-resolverdictionary-class.md)  
  
-   [La clase del Administrador de resolución (ResolverMgr)](../esb-toolkit/the-resolver-manager-resolvermgr-class.md)
