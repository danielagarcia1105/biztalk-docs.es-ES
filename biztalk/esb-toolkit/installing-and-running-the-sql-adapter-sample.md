# <a name="installing-and-running-the-sql-adapter-sample"></a>Instalar y ejecutar el ejemplo del adaptador SQL
El ejemplo de adaptador de SQL muestra cómo puede usar el marco de trabajo de resolución dinámica para ejecutar un procedimiento almacenado de SQL mediante el uso de un proveedor de adaptador de WCF-Custom. El ejemplo usa la aplicación de cliente de prueba proporcionada con el ejemplo de itinerario en rampa.
