# <a name="contents-of-the-biztalk-esb-toolkit"></a>Contenido del Kit de herramientas ESB de BizTalk
El [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] proporciona una guía de arquitectura, patrones y prácticas y una colección de componentes de BizTalk Server y .NET Framework para simplificar el desarrollo de soluciones de escala empresarial en la plataforma de Microsoft. El Kit de herramientas también proporciona capacidades para ayudar a los desarrolladores extender de mensajería existentes y las soluciones de integración. El [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] consta de una colección de componentes que admiten e implementan un entorno de mensajería con acoplamiento flexible que simplifica el proceso de creación de aplicaciones empresariales basadas en mensajes dinámica interoperativos.  

 El [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] contiene los siguientes componentes:  

- [Servicios Web ESB](../esb-toolkit/esb-web-services.md). Estos exponen funcionalidades principales para los consumidores ESB, como el enrutamiento basado en itinerarios, administración de excepciones, resolución dinámica de extremos y asignaciones, las operaciones de BizTalk, interoperabilidad de Universal Description, Discovery and Integration (UDDI), y transformación del contenido del mensaje.  

- [Portal de administración de ESB](../esb-toolkit/esb-management-portal.md). Esto proporciona compatibilidad para las alertas de excepción y error de seguimiento, el reenvío de mensajes y notificaciones, UDDI integración, informes y análisis y capacidades de configuración.  

- [Los componentes de interoperabilidad de canalización ESB](../esb-toolkit/esb-pipeline-interop-components.md). Estos proporcionan a BizTalk Server en componentes de canalización que facilitan la interoperabilidad con sistemas externos.  

- [Marco de administración de excepciones](../esb-toolkit/exception-management-framework.md). Captura y consolida las excepciones de mensajería de BizTalk y orquestación subsistemas y proporciona un mecanismo para controlar los mensajes de error único. Incluye el servicio Web de excepción, la API de administración de excepciones y componentes que enriquecen, procesarán y pasan los detalles de la excepción en el Portal de administración de ESB.  

- [Marco de proveedores de adaptador y la resolución ESB](../esb-toolkit/esb-resolver-and-adapter-provider-framework.md). Esto implementa una arquitectura conectable y configurable para resolver dinámicamente los puntos de conexión y los requisitos de transformación y enrutamiento de mensajes.  

- [Basado en itinerarios enrutamiento y procesamiento](../esb-toolkit/itinerary-based-routing-and-processing.md). El mecanismo de procesamiento de itinerario proporciona una funcionalidad ligera para describir de forma dinámica, enviar y ejecutar varias invocaciones de servicio o solicitudes de enrutamiento y transformación.  

- [Aplicaciones de ejemplo de Kit de herramientas de BizTalk ESB](../esb-toolkit/biztalk-esb-toolkit-sample-applications.md). Las aplicaciones de ejemplo incluido muestran implementaciones posibles de la [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] y cómo puede usar el [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] características en sus propias aplicaciones SOA y ESB.
