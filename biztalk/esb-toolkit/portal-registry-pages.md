# <a name="portal-registry-pages"></a>Páginas del portal de registro
Cuando mueva el mouse sobre la **registro** pestaña, un menú emergente proporciona una lista de las tareas del portal de registro siguientes:  
  
-   [Nueva página de entrada de registro](../esb-toolkit/new-registry-entry-page.md). Este elemento de menú abre una página donde puede ver una lista de los puntos de conexión existentes en una aplicación de Microsoft BizTalk Server ESB especificada. Un vínculo en la fila para cada punto de conexión disponible abre la [página de detalles de registro](../esb-toolkit/registry-details-page.md) solicitar donde puede editar los detalles de registro o crear una nueva publicación de Universal Description, Discovery y Integration (UDDI) para el actualmente Configurar servidor UDDI.  
  
-   [Página de solicitudes pendientes de administración de](../esb-toolkit/manage-pending-requests-page.md). Este elemento de menú abre una página que enumera todas las solicitudes pendientes de registro de registro UDDI y le permite administrar estas solicitudes. También puede ver un historial de solicitudes anteriores y su estado.
