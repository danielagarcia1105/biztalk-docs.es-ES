# <a name="administration-with-the-biztalk-esb-toolkit"></a>Administración con el Kit de herramientas de ESB de BizTalk
Esta sección describe un área importante de la actividad administrativa relacionada con las aplicaciones basadas en el [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] y Microsoft BizTalk Server; en concreto, se trata el Portal de administración de ESB. El Portal de administración de ESB es una herramienta basada en Web para la supervisión de aplicaciones de BizTalk Server que genera alertas cuando surgen de excepciones y administra determinadas características de BizTalk ESB Toolkit, como el registro de Universal Description, Discovery, y Entradas de Integration (UDDI).  
  
 Esta sección contiene los siguientes temas:  
  
-   [Administración mediante el Portal de administración de ESB](../esb-toolkit/administration-using-the-esb-management-portal.md)  
  
-   [Referencia de características de Portal de administración de ESB](../esb-toolkit/esb-management-portal-feature-reference.md)
