# <a name="installing-and-running-the-resolver-service-sample"></a>Instalar y ejecutar el ejemplo de servicio de resolución
El ejemplo de servicio de resolución usa una aplicación de línea de comandos para ejecutar los métodos del servicio Web de resolución y guardar los resultados en un archivo de disco. Puede abrir el proyecto de cliente de prueba para ejecutarlo y examinar el código para ver cómo puede usar el servicio Web de resolución en su propia arquitectura orientada a servicios (SOA) y aplicaciones de ESB.
