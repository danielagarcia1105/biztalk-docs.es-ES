# <a name="install-the-itinerary-on-ramp-sample-using-the-install-scripts"></a>Instalar el ejemplo en rampa itinerario con los Scripts de instalación
En esta sección se describe cómo puede instalar el ejemplo de itinerario en rampa de las secuencias de comandos de instalación proporcionadas con el [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)].  
  
### <a name="to-install-the-itinerary-on-ramp-sample-from-the-install-scripts"></a>Para instalar el ejemplo de itinerario en rampa de las secuencias de comandos de instalación  
  
1.  En el menú **Inicio** , haga clic en **Ejecutar**.  
  
2.  En el **ejecutar** cuadro de diálogo, escriba **cmd**, y, a continuación, presione ENTRAR para abrir un símbolo del sistema.  
  
3.  Ejecute el comando siguiente, reemplazando el \<ruta de acceso\> parámetro con la ruta de acceso completa al archivo .cmd que desea instalar (la ruta predeterminada en esta versión es \Source\Samples\Itinerary\Install\Scripts\\):  
  
    ```  
    <path>\Setup_bin.cmd  
    ```
