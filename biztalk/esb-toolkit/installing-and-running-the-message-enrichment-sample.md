# <a name="installing-and-running-the-message-enrichment-sample"></a>Instalar y ejecutar el ejemplo de enriquecimiento de mensajes
El ejemplo Message Enrichment muestra cómo desarrollar una orquestación como un servicio personalizado de itinerarios para aplicar el modelo de diseño de enriquecimiento del mensaje por consultar una base de datos e incorporación de la respuesta a su mensaje. El ejemplo usa la aplicación de cliente de prueba proporcionada con el ejemplo de itinerario en rampa.
