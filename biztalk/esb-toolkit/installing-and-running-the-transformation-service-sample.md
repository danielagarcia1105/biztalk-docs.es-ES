# <a name="installing-and-running-the-transformation-service-sample"></a>Instalar y ejecutar el ejemplo de servicio de transformación
El servicio Web de transformación expone directamente las asignaciones implementadas en Microsoft BizTalk a través de un servicio Web, lo que elimina un viaje por el mensaje a la base de datos de cuadro de mensaje de BizTalk. El servicio Web de transformación permite que aplicaciones externas para enviar un documento a la aplicación Microsoft.Practices.ESB y hacer que se transforma utilizando un mapa que se implementan dentro de BizTalk.
