# <a name="manage-pending-requests-page"></a>Página de administración de solicitudes pendientes
La figura 1 muestra la página Administrar solicitudes pendientes, que muestra una lista de pendientes registro solicita y proporciona un vínculo para ver el historial de solicitudes.  
  
 ![Página de solicitudes pendientes de administración de](../esb-toolkit/media/ch8-managependingrequestspage.gif "Ch8-ManagePendingRequestsPage")  
  
 **Figura 1**  
  
 **La página de solicitudes pendientes para administrar Portal de administración ESB**  
  
 En la lista siguiente se explica cómo puede usar las características de la página de solicitudes pendientes para administrar Portal de administración ESB:  
  
-   Haga clic en el **ViewDetails** icono (la lupa) para abrir el [página de detalles de registro](../esb-toolkit/registry-details-page.md), donde puede ver los detalles de la solicitud pendiente, publicarlo, actualizarlo o eliminarlo.  
  
-   Haga clic en el **aprobar** icono (la marca de verificación) para aprobar la solicitud pendiente.  
  
-   Haga clic en el **rechazar** icono (la marca cruzada) para rechazar la solicitud pendiente.  
  
-   Haga clic en el **ver el historial de solicitudes** vínculo para abrir una página que muestra una gama de información para solicitudes de registro anteriores de Universal Description, Discovery y Integration (UDDI).
