# <a name="implementing-design-patterns-in-itineraries"></a>Implementar modelos de diseño en itinerarios
En este tema se describe patrones comunes de arquitectura orientada a servicios (SOA) y las categorías que se pueden implementar con la [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)]. Cada categoría de modelo incluye una descripción de cada modelo relacionado y vínculos a información más detallada. La descripción de cada patrón individual incluye referencias a relacionados [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] aplicaciones de ejemplo que implementan los patrones y vínculos a las actividades de desarrollo diseñadas para mostrar a los desarrolladores cómo crear soluciones ESB para implementar estos patrones. Esta sección incluye las siguientes secciones de tema:  
  
-   [Patrones de enrutamiento de mensajes](../esb-toolkit/message-routing-patterns.md)  
  
-   [Patrones de transformación de mensajes](../esb-toolkit/message-transformation-patterns.md)  
  
-   [Patrones de mediación de servicio](../esb-toolkit/service-mediation-patterns.md)  
  
-   [Patrones de la administración de servicio](../esb-toolkit/service-management-patterns.md)
