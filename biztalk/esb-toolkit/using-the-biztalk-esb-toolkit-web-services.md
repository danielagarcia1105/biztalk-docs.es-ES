# <a name="using-the-biztalk-esb-toolkit-web-services"></a>Usar los servicios Web del Kit de herramientas de ESB de BizTalk
La [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] expone todos sus principales características como servicios Web; Esto permite a las entidades fuera de ESB, podrían ejecutarse en plataformas de otros fabricantes, para participar en intercambios de ESB y aprovechar las ventajas de las capacidades ESB. Todos los servicios Web se encuentran en subcarpetas independientes de la carpeta \Web del directorio de instalación de ESB. El [!INCLUDE[esbToolkit](../includes/esbtoolkit-md.md)] incluye los servicios Web de ESB siguientes:  
  
-   [Los servicios Web de itinerario en rampa](../esb-toolkit/the-itinerary-on-ramp-web-services.md)  
  
-   [El servicio Web de resolución](../esb-toolkit/the-resolver-web-service.md)  
  
-   [El servicio Web de transformación](../esb-toolkit/the-transformation-web-service.md)  
  
-   [El servicio Web de control de excepciones](../esb-toolkit/the-exception-handling-web-service.md)  
  
-   [El servicio Web de las operaciones de BizTalk](../esb-toolkit/the-biztalk-operations-web-service.md)
