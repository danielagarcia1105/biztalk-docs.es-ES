# <a name="update-center-for-host-integration-server-and-ole-db-provider-for-db2"></a>Centro de actualización para Host Integration Server y el proveedor OLE DB para DB2

Buscar y administrar las actualizaciones de Host Integration Server (HIS) y el proveedor Microsoft OLE DB para DB2 (incluido en Feature Packs de Microsoft SQL Server).

## <a name="overview"></a>Información general
Esta página enumera las actualizaciones más recientes para las distintas versiones de Host Integration Server y proveedor Microsoft OLE DB para DB2. 

> [!NOTE]
> El [Microsoft Product Lifecycle Page](http://support.microsoft.com/lifecycle/) muestra el estado admitido para las distintas versiones HIS.

## <a name="biztalk-adapters-for-host-systems"></a>Adaptadores de BizTalk para sistemas host
Hay cuatro adaptadores de BizTalk para sistemas de Host que se incluyen con su:

* Adaptador de BizTalk para DB2
* Adaptador de BizTalk para aplicaciones Host
* Adaptador de BizTalk para los archivos de Host
* Adaptador de BizTalk para MQ Series Client (MQSC)

Estos adaptadores son actualizados por las actualizaciones acumulativas de HIS.
